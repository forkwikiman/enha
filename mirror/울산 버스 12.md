울산광역시 중구의 마을버스. 위 그림의 초록색/연두색 노선. 노선은 다음과 같다. (굵게 표시된 곳에서 시내버스로 환승 가능)

테크노파크 → 동원로얄듀크2차 → 함월고등학교 → 유곡중학교 → 동덕현대2차 / e-편한세상→유곡푸르지오(선택정차) → 축산농협`[4]` →
선경2차정문 → 우정시장 → **우정동주민센터**입구 → 청아예식장 → **목살골목**입구 → 성남동매일마트 → 국민은행 → 소방서사거리 →
<del>시계탑사거리</del>`[5]` → **박병원** → <del>안국한의원</del> → 현대자동차`[6]` → 역전시장 →
울산중고등학교 → 서울마트 → 동덕현대아파트 → 성지아파트 → 남운럭키 → 중구청앞`[7]` → **학성여고** → 홈플러스 →
울산MBC입구 → 울산교회 → 중앙동주민센터 → 문화의거리 → 국민은행 → **목살골목**입구 → 청아예식장 → **우정동주민센터**입구 →
선경2차정문 → 축산농협 → 동덕현대2차 / 유곡푸르지오→e-편한세상(선택정차) → 유곡중학교 → 함월고등학교 → 동원로얄듀크2차 →
테크노파크.

노선 생성 직후에는 박병원->[월마트](%EC%9B%94%EB%A7%88%ED%8A%B8.md)(현
[이마트](%EC%9D%B4%EB%A7%88%ED%8A%B8.md) 학성점)->중앙동주민센터 로 운행하였으나, 어느 순간부터(정확한
시점 추가바람) [홈플러스](%ED%99%88%ED%94%8C%EB%9F%AC%EC%8A%A4.md)로 운행하게 되었다. 또한
2010년대 유곡푸르지오 완공 이후 동덕현대와 유곡푸르지오로 경유지를 이원해 운행하고 있다. 비율은 1:1. 원래 우정동-목살골목 구간에서
동헌길을 이용했으나, 장춘로 개통 이후 장춘로로 운행중이다.

**태화시장**을 지나는 이면도로를 경유하는 관계로 장날(5, 10일)에는 다소 소요시간이 추가될 수 있다. 또한 등교시간대 전체와 하교시간대 2회에 한해 함월고 지원운행을 하고 있다. 기존에는 원유곡까지 진입했으나, 혁신도시 건설로 동네가 없어지는 바람에 (...).

2014년 8월 30일 노선 개편으로 기점이 유곡중학교에서 테크노파크로 이전되었다. 그래서 함월고가 기본정차하는 정류장으로 바뀌었고, 노선이
다운동까지 연장되었다.
