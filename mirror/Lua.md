# 1. 개요 ¶

1993년 개발된 [프로그래밍언어](%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D%20%EC%96%B8%EC%96%B4.md).
이름은 달을 의미하는 [포르투갈](%ED%8F%AC%EB%A5%B4%ED%88%AC%EA%B0%88.md)어 단어라고 한다. 스크립팅
언어를 지향하는 작고 가벼운 언어로, 프로그래머가 아닌 사람들도 사용하기 쉽다는 장점을 지녔는데다가 무엇보다 전부 몽땅 무료라는 캐장점을
지니고 있다. 게다가 오픈소스.`[1]`

  

[[edit](http://rigvedawiki.net/r1/wiki.php/Lua?action=edit&section=2)]

# 2. 설명 ¶

태생 자체가 가벼운 스크립트 언어이며, [C](C%28%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D%20%EC%96%B8%EC%96%B4%29.md)/[C++](C++.md)에 접착시키기 매우 쉽다. 개발 과정에 있어서 재컴파일이나
리로딩없이 바로 설정 변화를 적용할 수 있는 점 때문에 특히 게임업계에서 많이 쓰인다. 한국에선 [월드 오브 워크래프트](%EC%9B%94%EB%93%9C%20%EC%98%A4%EB%B8%8C%20%EC%9B%8C%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8.md)에서 유저들이 UI를 직접 제작할 수 있게 됨으로서 일반 게이머들에게도 인지도를 얻었는데, 사실 그전에도 여러 게임에 쓰였다.
[루아가 쓰인 게임들 리스트](http://en.wikipedia.org/wiki/Category:Lua-
scripted_video_games) 와우에서처럼 커스텀 UI 설정 언어로도 쓰이고, RPG게임 등에서 스크립트로도 쓰인다. 제작과정에서
내부적으로 활용하는 경우도 있고, API를 유저들에게 공개하여 MOD제작 등에 쓰이기도 한다. 국내게임으론 [라그나로크 온라인](%EB%9D%BC%EA%B7%B8%EB%82%98%EB%A1%9C%ED%81%AC%20%EC%98%A8%EB%9D%BC%EC%9D%B8.md)에서
NPC의 AI와 호문클루스의 AI, 용병의 AI에 Lua가 쓰였으며 이중 호문클루스와 용병은 유저가 직접 변경이 가능했다. 3차 직업군이
등장할 즈음에 용병의 AI를 변경하여 거의 0프레임수준의 스킬이 가능하다는게 여기저기 알려져 너도 나도 창용병 무명섬 3층 고고싱을 외친적이
있다.  
게리모드에서도 루아를 지원해주고, 일부 소스게임(혹은 모드)들도 lua를 사용한다. 특이한건 [GTA 4](GTA%204.md)에도
Lua 스크립트 모드가 있다. 심지어 [TAS](Tool-Assisted%20Speedrun.md) 영상을 만드는 데에도 일부
사용한다.

  

한게임에서 제공중인 오픈 마켓형 게임 툴인 아이두게임도 기본 언어는 Lua로 이루어져있다. 초보 프로그래머들을 위해서 배려를 해준
것이라고...

  

[하프 라이프 2](%ED%95%98%ED%94%84%20%EB%9D%BC%EC%9D%B4%ED%94%84%202.md)의 모드격
게임인 [게리 모드](%EA%B2%8C%EB%A6%AC%20%EB%AA%A8%EB%93%9C.md)에서 유저들이 직접 메커니즘을
디자인할때 쓰이는 언어다. 이 언어 덕분에 상자로 만든 건물부터 투석기, 전투기, 헬리콥터, [골드버그장치](%EA%B3%A8%EB%93%9C%EB%B2%84%EA%B7%B8%20%EC%9E%A5%EC%B9%98.md), 컴퓨터(!)까지
유저 맘대로 만들 수 있게 되었다.

  

언어적 특징으로는 최대한 적은 수의 기본 데이터형만을 지원함으로써 성능과 크기 사이의 균형을 맞추는 것을 들 수 있다. 즉, 가볍고 빠른
언어라는 것. 배열, 집합, 해쉬 테이블, 리스트같은 전형적인 데이터 구조는 전부 테이블 자료형으로 구현하고 그 외에 네임스페이스나 객체들도
테이블로 표현하는 등, 어떻게든 가벼운 언어를 만드려는 게 특징.

  

[아이패드](%EC%95%84%EC%9D%B4%ED%8C%A8%EB%93%9C.md)로 만든 Lua개발 툴도 있다! 이름은 공개 당시
Codify, 현재 Codea이다.[사이트 링크](http://twolivesleft.com/Codea/). 보면 막 패드에서 이래저래
코딩해서 만든다. `[2]` <del>그리고 패드는 개발자용 장난감이 되겠지</del>

# 3. [파이썬](%ED%8C%8C%EC%9D%B4%EC%8D%AC.md)과 비교 ¶

비슷한 컨셉의 [파이썬](%ED%8C%8C%EC%9D%B4%EC%8D%AC.md)과 많이 비교가 되는데, '배터리가 포함된'`[3]`
파이썬은 어느정도 단독으로 프로그램을 만들 수 있는 환경인 반면, 루아는 기본 기능만을 포함하고 있어 단독으로 응용프로그램을 만들기는
어렵다. 물론 그만큼 가볍다는 이야기이도 하다.

  

[[edit](http://rigvedawiki.net/r1/wiki.php/Lua?action=edit&section=4)]

# 4. 여담 ¶

최근에는 [Corona sdk](http://coronalabs.com/)라는 게 나와서 lua만 배워서 게임 하나 뚝딱 만들어 내는 게
가능해졌다. 아직 [iOS](iOS.md)만 지원하지만, 추후 윈도 데스크탑용도 출시 예정이라고 한다.
