## 3. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

참고로 디스토션 드라이브 및 아스트랄 히트의 기술 명칭에 공통적으로 들어간 "사자신인법(獅子神忍法)"은 "시시가미 인법"이라고 읽을 수도
있다.

  

★ 드라이브 - 버닝 하트(バーニングハート / Burning Heart) : 상대를 불태우는 공격입니다. 명중하면 풍림화산 아이콘 하나가
채워집니다.  
★ 오버 드라이브 - 시시가미 인법 궁극오의 '만구풍림화산' 개(獅子神忍法・究極奥義「萬駆風林火山」改)

  

★ 필살기  

  * 비술 반그 순간이동 술법(秘術・バング瞬間移動の術 / Secret Art: Bang's Shadowstep Jutsu) : 순간이동을 합니다. 버튼에 따라 나타나는 위치가 달라집니다.
  * 비술 질풍 반그 순간이동 술법(秘術・疾風バング瞬間移動の術 / Secret Art: Hurricane Bang's Shadowstep Jutsu) : 공중에서 순간이동을 합니다. 버튼에 따라 나타나는 위치가 달라집니다.
  * 진공열풍 반그 떨구기(真空烈風バング落とし / Bang's Void Tempest Kick) : 강력한 잡기 기술입니다.
  * 승천분쇄 반그 떨구기 Ⅱ (昇天粉砕バング落とし・改 / Bang's Void Tempest Kick EX) : 공중에서 상대를 잡는 강력한 잡기 기술입니다.
  * 반그 쌍장타 금강극(バング双掌打・金剛戟 / Bang's Unstoppable Double Palm Thrust) : 상대를 날려버리는 공격입니다.
  * 반그 쌍장타 천강극(バング双掌打・天剛戟 / Bang's Heavenly Double Palm Thrust) : 공중에서 상대를 날려버리는 공격입니다.
  * 반그류 수리검술(バング流手裏剣術 / Bang-Style Shuriken) : 상대를 향해 못을 날립니다. 버튼에 따라 날리는 못 종류가 달라집니다. 사용 후에는 못의 개수가 줄어듭니다.
  * 못 설치(지상) : 화면에 못을 설치합니다. 설치된 못 가까이에서 방향키를 누르면 그 방향으로 고속 이동을 합니다. 사용 후에는 못의 개수가 줄어듭니다.
  * 못 설치(공중) : 공중에서 화면에 못을 설치합니다. 설치된 못 가까이에서 방향키를 누르면 그 방향으로 고속 이동을 합니다. 사용 후에는 못의 개수가 줄어듭니다.
  * 열공 무사사비 술법(裂空ムササビの術 / Tempest Flying Squirrel Jutsu) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 사자신인법 초오의 '만구활살대분화'(獅子神忍法・超奥義・「萬駆活殺大噴火」 / Shishigami-Style Secret Technique: Fatal Eruption) : 상대를 향해 돌진해 연속 공격을 합니다.
  * 사자신인법 궁극오의 '만구풍림화산'(獅子神忍法・究極奥義・「萬駆風林火山」 / Shishigami-Style Forbidden Technique: Fu-Rin-Ka-Zan) : 풍림화산을 발동해 파워 업을 합니다. 사용 후에는 고속으로 이동할 수 있으며 일반 가드는 불가능합니다.
  * 사자신인법 절오의 '만구람화전'(獅子神忍法・絶奥義・「萬駆嵐禍陣」 / Shishigami-Style Ultimate Technique: Steel Rain) : 상대의 머리 위에서 여러 개의 못을 뿌립니다. 사용 후에는 못의 개수가 줄어듭니다.
  * 사자신인법 폭렬오의 '만구아수라무쌍권'(獅子神忍法・爆裂奥義「萬駆阿修羅無双拳」 / Shishigami-Style Technique: Bang's Infinite Chaos-Fist of the Void) : 반그가 두 명으로 분신해 크로스 촙으로 공격합니다.
  * 사자신인법 격오의 '격만구질풍격'(獅子神忍法・激奥義・「激萬駆疾風撃」 / Shishigami-Style Secret Technique: Hurricane Fury)`[1]` : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 아스트랄 히트  

  * 사자신인법 열혈최종오의 '구극만구'(獅子神忍法・熱血最終奥義・「究極萬駆」 / Shishigami-style Forbidden Technique: The Ultimate Bang) : 가드 불능 공격입니다.
