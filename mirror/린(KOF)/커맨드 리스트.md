# 1. [KOF 2000](KOF%202000.md) ¶

  

★ 기본잡기  

  * 악참 : 근접해서 ← or → + C
  * 악참귀 : 근접해서 ← or → + D  

★ 특수기  

  * 변발권 : → + A
  * 비의팔극권 타개 : ↘ + C  

★ 필살기  

  * 철참무 나살 : ↓↘→ + A or C  
┗ 철참무 나의 : →↘↓↙← + A or C  

┗ 철참무 파경 : → + A or C

  * 무영홍사수 : →↓↘ + A or C
  * 안개 : ↓↙← + A or C
  * 비모각 : ←↙↓↘→ + B or D
  * 비천각 : →↘↓↙← + B or D   

★ 초필살기  

  * 비적오의 난무 - 독나방 : ↓↙←↙↓↘→ + A or C
  * 비적오의 영향 : ←↙↓↘→←↙↓↘→ + B or D  

★ MAX 초필살기  

  * 비적오의 난무 - 독나방 : ↓↙←↙↓↘→ + AC
  * 비적오의 독수공 : ↓↓↓ + AC (잡기)
