## 3. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

★ 드라이브 - 댄싱 엣지(ダンシングエッジ / Dancing Edge) : 고속으로 돌진 공격을 합니다. 명중했을 때 →를 이용해 앞으로
크게 뛰어오를 수 있습니다.  

┗급정지 : 그 자리에서 급정지해 착지합니다.  
┗뒤 돌기 : 상대 뒤쪽으로 돌아 들어갑니다.  
┗뛰기 : 수직 방향으로 뛰어오릅니다.

★ 오버 드라이브 - 두 마리처럼 된다옹! (ほぼ2匹になるニャス！ / Almost Becoming Two!)

  

★ 특수기  

  * 앉아 걷기 : 자세를 아주 낮게 한 채로 전진합니다.  

★ 필살기  

  * 고양이 혼 원! (ネコ魂ワン！ / Cat Spirit One!) : 손톱을 휘두르며 앞으로 연속 공격합니다.
  * 고양이 혼 투! (ネコ魂ツー！ / Cat Spirit Two!) : 손톱을 휘두르며 아래쪽으로 연속 공격합니다.
  * 고양이 혼 쓰리! (ネコ魂スリー！ / Cat Spirit Three!) : 앉아서 가드할 수 없는 연속 공격을 합니다. 버튼을 누르고 있으면 연속 공격 타이밍을 바꿀 수 있습니다.
  * 필살 고양이 마구! (必殺ネコ魔球！ / Kitty Litter Special!) : 다양한 물건을 던져 상대를 공격합니다. C판에서는 버튼을 누르고 있으면 공격 타이밍을 바꿀 수 있습니다.
  * 고양이 뛰기 (ねこっとび！ / Cat Jump!) : 앞으로 재빠르게 뛰어오릅니다.
  * 타오 찐득이 (タオぴったん / Sticky Kitty) : 버튼을 누르고 있는 동안 벽에 붙어 있습니다. 붙어 있을 때 →를 이용해 앞으로 빨리 뛰어오를 수 있습니다.
  * 댐싱 엣지! (だましんぐエッジ！ / Trick Edge!) : 화면 끝을 향해 크게 뛰어오른 다음 돌진 공격을 합니다. 돌격 공격 전에 ↓ 또는 ←를 이용해 돌진 공격 궤도를 바꿀 수 있습니다.
  * 쓱싹쓱싹! (ギッザギザ！ / Slashy-Slashy!) : 버튼을 누르고 있는 동안 톱으로 공격합니다.
  * 고양이 혼 앵콜! (ネコ魂アンコール！ / Cat Spirit Encore!) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 헥사 엣지 (猫の人直伝・ヘキサエッジ / Cat Person's Secret Art: Hexa-Edge) : 앞으로 돌진하면서 연속 공격합니다.
  * 우당탕탕 콰르르릉! (メッタメタのギッタギタ！ / I'mma Beat the Crap Outta You!) : 상대에게 뛰어들어 가드 불능 공격을 합니다. 단, 상대가 앉아 있을 때는 맞지 않습니다.
  * 두 마리처럼 된다! (ほぼ二匹になる！ / Almost Becoming Two!) : 자신과 똑같이 움직이는 분신을 만들어냅니다.
  * 유니존 냐이브! (ゆにぞんニャいぶ！ / Aerial Dancing Edge!!) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 아스트랄 히트  

  * 어택 넘버 원! (あたっくにゃんばーわん！ / Attack Meow Pow!) : 화면 구석으로 이동한 다음 돌진 공격을 합니다.
