## 2. BlazBlue : ChronoPhantasma ¶

★ 드라이브 - 우로보로스

  

★ 오버 드라이브 - 요르문간드 : 드라이브 공격이 강화되고, 상대의 체력을 자동흡수하는 링이 출현한다.

  

★ 필살기  

  * 사인아 : ↓↘→ + D
  * 사교: →↓↘ + D
  * 사찰 : ↓↙← + D  
┠ 열섬아 : 사찰 or 사활 중 A  
┠ 아승각 : 사찰 or 사활 중 B  
┠ 잔영아 : 사찰 or 사활 중 C  
┠ 자세 중단 : 사찰 중 D  
┗ 사활 : 사찰 중 ←← or →→

  * 비겸돌 : 공중에서 ↓↙← + B
  * 아쇄충 : ↓↘→ + C  

★ 디스토션 드라이브  

  * 사익붕천인 : ↓↘→↓↘→ + B
수직 방향으로 돌려차기를 해 상대를 매우 높게 띄우는 기술.  
※ 오버드라이브 - 모션 변경, 데미지 증가 `[13]`  

언리미티드 - 상대와의 거리에 상관없이 무조건 기술 적중 범위로 순간이동  

  * 교룡열화참 : →↘↓↙←→ + C
상대를 조그마한 녹색의 장판으로 붙잡고 우로보로스 사슬로 끌고와서 단검으로 난무를 펼치고 우로보로스로 날려버리는 기술.  
※ 오버드라이브 - 히트 수, 연출시간 증가 `[14]`  

언리미티드 - 상대와의 거리에 상관없이 무조건 상대 밑에 장판 발생, 특정 기술 적중 시 난무와 우로보로스 타격이 일반 필살기로 사용됨  

  * 대사무련장 : 사찰 or 사활 중 →↘↓↙←→ + D `[15]`
상대를 우로보로스 사슬로 붙잡아 공중으로 조금 띄운다음 발차기 2번으로 상대를 바닥에 눕힌뒤 마구 밟고 날리는 기술.  
※ 오버드라이브 - 히트수, 연출시간 증가  

언리미티드 - 특별한 효과 없음  

  * 언리미티드 전용기 : 오버드라이브 중에 ↓↘→↓↘→ + C
상대의 뒤로 이동하여 크래시트리거를 사용한 뒤에 교룡열화참의 단검난무를 평소보다 길게 한 뒤에 대사무련장의 OD연출 증가분을 그대로
사용한다.

## 3. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

★ 드라이브 - 우로보로스(ウロボロス / Ouroboros) : 쇠사슬을 날립니다. 레버와의 조합에 따라 여러 방향으로 날릴 수 있습니다.  
　┠ A파생 : 늘어난 쇠사슬을 집어넣습니다.  
　┠ B파생 : 공중으로 올라갔다가 쇠사슬 끝으로 이동합니다.  
　┠ C파생 : 잠시 그 자리에 멈췄다가 순식간에 쇠사슬 끝으로 이동합니다.  
　┗ D파생 : 쇠사슬로 고속 이동합니다.`[16]`

  

★ 오버 드라이브 - 요르문간드(ヨルムンガンド / Jörmungandr)

  

★ 필살기  

  * 사인아(蛇刃牙 / Venom Sword) : 전방에 검은 기운의 덩어리를 발사합니다.
  * 사찰(蛇刹 / Serpent's Benediction) : 여러 추가 기술을 위한 자세를 잡습니다. 자세를 잡는 도중에 D를 누르면 중단할 수 있습니다.  
┣ 열섬아(烈閃牙 / Falling Fang) : 앞으로 날아올라 내려치는 중단 공격입니다.  
┣ 아승각(牙昇脚 / Rising Fang) : 위쪽으로 차올리는 대공 기술입니다.  
┣ 잔영아(残影牙 / Devouring Fang) : 나이프로 상대의 발밑을 노리는 하단 기술입니다.  
┣ 사활(蛇滑 / Serpent's Redemption) :
[추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  
┗ 자세 중단(構え中断 / Serpent's Haste) : 자세를 종료합니다.

  * 비겸돌(飛鎌突 / Wind Serpent's Fang) : 공중에서 내리찍습니다.
  * 사교(蛇咬 / Hungry Coils) : 45도 각도로 우로보로스를 발사합니다.
  * 아쇄충(牙鎖衝 / Bloody Fangs) : 상대를 순간적으로 무방비 상태로 만드는 커맨드 잡기 기술입니다.  

★ 디스토션 드라이브  

  * 사익붕천인(蛇翼崩天刃 / Serpent's Infernal Rapture) : 상대를 공중으로 차올리는 디스토션 드라이브입니다.
  * 교룡열화참(蛟竜烈華斬 / Eternal Coils of the Dragon Serpent) : 상대를 붙잡아서 나이프로 마구 공격하는 디스토션 드라이브입니다.
  * 대사무련장(大蛇武鍊葬 / The Serpent's Unholy Wrath) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)
★ 아스트랄 히트  

  * 천혼명락(千魂冥烙 / Hungry Darkness of 1000 Souls) : 땅에서 우로보로스를 출현시켜 공격합니다.
