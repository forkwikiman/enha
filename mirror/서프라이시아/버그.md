## 1. [개요](%EA%B0%9C%EC%9A%94.md) ¶

서프라이시아는 1인 작품임에도 데이터베이스가 매우 방대한 만큼, 제작자가 실수한 부분이 많이 존재한다.

  

아래 적혀있는 버그들의 대부분은 단순히 제작자의 설정 미스로 1.13 패치와 동시에 해결될 예정이다.

  

대체로 큰 문제가 되지는 않지만, [쯔꾸르](%EC%AF%94%EA%BE%B8%EB%A5%B4.md)라는 툴 자체의 버그 역시 상당량
존재한다. 해당 버그는 수정할 방법도 없는 주제에 플레이어들에게 매우 치명적인 피해를 주기 때문에 플레이어 스스로가 저장을 습관화할 필요가
있다.

## 2. **툴 자체의 버그** ¶

  * 상태변화 장비 장착 후 제대로 표기되지 않는 버그
  * RPG_RT 어드레스 읽기 위반 튕김
  * 2배 버프 걸리는 장비 장착후 능력치가 999를 뚫어서 표기되는 버그
  * 비행선에서 shift키를 누르면 인덱스를 초과했다며 튕김`[1]`
  * 적 공격 타이밍에 아이템을 사용할 시 아이템 사용이 씹힘
  * 아군이 응원가 등의 특수 버프 등을 걸려고 행동을 선택하다가, 행동 직전 버프가 걸리면 행동이 씹힘
  * 가끔 기절했다가 부활한 파티원의 게이지가 차오르지 않음  

다른 [RPG2003](RPG2003.md) 게임에서도 발생하는 버그로, 서프라이시아에서만 나오는 버그는 아니다.

### 4.1. 몬스터 관련 ¶

  * **패턴 오류**  

    * 분노 등의 능력치 상승 스킬 : [매번 시전후 조금씩 능력치가 상승](http://cafe.naver.com/surprisia2008/11355)
    * 원주민 전사 (2부) : 다른 적에게도 광폭화를 검
    * 아누비스 (3부) : 석판을 사용하지 않음`[3]`
    * 역삼각 (3부) : [시간의 탑 역삼각](http://cafe.naver.com/surprisia2008/10346)이 적외선 굴절없이 잡힘
    * 암모나이트 (3부) : 모으기 없이 소용돌이 사용
    * 신전 미사일 마신병 (종장) : 미사일에 에너지장을 검
    * [핵발전소 잡몹](http://cafe.naver.com/surprisia2008/11270) (종장) : 『용접』 기술 명중시 불붙음, 감전이 걸리는데 정화시 그림자 상태가 됨.
    * 누더기 (현상금) : 좀비가 없는데도 좀비를 먹고 체력을 회복함
    * 작가 (현상금) : 훔치기시 우쿨렐레가 뜸, 재도전시 작가에게 사용하면 무효화되고, 뒤덮인잉크는 생성된채 남아 있음.
    * S급용 네마리 (현상금) : 용의 들끓는 피 시전시 용비늘꿰뚫기 내성 계속 증가, 보스가 죽어도 도마뱀이나 기가리온 구체 등, 부하들이 사라지지 않음.
    * 타이탄 (현상금) : 가끔 재전투시 준비없이 초강력안광 등을 날림  

  * **상태 관련**  

    * 시간의탑 스켈레톤 : 언데드임에도 불구하고 100% 의식상실에 걸림
    * 불뱀 : 화상에 걸림
    * 맹독의새끼용 : 독에 걸림
    * 글레이쟈드 : S급인데 멍해짐 상태이상 걸림
    * 타이탄 : SS급이 4식에서 상태이상에 걸림 
    * 크라켄 : S급이 불붙음 상태이상에 걸림  

  * **기타**  

    * 매드럼 : 빈사상태 표시 누락
    * 원주민 토템 (2부) : 부활시 [HP 2000](http://cafe.naver.com/surprisia2008/8355)을 회복하고 부활. 800으로 수정해야 할듯.

### 4.2. 아군 스킬 관련 ¶

  * **스킬 변동**  

    * 작가 : 마법을 배울 수 있음
    * 미르비스 : 2회차 이후 전회차에 배운 강화마법을 유지하고 있음

  * **스킬 효과**   

    * 맹독찌르기 : 잔챙이로 나오는 장수하늘소에게 [독이 두개가 걸림](http://cafe.naver.com/surprisia2008/14001)

      * 사용시 이로운 상태이상 제거 `[4]`

    * 힐스 Lv2 : 사용시 공격마법 모션이 발동
    * 음유시인의 노래/광시곡 : 시전 중 파티 이탈시 연주효과 지속, 광시곡은 [기적 상태방어 누락](http://cafe.naver.com/surprisia2008/8701) 발생
    * 정신집중 : 효과 중첩가능.
    * 지우기,지우기스 : 적이 격노 등 능력치 상승버프 사용후 시전시 비물리 데미지 감소
    * 비전의격류 : 가끔 이 상태에서 보조마법 사용가능
    * 아드레날린 : 정화의방패, 정화의목걸이, 챔피언의메달 등에 씹힘.
    * 운석 : 서낀은 다른 최종기보다 MP 소모율이 더 나감.
    * 수류탄 제작 : 재료들이 감소되지 않아 무한 제작가능
