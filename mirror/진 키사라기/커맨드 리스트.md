## 4. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

★ 드라이브 - 영도; 프로스트 바이트(零刀; フロストバイト / Frost Bite) : 상대를 얼립니다.

  

★ 오버 드라이브 - 절도; 프로스트 엔드(絶刀; フロストエンド / Frost End)

  

★ 필살기  

  * 빙상검(氷翔剣 / Ice Blade) : 앞쪽으로 장풍을 날립니다.
  * 눈보라(吹雪 / Blizzard) : 대각선 위쪽으로 대공 공격을 합니다.
  * 열빙(裂氷 / Violent Ice) : 바로 위쪽으로 대공 공격을 합니다.
  * 무창 첨정참(霧槍·尖晶斬 / Crystal Strike) : 앞으로 돌진하는 공격입니다.
  * 설화진(雪華塵 / Snow Flower Storm) : 앞쪽을 연속으로 공격합니다.
  * 빙상격(氷翔撃 /Flying Ice Strike) : 명중하면 상대를 얼리는 장풍을 날립니다.
  * 빙연쌍(氷連双 / Dual Ice Strike) : 범위 안에 대공 공격을 합니다. 버튼을 누르고 있으면 공격 타이밍을 바꿀 수 있습니다.
  * 무창 돌정격(霧槍·突晶撃 / Freezing Crystal Strike) : 앞으로 돌진하는 공격입니다.
  * 빙참섬(氷斬閃) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)
  * 빙참격(氷斬撃) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 동아빙인(凍牙氷刃 / Ice Fang) : 앞쪽에 얼음 충격파를 날립니다.
  * 빙익월명(氷翼月鳴 / Moonsong) : 대각선 위쪽으로 얼음 화살을 날립니다.
  * 허공진 설풍(虚空陣·雪風 / Empty Sky Form: Winter's Reposte) : 강력한 반격기입니다.  

★ 아스트랄 히트  

  * 연옥빙야(煉獄氷夜 / Arctic Dungeon) : 지면 전체에 가드 불능 공격을 합니다.
