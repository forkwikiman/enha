## 3. 개요와 특징 ¶

은평공영차고지에서 출발하여 [남대문시장](%EB%82%A8%EB%8C%80%EB%AC%B8%EC%8B%9C%EC%9E%A5.md)까지
운행하는 지선버스. [유성운수](%EC%9C%A0%EC%84%B1%EC%9A%B4%EC%88%98.md)에서 운영하는
[서울특별시](%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C.md) 지선버스 중 하나.
[2004년](2004%EB%85%84.md) 버스 개편 전에는 도시형버스 2번으로 운행하던 노선이 단축된 노선이다.(조금 구체적으로
말하면 마포 구간 대체 노선이다. 나머지 구간은 대체로
[171번](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%20171.md)에 계승됨)  
처음에는 7013번으로 단독 운행했으나 2012년 7월 현재에는 2개의 지선을 보유하고 있다.

  

덤으로, 회차구간이 몇번이나 바뀌기도 하였다.  

  * 개편 초기 : 아현초등학교 - [충정로역](%EC%B6%A9%EC%A0%95%EB%A1%9C%EC%97%AD.md)(5호선) - 경찰청 - [서울역](%EC%84%9C%EC%9A%B8%EC%97%AD.md) 회차
  * 1차 변경 : 아현초등학교 - [충정로역](%EC%B6%A9%EC%A0%95%EB%A1%9C%EC%97%AD.md)(5호선) - 경찰청 -> [서울역](%EC%84%9C%EC%9A%B8%EC%97%AD.md) -> 남대문시장(구.시경앞)[신세계 본점](%EC%8B%A0%EC%84%B8%EA%B3%84%EB%B0%B1%ED%99%94%EC%A0%90.md) -> [회현역](%ED%9A%8C%ED%98%84%EC%97%AD.md) -> [한국경제신문](%ED%95%9C%EA%B5%AD%EA%B2%BD%EC%A0%9C%EC%8B%A0%EB%AC%B8.md)사 -> 경찰청
  * 2차 변경 : 아현초등학교 - [충정로역](%EC%B6%A9%EC%A0%95%EB%A1%9C%EC%97%AD.md)(2호선) - [한국경제신문](%ED%95%9C%EA%B5%AD%EA%B2%BD%EC%A0%9C%EC%8B%A0%EB%AC%B8.md)사 -> [서울역](%EC%84%9C%EC%9A%B8%EC%97%AD.md) -> [남대문시장](%EB%82%A8%EB%8C%80%EB%AC%B8%EC%8B%9C%EC%9E%A5.md)(구.시경앞)신세계 본점 -> [회현역](%ED%9A%8C%ED%98%84%EC%97%AD.md) -> [한국경제신문](%ED%95%9C%EA%B5%AD%EA%B2%BD%EC%A0%9C%EC%8B%A0%EB%AC%B8.md)사 
(이로써 [마포역](%EB%A7%88%ED%8F%AC%EC%97%AD.md) ~
[남대문시장](%EB%82%A8%EB%8C%80%EB%AC%B8%EC%8B%9C%EC%9E%A5.md) 구간이 사실상 개편전 77번
대체가 되었다. 참고로 구 77번은 현재
[463번](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%20463.md) 간선버스로, 개편되면서 기존
종근당 경유에서 만리동 경유로 바뀌었다.)  

  * 3차 변경 : 아현초등학교 - [충정로역](%EC%B6%A9%EC%A0%95%EB%A1%9C%EC%97%AD.md)(2호선) - [한국경제신문](%ED%95%9C%EA%B5%AD%EA%B2%BD%EC%A0%9C%EC%8B%A0%EB%AC%B8.md)사 -> [서울역](%EC%84%9C%EC%9A%B8%EC%97%AD.md) -> [회현역](%ED%9A%8C%ED%98%84%EC%97%AD.md) -> 남산3호터널U턴 -> [남대문시장](%EB%82%A8%EB%8C%80%EB%AC%B8%EC%8B%9C%EC%9E%A5.md)(구.시경앞) -> 한국경제신문사
([서울역](%EC%84%9C%EC%9A%B8%EC%97%AD.md) 고가도로 차량통행 제한으로 인한 변경. 503, 505번
간선버스와 비슷한 회차 경로다. 초기에는
[7011번](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%207011.md)과 함께 '염천교'
정류장([HSBC](HSBC.md) 서울지점 앞 위치)을 통과했으나 지금은 정차)

  

[마포구](%EB%A7%88%ED%8F%AC%EA%B5%AC.md)청에서 바로
[망원역](%EB%A7%9D%EC%9B%90%EC%97%AD.md)으로 갈 듯 하면서 성산2교, 성미산 쪽으로 돌아가고, 상수역
이후에는 신수동, 용강동의 골목길로 들어가며, 용강동에서 [마포역](%EB%A7%88%ED%8F%AC%EC%97%AD.md)으로 진출할
때에는 마땅히 U턴할 방법이 없어 [강변북로](%EA%B0%95%EB%B3%80%EB%B6%81%EB%A1%9C.md)의
[마포대교](%EB%A7%88%ED%8F%AC%EB%8C%80%EA%B5%90.md) 북단 분기점을 이용하여 삥 돌아 회차하고,
마포대로에서는 [중앙버스전용차로](%EC%A4%91%EC%95%99%EB%B2%84%EC%8A%A4%EC%A0%84%EC%9A%A9%EC%B0%A8%EB%A1%9C.md)도 이용하는 노선. 263번이 마장동 차고지를 쓸 수 없게 되어 염곡동으로 연장되었을 적에 여의도 구간을
단축하고 7013A/B번처럼 [강변북로](%EA%B0%95%EB%B3%80%EB%B6%81%EB%A1%9C.md)를 삥 돌아 회차하여
[마포역](%EB%A7%88%ED%8F%AC%EC%97%AD.md)으로 돌아오는 방식으로 노선 변경을 시도했다가 마장동과 여의도 쪽
민원 때문에 결국 여의도 구간이 존치되어
[463번](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%20463.md)으로 출발지 번호만 변경됬다.

  

7013A/7013B로 분리 운행하고 있으며, **두 노선의 차이점은
[광흥창역](%EA%B4%91%ED%9D%A5%EC%B0%BD%EC%97%AD.md)을 직접 경유하느냐 경유하지 않느냐**로 나뉜다.
분리운행하는 노선이 으레 그렇듯이 배차간격은 그렇게 좋지 못하다. 한 노선이 곧 오면 뒤의 노선은 20분 뒤에 온다는 도착정보 표시를 보기가
쉽다. 반대로 A/B노선이 몰려오는 경우도 꽤 있는데, 이럴 때 차를 놓치면 조금 난감하다.(몰려 왔으니 보나마나 다음 차는 한참 있다 오기
때문) 배차간격 문제로 A/B 분기노선들이 상당수 통합된데 반해 이 노선은 A/B 각각의 단독구간이 있기 때문에 통합은 어렵다.

  

[서울 지하철 6호선](%EC%84%9C%EC%9A%B8%20%EC%A7%80%ED%95%98%EC%B2%A0%206%ED%98%B8%EC%84%A0.md)이 [마포구](%EB%A7%88%ED%8F%AC%EA%B5%AC.md)를 관통하고 있지만, 6호선의 위 아래
지역으로는 은근히 교통이 불편한 경우가 많다.
[7011번](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%207011.md)이 망원동, 상수동 부근의 교통
사각지대를 지난다면 이 버스는 신수동, 용강동, 도화동을 지나고 있어 이 지역에 사는 주민들의 발이 되어 주고 있다.
