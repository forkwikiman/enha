# 1. 설명 ¶

사슬낫을 이용하는 프리미엄 용병이다. 인기가 출시부터 바닥을 기는데 기본기 연계가 미끄럽지 않고 공중공격도 좋은편도 아니며 무엇보다 성능도
좋지가 않다. D꾹 콤보도 한번끝나면 강제기립이라서 여러번 할수도 없다.

  * D꾹후 DD:**어플러즈** 버닝콜 이후에만 나오며 사슬낫으로 적을 모으고 내려찍어 스턴상태에 만든다.쓰러진 적도 스턴에 걸리게 하지만 더 중요한게 있으니...
  * 어플러즈 후 D연타:퍼니쉬먼트.**드네가 쓰기 어려운이유,드네가 고인인 이유** 이게 D누른 횟수에 따라 다운,무끓,스턴,넉백(물론 쓰러진 적에겐 그런거 없다.)한번 누르면 다운,두번 누르면 무끓,세번 누르면 스턴,네번 누르면 넉백인데.... 이게 반응속도가 빨라야하며, 잠시라도 주춤거리면 다시 버닝콜이 나와버리니... 사실상 잘 보이는건 넉백뿐일것이다. 무엇보다 대시공격의 무끓이 훨신 쉬운관계로... 그래도 나름 간지는 나는편이다.
  * A D : 발로 내려찍어 강경직을 주는 뭔가 아쉬운 공격...이긴한데 빠르게 D꾹시 무끓이 나온다.원캐라면 꼭 익히자.

  * 대시 DDD :2타에 무끓,3타에 띄우는 대시공격. 3타전까지는 대시 D꾹과 연계 가능하다.
  * 대시 D꾹 :**블레이드러너.**순간적으로 전진이동하며 적 타격시에 공중으로 순간이동하며 바닥을 폭파시켜 적을 띄운다.
  * 반격:뒤로 잠시 물러서서 표창같은걸 던진다.

## 3.2. 다크컨빅션 ¶

화상 비스무리한 상태이상이 걸리며 적을 멀리 날려버린다.
