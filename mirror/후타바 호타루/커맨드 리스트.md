#### 1 [아랑 MOW](%EC%95%84%EB%9E%91%20MOW.md) ¶

  

★ 기본잡기  

  * 천락투 : 근접해서 ← or → + C  

  * 반전투 : 공중 근접해서 ↑ 이외 방향 + C  

★ 특수기  

  * 공중 뒤돌아보기 : 공중에서 CD  

  * 고축타 : ↓ + B  

★ 컴비네이션  

  * 점프 A ▶ B or C  

★ TOP기  

  * 선회연 : CD  

★ 필살기  

  * 발기장 : ↓↘→ + A or C  

  * 전신상 : →↓↘ + B or D  

  * 연격축 : 공중에서 ↓↙← + B or D  

  * 쌍장진 : ↓↙← + A or C  

  * 사상퇴 : ←↙↓↘→ + D (잡기)  

  * 호미각 : ↓↙← + B or D  

★ 초필살기  

  * 쌍장 천련화 : ↓↘→↓↘→ + A  

  * 천상 난희 : ↓↘→↓↘→ + B  

★ 잠재능력  

  * 쌍장 천련화 : ↓↘→↓↘→ + C  

  * 천상 난희 : ↓↘→↓↘→ + D  

#### 2 [네오지오 배틀 컬리시엄](%EB%84%A4%EC%98%A4%EC%A7%80%EC%98%A4%20%EB%B0%B0%ED%8B%80%20%EC%BB%AC%EB%A6%AC%EC%8B%9C%EC%97%84.md) ¶

  

★ 기본잡기  

  * 천락투 : 근접해서 CD  

  * 반전투 : 공중 근접해서 CD  

★ 특수기  

  * 천격축 : → + B (사용 후 연운 십육수의 자세)  

  * 선격수 : ↘ + C (사용 후 이연의 자세)  

  * 고축타 : ↓ + B  

★ 필살기  

  * 발기장 : ↓↘→ + A or C  

  * 전신상 : →↓↘ + A or C  

  * 쌍장진 : ↓↙← + A or C  
┗ 브레이킹 : AB

  * 호미각 : ↓↙← + B or D  
┗ 브레이킹 : 히트 중 AB

  * 연격축 : 공중에서 ↓↙← + B or D  

  * 사상퇴 : ←↙↓↘→ + B or D (잡기)  
┗ 브레이킹 : 잡은 후 AB

  * 연운 십육수의 자세 : ↓ + AC  
┣ 단장진 : A (자세 해제)  
┣ 연운 하조 : B  
┣ 연합 쌍장타 : C, A (자세 해제)  
┣ 유연선각 : D (자세 해제)  
┣ 선회연 : ↑ + B or D (자세 해제)  
┣ 전소 연무 : ↓ + D, → + B (↓ + D까지 사용시 자세 해제)  
┗ 자세 해제 : ←←  
┣ 이연의 자세 : ← + BD  
┃ ┣ 연룡반타 : A (연운 십육수의 자세로 이행)  
┃ ┣ 연인각 : B (자세 해제)  
┃ ┣ 배격수 : C  
┃ ┣ 연미각 : D (자세 해제)  
┃ ┃ ┗ 브레이킹 : 히트 중 AB  
┃ ┣ 배신 천락투 : ↓↘→ + A or C (잡기, 자세 해제)  
┃ ┣ 이연 유무 : ↓↘→ + B or D (자세 해제)  
┗ ┗ 자세 해제 : ←←  

★ 초필살기  

  * 쌍장 천련화 : ↓↘→↓↘→ + A or C (게이지 2개 소비)  

  * 천상 난희 : ↓↘→↓↘→ + B or D  

#### 3 [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * 천락투 : 근접해서 ← or → + C or D  

  * 반전투 : 공중 근접해서 ← or → + C or D  

★ 특수기  

  * 천격축 : → + B (노 캔슬 사용 후 연운 십육수의 자세)  

  * 선격수 : ↘ + C (사용 후 이연의 자세)  

  * 고축타 : ↓ + B  

★ 필살기  

  * 발기장 : ↓↘→ + A or C  

  * 전신상 : →↓↘ + A or C  

  * 쌍장진 : ↓↙← + A or C  
┗ 브레이킹 : AB

  * 호미각 : ↓↙← + B or D  
┗ 브레이킹 : 히트 중 AB

  * 연격축 : 공중에서 ↓↙← + B or D  

  * 사상퇴 : ←↙↓↘→ + B or D (잡기)  
┗ 브레이킹 : 잡은 후 AB

  * 연운 십육수의 자세 : ↓↓ + A  
┣ 단장진 : A (자세 해제)  
┣ 연운 하조 : B  
┣ 연합 쌍장타 : C, A (자세 해제)  
┣ 유연선각 : D (자세 해제)  
┣ 선회연 : ↑ + B or D (자세 해제)  
┣ 전소 연무 : ↓ + D, → + B (↓ + D까지 사용시 자세 해제)  
┗ 자세 해제 : ←←

  * 이연의 자세 : ↓↓ + B  
┣ 연룡반타 : A (연운 십육수의 자세로 이행)  
┃ ┗ 브레이킹 : 히트 중 AB  
┣ 연인각 : B (자세 해제)  
┣ 배격수 : C  
┣ 연미각 : D (자세 해제)  
┃ ┗ 브레이킹 : 히트 중 AB  
┣ 배신 천락투 : ↓↘→ + A or C (잡기, 자세 해제)  
┣ 이연 유무 : ↓↘→ + B or D (자세 해제)  
┗ 자세 해제 : ←←  

★ 초필살기  

  * 쌍장 천련화 : ↓↘→↓↘→ + A or C  

★ 리더 초필살기  

  * 천상 난희 : ↓↘→↓↘→ + E
