Code Name: Atlantis  
Tech Level: Secret (비밀)  
Classification(분류) : Military Sub

[Supreme Commander](Supreme%20Commander.md)의 유닛 이름. 아틀란티스는 [United EarthFederations](United%20Earth%20Federations.md)에만 있는 T4 [잠수항모](%EC%9E%A0%EC%88%98%20%ED%95%AD%EB%AA%A8.md)이며 다른 진영의 T3 항공모함의 3배인
150대까지 수송할 수 있지만, 대신 다른 진영에서 가지고있는 T3
[항공모함](%ED%95%AD%EA%B3%B5%EB%AA%A8%ED%95%A8.md)을 가질수 없다. 부상시 3티어 대공포탑과 맞먹는
대공미사일과, 약 700 가량의 데미지를 주는 어뢰로 무장하고 있기 때문에 한두기 날아다기는 공중유닛이나, 티어 1 따위의 대함 잠수함
따위는 일도 아니고, 4발씩 쏘는 통에 2~3티어 대잠 잠수함들도 다량으로 운용하고 있지 않다면 침수되기 일쑤다. 밑에는 수송가능 유닛들을
적어놨지만, 공군류고, 일단 지구연방군의 공군유닛은 모두 수송가능하고, 적의 유닛을 사용할 수 있게 된 경우 [CybranNations](Cybran%20Nations.md)의 Experimental Gunship: Soul Ripper 와 [AeonIlluminate](Aeon%20Illuminate.md)의 Flying Fortress: CZAR는 수송이 불가능하다. (CZAR은
계속 공중에 떠있는다)
