## 2. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

★ 드라이브 - 소드 서머너 改(ソードサマナー改 / Sword Summoner ΙΙ) : 공간으로 검을 소환해 공격합니다.  
★ 오버 드라이브 - 엘리시움(エリュシオン)

  

★ 필살기  

  * 스파이크 체이서(スパイクチェイサー / Spike Chaser) : 땅 위에서 검을 소환해 공격합니다. 기술 사용 후에 C를 누르면 비거리가 증가합니다.
  * 시클 스톰(シクルストーム / Sickle Storm) : 땅에서 회전하는 검을 소환해 공격합니다. 기술 다음에 C를 누르면 나타나는 위치가 달라집니다.
  * 크레센트 세이버(クレセントセイバー / Crescent Saber) : 상대를 공격합니다. 기술 사용 후에 C를 누르면 공격을 중단합니다.
  * 그래비티 시드(グラビティシード / Gravity Shield) : 상대의 이동 속도를 늦추는 중력장을 만듭니다. 버튼에 따라 중력장 출현 위치가 달라집니다.
  * 액트 펄서 Zwei(アクトパルサーツヴァイ / Act Parser Zwei) : 정면으로 고속 대시합니다.
  * 액트 펄서 Zwei 블레이드(アクトパルサーZwei・ブレイド / Act Parser Zwei: Blade) : 정면으로 고속 대시한 다음 발밑을 공격합니다.
  * 액트 펄서 Zwei 기병대(アクトパルサーZwei・キャバリエ / Act Parser Zwei: Cavalier) : 정면으로 고속 대시한 다음 위쪽으로 공격합니다.
  * 엑시거 네일(エクシーガネイル) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 레거시 엣지(レガシーエッジ / Legacy Edge) : 앞쪽에 여러 자루의 검을 소환해 공격합니다.
  * 캘러미티 소드(カラミティソード / Calamity Sword) : 앞쪽에 거대한 검을 소환해 공격합니다.  

★ 아스트랄 히트  

  * 멸망의 검(滅びの剣 / Sword of Destruction) : 강력한 잡기 기술입니다.
