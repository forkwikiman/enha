## 1. Blazblue : Continuum Shift ¶

★ 드라이브  
디토네이터 : 고유 게이지를 소비해 이그니스를 소환한다. 이그니스는 항상 레리우스 옆에 붙어 다닌다. 이그니스를 이용해 공격하면 고유
게이지가 소비되고, 이그니스 소환을 해제하면 고유 게이지가 원래대로 천천히 회복된다. 하지만, 이그니스가 나와있을 때 고유 게이지가 0이
되면 이그니스가 기능정지 되어서 고유 게이지가 전부 회복될 때까지 쓸 수 없게 된다.

  

레리우스의 각종 D기술은 이그니스의 고유 게이지가 다 닳으면 사용할 수 없다.

  

☆ :레리우스가 별도의 모션을 취하지 않는 드라이브  

  * D (이그니스를 소환 및 소환 해제)  

    * → + D 상단, ☆
    * ↓ + D 상단 2회, ☆
    * ← + D 공중, ☆, ♢
  * 공중에서 ↑ + D, → + D, ↓ + D 각각 위, 앞, 아래로 이그니스가 공격한다. 공중 대시중에도 사용 가능하다.  

★ 통상기  

  * A : 상  

    * ↓ + A : 하단
    * → + A : 중단, ♢
    * 공중 : 상단
  * B : 상, ♢  

    * ↓ + B : 하단, ♢
    * → + B : 상단, ♢, 띄움
    * 공중 : 중
  * C : 상 2회  

    * ↓ + C : 상단, ♢
    * → + C : 상단, ♢
    * ↘ + C : 하단, 다운, ♢
    * 공중 : 중단  

★ 필살기  

  * 이드 로이거 : ↓↘→ + C : 상단  
┗하스 : 이드 로이거 명중 후 ↓↙← + A  
┗나이아 : 공중에서 이드 로이거 명중 후 ↓↙← + B

  * 레드 리 : ↓↘→ + A : 회피 기술. 날리는 공격은 물론, 근접 공격도 피할 수 있다.
  * 발 랜토 : ↓↙← + A : 상단, 사용후 일반 드라이브로 연계 가능.
  * 발 라이아 :↓↙← + B : 상단, 경직
  * 발 튜스 : ↓↙← + C : 돌진 후 3회 공격 첫타 ♠.
  * 기라 노즈 : ↓↙← + D : 중단, ♠
  * 기라 액트 : ↓↘→ + D : 하단, ♠
  * 벨 라퓌노 : 공중에서 ↓↙← + B : 상단, 막타 ♠  

★ 디스토션 드라이브 (히트 게이지 50% 소비)  

  * 레크 뷔놈 : →↘↓↙←→ + C 상단 (홀딩기, 데미지는 그렇지만 달려오는적한테 좋은기술)
  * 볼 테도 : →↘↓↙←→ + D 상단 (이그니스가 소환되어서 타격시 구체에가두고 날려버린다. 하지만 잡기 판정이 아니라 가드 가능)
  * 듀오 봐이오스 : ↓↘→↓↘→ + D 상단 (이그니스가 전진하면서 공격한다. 레리우스는 움직일 수 있어서 협공으로 가드를 깨기 편하다.)
  * ???? : ↓↙←↙↓↘→ + C (컨티뉴엄 시프트 익스텐드, 언리미티드 한정) `[1]`  

★ 아스트랄 히트 (히트 게이지 100% 소비)  

  * 얼터 오브 디 퍼펫 : ↓↙←↓↙← + D 상단 (캐릭터마다 연출씬이 다르다)

  * 기라 노즈 : ↓↙← + D

## 3. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

★ 드라이브 - 디토네이터(デトネーター / Detonator) : 이그니스를 소환해서 전투에 투입합니다.  
★ 오버 드라이브 - 맥심 댄스(マキシムダンス / Maximum Dance)

  

★ 필살기  

  * 이드 로이거(イド・ロイガー / Id Lauger) : 거대한 기계 손으로 상대를 세게 때립니다.  

┗ 이드 하스(イド・ハース / Id Haas) : 이그니스가 돌진 공격을 합니다.  
┗ 이드 나이아(イド・ナイア / Id Naiads) : 이그니스가 드릴 형태가 되어 상대를 관통합니다.

  * 레드 리(レド・リー / Led Ley) : 제자리에서 회전해 적의 공격을 피합니다.
  * 발 란트(バル・ラント / Val Lanto) : 이그니스가 돌진 공격을 합니다.
  * 발 라리어(バル・ライア / Val Lyra) : 이그니스가 상대의 등뒤로 이동해 공격합니다.
  * 발 튜스(バル・テュス / Val Tus) : 이그니스가 전진하면서 연속 공격을 합니다.
  * 기라 노즈(ギラ・ノーズ / Geara Nose) : 이그니스가 중단 공격을 합니다.
  * 기라 액트(ギラ・アクト / Geara Act) : 이그니스가 하단으로 장풍을 날립니다.
  * 벨 라피노(ベル・ラフューノ / Bel Lafino) : 이그니스가 드릴 형태가 되어 상대를 관통합니다.
  * 가드 레이스(ガド・レイス / Gaad Leis) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)
  * 기라 루기아(ギラ・ルギア / Geara Lugia) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 레크 비놈(レク・ヴィノム / Req Vinum) : 상대를 포박하는 필드를 펼쳐 공격합니다.
  * 볼 테드(ボル・テード / Vol Tedo) : 이그니스가 상대에게 강력한 일격을 날립니다.
  * 듀오 바이오스(ヂュオ・ヴァイオス / Duo Bios) : 이그니스가 연속으로 공격합니다.  

★ 아스트랄 히트  

  * 얼터 오브 더 퍼핏(アルター・オブ・ジ・パペット / The Puppeteer's Altar) : 이그니스가 상대를 붙잡고 레리우스의 고유 공간으로 끌어들입니다.
