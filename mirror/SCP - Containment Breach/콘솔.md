## 1. 개요 ¶

[SCP - Containment Breach](SCP%20-%20Containment%20Breach.md)에 있는 콘솔 명령어를
설명하는 페이지. [공식 위키 콘솔 항목](http://containmentbreach.wikia.com/wiki/Console)을 참고.

  

콘솔창을 출력하려면 게임 내에서 F3을 누르면 된다.

## 2. 기본 명령어 ¶

  * 096state : [SCP-096](SCP-096.md)의 위치 좌표를 출력한다.
  * 106state : [SCP-106](SCP-106.md)의 위치 좌표를 출력한다.
  * 173state : [SCP-173](SCP-173.md)의 위치 좌표를 출력한다.
  * 106speed 숫자 : 숫자 만큼 [SCP-106](SCP-106.md)의 속도를 조절한다. 여담으로 마이너스로 설정하면 뒷걸음친다(...).
  * 173speed 숫자 : 숫자 만큼 [SCP-173](SCP-173.md)의 속도를 조절한다.
  * asd : 디버깅을 목적으로하는 콘솔 명령어. 화면에 와이어 프레임이 나타나고 자동으로 노클립, 갓모드가 되고 카메라 포그 15 20 으로 설정된다.
  * brightness 숫자 : 밝기를 설정
  * camerafog 숫자 숫자 : 카메라 포그, 말그대로 안개를 설정한다. 숫가자 짧으면 짧을 수록 어두움이 더 강해져 시야가 짧아지고 넓으면 넓을 수록 더 밝게 보인다. ex : camerafog 100 100
  * camerapick : 가리키고 있는 텍스쳐의 모델 이름과 좌표를 출력한다.
  * debughud 명령문 : 플레이어, SCP-173, SCP-106, SCP-096 의 위치를 출력한다. 명령문은 "on", "1", "true"중 하나를 입력하면 debughud가 켜지고, "off", "0", "false"중 하나를 입력하면 꺼진다.
  * enable173 : SCP-173을 활성화 시킨다.
  * disable173 : SCP-173을 비활성화 시킨다.
  * enable106 : SCP-106을 활성화 시킨다.
  * disable106 : SCP-106을 비활성화 시킨다.
  * ending 코드 : 해당 [엔딩 코드](SCP%20-%20Containment%20Breach/%EC%BD%98%EC%86%94#s-5.md)에 맞는 엔딩을 실행한다. 
  * godmode 명령문 : 갓 모드(무적 모드)가 된다. 다만 [SCP-106](SCP-106.md)이 주머니 차원으로 끌고 가는 것은 그대로 당하고, [SCP-914](SCP-914.md)에서 매우 고움으로 돌려서 나온 건전지를 만지는 등의 특수한 상황에서는 죽을 수 있다.
  * halloween : [SCP-173](SCP-173.md)을 [잭 오 랜턴](%EC%9E%AD%20%EC%98%A4%20%EB%9E%9C%ED%84%B4.md) 스킨으로 바꾸는 이스터 에그, 과거 할로윈 업데이트때 추가된 이스터 에그다.
  * heal : 모든 부상과 출혈 상태를 제거하고 [SCP-008](SCP-008.md) 감염에 효과를 받지 않는다.
  * infect 숫자 : 숫자는 0 - 91 까지, [SCP-008](SCP-008.md) 에게 감염되는 정도를 설정한다. 이 이외의 숫자를 입력하면 높은 확률로 메모리 접근 위반(Memory access violation)이 뜨면서 게임이 강제종료 될 수 있다.
  * injure 숫자 : 숫자는 1 - 5 까지, 플레이어의 부상을 설정한다.
  * noclip 명령문 : 노클립 상태를 지정하는 명령문. 벽을 뚫고 다니며 날아다닐 수 있다.
  * noclipspeed 숫자 : 노클립 상태의 스피드를 설정한다.
  * reset096 : SCP-096의 추적 기능을 다시 리셋 시킨다.
  * sanic : GOTTA FO FAST!라는 메시지를 출력하며 플레이어 속도를 올린다. 카메라가 흔들리며, godmode가 켜져있지 않으면 얼마 뒤에 죽는다. [SCP-914](SCP-914.md)에 플레이어를 넣고 고움이나 매우 고움으로 돌리면 나오는 상태와 같다.
  * spawn 코드 : [NPC 코드](SCP%20-%20Containment%20Breach/%EC%BD%98%EC%86%94#s-3.md)를 입력하면 플레이어 위에 NPC를 소환한다.
  * spawnitem 코드`[1]` : 아이템 코드를 입력하면 해당 좌표에 코드에 맞는 아이템을 소환한다. 다만, [SCP-914](SCP-914.md)에 안약이나 구급상자를 넣고 매우 고움으로 돌려서 나오는 안약은 이 명령어로 생성할 수 없다.
  * status : 플레이어, 카메라, 방 정보를 출력한다.
  * teleport 코드 : [방 코드](SCP%20-%20Containment%20Breach/%EC%BD%98%EC%86%94#s-4.md)를 입력하면 현재 시드에 생성된 방으로 플레이어를 위치시킨다.
  * weed : scp-420-j나 420라고 쳐도 된다. 플레이어 주위에 20개의 [SCP-420-J](SCP-420-J.md)를 생성하며, 50%의 확률로 대마초가 나올수도있다.
  * wireframe 명령문 : 와이어프레임 모드로 전환하는 명령어이다.
