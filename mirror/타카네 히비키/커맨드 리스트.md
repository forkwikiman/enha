## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 기세를 살린다 : 근접해서 ← or → + CD  

★ 특수기  

  * 멎음을 잊지 않음은 중요하다 : 상대 다운시 ↘ + B
  * 준비를 게을리하지 않는 마음 : 스타트 버튼 지속
  * 종이 한 장 차이로 피한다 : AB  
┣ 거리를 좁히는 것은 중요하다 : 종이 한 장~ 중 → + C  
┗ 거리를 벌리는 것은 중요하다 : 종이 한 장~ 중 ← + C  

★ 필살기  

  * 원거리를 벤다 : ↓↘→ + A or B
  * 근거리를 벤다(●) : →↓↘ + B (모으기 가능)  
┣ 원거리를 벤다 : 근거리를 벤다 히트 중 ↓↘→ + A or B (기모드 한정)  
┗ 거합한다 : 근거리를 벤다 히트 중 ←↙↓↘→ + C (기모드 한정)

  * 수월을 뚫는다 : →↓↘ + C
  * 거합한다 : ←↙↓↘→ + C  
┣ 뽑으면 벤다 : 거합한다 히트 중 B  
┗ 뽑지 않으면 베지 않는다 : 거합한다 히트 중 BC  

★ 초오의  

  * 승리를 부르는 신기 : ↓↙←→ + AB  

★ 잠재오의  

  * 죽음을 두려워하지 않는 마음 : →←↙↓↘→ + B  

★ 난무오의  

  * 시체를 넘어간다 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, ↓↘→ + A  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, A, BC, A, ↓↘→ + B

## 2 [CAPCOM VS SNK 2](CAPCOM%20VS%20SNK%202.md) ¶

  

★ 기본잡기  

  * 기세를 살린다 : 근접해서 ← or → + 강P
  * 베어 쓰러뜨린다 : 근접해서 ← or → + 강K  

★ 특수기  

  * 후의 선을 취한다 : 근접 강P 지속   

★ 필살기  

  * 원거리를 벤다 : ↓↘→ + P
  * 근거리를 벤다 : ↓↙← + P (모으기 가능)
  * 종이 한 장 차이로 피한다 : KK  
┣ 거리를 좁히는 것은 중요하다 : 종이 한 장~ 중 → + K  
┗ 거리를 벌리는 것은 중요하다 : 종이 한 장~ 중 ← + K

  * 수월을 뚫는다 : →↓↘ + K
  * 거합한다 : ←↙↓↘→ + K  

★ 슈퍼콤보  

  * 승리를 부르는 신기 : ↓↙←↙→ + P
  * 죽음을 두려워하지 않는 마음 : →←↙↓↘→ + P (레벨 3 or 게이지 MAX 전용)
  * 시체를 넘어간다 : →↘↓↙←→ + K (레벨 3 or 게이지 MAX 전용)   
┣ 루트 1 : 히트 후 약P, 중P, 강P, 약P, 중P, 강P, 약P, 중P, ↓↘→ + 강P  
┣ 루트 2 : 히트 후 약P, 중P, 강P, 약P, 중P, 강P, 약K, 중K, ↓↘→ + 강K  
┗ 루트 3 : 히트 후 약P, 중P, 강P, 약K, 중K, 약K, 강K
