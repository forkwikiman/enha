## 1. 설명 ¶

2011년 9월 16일에 발매된, 배틀 스피리츠 히어로즈의 스타터덱. 주인공 히노보리 하지메가 초반에 사용하는 덱을 그대로 재현했다. 덱
자체로서의 밸런스가 굉장히 좋으며, 처음 등장하는 키워드 효과 '버스트'의 사용을 중점으로 둔 덱이다. 바신 단 덱 '태양의 브레이브
드래곤'에 밀리지 않는 강함을 자랑한다.

  

[대한민국](%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD.md)에는 2012년 5월 1일부터 판매에 들어갔으며,
명칭은 최강선 스타터덱 1탄. <del>그런데 히어로즈의 캐릭터 스타터는 이게 끝이다(…)</del>

  * 이카즈치 블룸　R　(1)  
스피릿 / 6(2) / 적 / 고룡 / 적  
1<1>4000 - 2<3>7000  
레벨1-레벨2 "이 스피릿의 소환시"  
상대의 브레이브/넥서스, 어느쪽인가 1개를 파괴한다.  
  
레벨1-레벨3 "이 스피릿의 어택시"  
자신의 버스트가 세트되어 있을 때, 상대의 스피릿 1체를 지정하여, 그 스피릿에게 어택할 수 있다.

1<1>4000 - 2<3>6000 - 3<5>9000

### 2.2. 넥서스 ¶

  * 영웅황의 신검　(3)  
넥서스 / 3(2) / 적 / 적  
1<0> \- 2<2>  
레벨1-레벨2  
자신의 버스트를 세트했을 때, 자신의 덱에서 1장 드로우 한다. 이 효과는 1턴에 1회밖에 사용할 수 없다.  
  
레벨2 "자신의 어택 스텝"  
합체(브레이브)하지 않은 계통:"패황"을 지닌 자신의 스피릿 전부를 BP+3000한다. 자신의 버스트가 세트되어 있는 동안, 추가로
합체(브레이브)하지 않은 계통:"패황"을 지닌 자신의 스피릿 전부를 BP+5000한다.  

  * 영웅황의 어순　(3)  
넥서스 / 3(1) / 백 / 백  
1<0> \- 2<3>  
레벨1-레벨2 "서로의 어택 스텝"  
스텝 개시시, 상대의 합체(브레이브) 스피릿 1체를 지정한다. 이 턴 동안, 지정된 합체(브레이브) 스피릿은 배틀할 수 없다.  
  
레벨2  
자신의 버스트를 세트하고 있는 동안, 카드명에 "영웅황"이 들어간 자신의 넥서스 전부는 파괴되지 않는다.

### 2.3. 매직 ¶

  * 쌍익난무　(3)  
매직 / 4(2) / 적  
버스트 : 상대의 "이 스피릿/브레이브의 소환시" 발휘후  
자신은 덱에서 2장 드로우 한다. 그후 코스트를 지불하는 것으로, 이 카드의 메인 효과를 발휘한다.  
  
메인 : 자신은 덱에서 2장 드로우 한다.  

  * 폭렬십문인　R　(1)  
매직 / 5(3) / 적  
버스트 : 상대의 "이 스피릿/브레이브의 소환시" 발휘후  
BP6000이하의 상대 스피릿 1체와, 상대의 합체(브레이브) 스피릿의 브레이브 1개와, 상대의 넥서스 1개를 파괴한다. 그후 코스트를
지불하는 것으로, 이 카드의 메인 효과를 발휘한다.  
  
메인 : 자신의 트래쉬에 있는 버스트 효과를 지닌 카드 1장을 패로 되돌린다.  

  * 패왕폭염격　U　(2)  
매직 / 6(3) / 적  
버스트 : 자신의 라이프 감소후  
BP4000이하의 상대 스피릿 3체를 파괴한다. 그후 코스트를 지불하는 것으로, 이 카드의 플래쉬 효과를 발휘한다.  
  
플래쉬 : 상대의 합체(브레이브) 스피릿 1체를 파괴한다.  

  * [절갑빙순](%EC%A0%88%EA%B0%91%EB%B9%99%EC%88%9C.md)　U　(2)  
매직 / 4(1) / 백

플래쉬 : 이 배틀이 종료했을 때, 어택 스텝을 종료한다.  

  * 갑룡봉절파　U　(2)  
매직 / 5(3) / 백  
버스트 : 상대의 "이 스피릿/브레이브의 소환시" 발휘후

플래쉬 : 자신의 스피릿 1체를 회복시킨다.
