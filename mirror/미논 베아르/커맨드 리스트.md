# 1. [KOF MI](KOF%20MI.md) ¶

  

★ 기본잡기  

  * 필리 스루: 근접해서 ← or → + C
  * 포니 스루: 근접해서 ← or → + D  

★ 특수기  

  * 원 레볼루션: →→ + D (연타 가능)  

★ 필살기  

  * 파이어 볼: ↓↘→ + A or C  
┗ 파이어 샷: ↓↙← + A or C

  * 윈드 스톰: →↘↓↙← + A or C
  * 워터 실드: →↓↘ + A or C
  * 라이트닝 볼트: →↘↓↙← + D (잡기)  

★ 초필살기 (☆은 기게이지 소모 개수 표시)  

  * 메테오 스트라이크(☆): ↓↘→↓↘→ + A or C  
┗ 익스플로전(☆): ↓↙← + C

  * 선더 볼트(☆☆): ↓↙←↓↙← + B or D (잡기)  

★ 스타일리쉬 아트  

  * 01: A ▶ → + A ▶ → + C ▶ D
  * 02: A ▶ → + A ▶ → + C ▶ B ▶ D
  * 03: B ▶ B ▶ B
  * 04: B ▶ B ▶ D
  * 05: → + B ▶ → + D
  * 06: → + B ▶ ↓ + D ▶ D
  * 07: C ▶ D
  * 08: C ▶ C ▶ C
  * 09: C ▶ C ▶ ↑ + D
  * 10: C ▶ ← + C ▶ C
  * 11: C ▶ → + C ▶ ↓ + D
  * 12: ↘ + C ▶ D ▶ D
  * 13: → + B ▶ ↓ + D ▶ ↓ + CD (키튼 스타일로 이행)
  * 14: C ▶ C ▶ ↓ + CD (키튼 스타일로 이행)
  * 15: C ▶ → + C ▶ ↓ + CD (키튼 스타일로 이행)
  * 16: ↘ + C ▶ D ▶ ↓ + D ▶ ↓ + CD (키튼 스타일로 이행)
  * 17: 키튼 스타일 중 →
  * 18: 키튼 스타일 중 ←
  * 19: 키튼 스타일 중 ↑ (키튼 스타일 해제)
  * 20: 키튼 스타일 중 D (키튼 스타일 해제)
  * 21: 키튼 스타일 중 B ▶ B (키튼 스타일 해제)
  * 22: 키튼 스타일 중 A ▶ C (키튼 스타일 해제)
  * 23: 키튼 스타일 중 A ▶ D (키튼 스타일 해제)
  * 24: 키튼 스타일 중 C ▶ C ▶ C (키튼 스타일 해제)
