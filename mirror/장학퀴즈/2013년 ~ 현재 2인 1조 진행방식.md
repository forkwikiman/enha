## 1. 2013 ~ 2015년 2인 1조 팀별 퀴즈 진행방식 - 838회 ~ 현재 ¶

  * 개인전에서 2인 1조 연승제형식으로 진행되며, 주제형 퀴즈 형식에서 종합형 퀴즈 형식으로 바뀐다.
  * 865회부터는 챔피언 팀과 도전자 3팀이 1라운드부터 대결을 펼친다.`[1]`
  * 869회부터는 팀명이 학교이름으로 변경되며, 같은 학교 학생이 아니면 참가할 수 없다.
  * 2014년에는 전국을 7개 권역으로 나누어 각 지역의 도전팀들이 챔피언에게 맞서는 방식으로 진행된다.
  * 2015년에는 다시 도전자들의 출신 지역이 자유로워지고, 챔피언을 포함한 5팀이 대결하는 형식으로 진행된다.
  * 왕중왕전은 역대 2인 1조 대결에서 다승을 거둔 8~10팀이 1조에 4~5팀씩 대결을 해서 각 조 1, 2위에게는 제왕전 진출권이 주어진다. 형식은 일반 진행 형식과 동일하며, 2013~2014 왕중왕전 예선전 파이널 라운드는 50점이 주어진 연상문제 5문제를 4팀이 버저형식으로 푼다.`[2]`

#### 3.2.1. 군계일학 객관식 퀴즈 ¶

  * 삼지선다 객관식으로 교과형과 창의력 퀴즈로 구성된 5문제`[3]`가 출제되며 정답팀의 수에 따라 점수가 차등 배점된다.(10~40점)
  * 879회부터는 10문제로 늘어났다.

#### 3.2.2. 상부상조 1+1 퀴즈`[4]` ¶

  * 기본 문제와 보너스 문제로 구성된 문제가 출제된다. 
  * 버저를 누른 팀에게 우선권이 주어지며 기본 문제를 맞히면 보너스 문제 우선권을 획득한다.
  * 단, 이 때 팀원이 번갈아 가면서 맞혀야 함.
  * 838회와 839회는 20점 문제 9문제, 40점 문제 1문제로 구성된다.(기본문제, 보너스 문제 포함)
  * 840회~878회는 20점 문제 1문제, 40점 문제 3문제, 60점 문제 1문제로 구성된다.(기본문제, 보너스 문제 포함)`[5]`
  * 879회~889회는 40점 문제 3문제, 60점 문제 2문제로 구성된다.(기본문제, 보너스 문제 포함)
  * 중간에 상품이 걸려 있는 문제는 기본문제를 맞히는 팀에게 증정된다.

#### 3.2.3. 박학다식 필기형 퀴즈 ¶

  * 순서, 연산, 선택 등 통합적 지식 테스트로 5문제 출제(50점, 마지막 문제는 100점)
  * 838회~846회는 5문제, 847회~880회는 3문제가 출제된다.`[6]`
  * 838회~878회의 **1차 문제 유형** : 5개의 보기 중 정답 2개 고르기(50점, 절반 정답시 30점), 4개 보기 순서대로 나열하기(50점, 부분점수 없음), 4개 보기에 들어갈 숫자의 합 맞히기(100점, 정답팀이 없을 시 정답에 가까운 순서대로 80, 60, 40, 20점 부여`[7]`) 
  * 879회부터 적용된 **2차 문제 유형** : 3개의 빈칸 단어 맞히기(2문제-100점, 2개 정답시 60점, 1개 정답시 30점 부여), 4개 보기에 들어갈 숫자의 합 맞히기`[8]`
  * 문제를 다 푼 다음에는 1위 팀(864회부터는 상위 2팀 진출)만 결승에 진출한다.`[9]``[10]`

### 3.3. 3라운드(파이널 라운드) ¶

  * 파이널에 올라온 두 팀의 대결. 세 개의 단어를 듣고 푸는 연상 퀴즈로 문제 선택 우선권은 연승 중인 팀이 갖는다.`[11]`
  * 865회부터는 예선전 1위 팀이 문제 선택 우선권을 갖는다(동점일 경우에는 챔피언이 우선권을 가짐).
  * 총 9문제, 3x3 빙고판으로 진행된다.
  * 문제를 맞힌 팀이 다음 문제를 선택할 권한이 주어지며, 만약에 두 팀 모두 답을 맞히지 못한 경우에는 이전 문제를 맞힌 팀(이전 선택팀)이 문제를 선택한다.
  * 먼저 빙고를 완성하는 팀이 우승하며, 두 팀 모두 빙고를 완성하지 못한 경우 문제를 많이 맞힌 팀`[12]`이 우승한다.`[13]`

### 4.2. 2라운드 ¶

  * 1라운드 1위부터 찬스를 선택할 수 있는 기회가 주어진다.

  * 896회부터는 20문제가 주어지며, 904회부터는 한 팀당 한 문제에 한 번의 기회만 주어진다.

#### 4.2.1. 찬스 유형 ¶

  * X2 : 말 그대로 문제를 맞히면 배점 점수에 2배가 주어진다.
  * 긴가민가 : 한 팀이 정답을 최대 2개 선택할 수 있다.`[14]`
  * 독점 찬스 : 사용한 한 팀만 답을 말할 수 있으며, 다른 팀들에게는 기회가 아예 주어지지 않는다.

  * 추가 점수가 걸려있는 한국사 문제는 1라운드에선 주어지지 않으나, 일반 한국사 문제는 출제될 수 있다.

### 5.3. 3라운드(890회~942회) ¶

  * 종합형 퀴즈인 멀티미디어 퀴즈 5문제는 각 30점이 부여된다.
  * 문제 선택은 점수가 높은 팀부터 선택하며, 만약에 모두 답을 맞히지 못한 경우에는 이전 정답팀(선택팀)에게 선택권이 부여된다.
  * 공통쓰기 문제`[16]`는 최소 0점에서 최대 50점까지 점수를 걸수 있으며, 도중에 점수차가 많이 날 경우 남은 문제와 상관없이 [콜드게임](%EC%BD%9C%EB%93%9C%EA%B2%8C%EC%9E%84.md)으로 우승팀이 결정된다.`[17]`

#### 5.3.1. 공통사항 ¶

  * 890회부터는 중간에 출제되는 '자랑스러운 한국사' 문제`[18]`는 맞히면 문제배점에 10점이 더 가산되거나 상품을 획득한다.`[19]`
  * 890회~942회 방송분에서는 중간에 1위 팀과 200점 이상 차이 나는 팀은 자동 탈락되는 제도가 있었다.`[20]``[21]``[22]`
  * 890회부터는 한 문제 당 기회는 무제한으로 주어지고, 맞히지 못하면 10점 감점이 있었으며, 931회부터는 최종 문제를 제외한 모든 문제에 대한 감점 제도가 폐지됨과 동시에 팀 당 답할 수 있는 기회는 한 번으로 제한된다.

### 6.1. 1라운드 ¶

  * 931회부터 적용된 규칙으로, 모든 팀이 공통으로 4지선다 15문제를 풀며, 배점은 10점이 부여된다.
  * 943회부터는 혼자 정답을 맞힌 팀에게 20점이 주어지며, 최하위 팀을 제외한 나머지 4팀이 다음 라운드에 진출한다.
  * 이때 1라운드 점수는 다음 라운드로 누적되지 않는다.

### 6.2. 2라운드 ¶

  * 2팀씩 맞대결을 펼치며, 상대팀 선정은 1라운드 1위 팀에게 주어진다.
  * 각 대결마다 주관식 7문제를 버저형식으로 진행해서 각 대결 승자팀이 먼저 올라가고, 패배한 두 팀이 대결을 펼쳐 승리한 팀이 추가 진출한다.

### 6.3. 3라운드 ¶

  * 3팀에게 기본점수 300점이 주어지며, 3단계 연상퀴즈 5문제(도움말에 따라 30~10점 차등 배점)와 공통쓰기 1문제가 주어진다.
  * 문제 선택은 2라운드 1경기 승리팀부터 선택하며, 만약에 모두 답을 맞히지 못한 경우에는 이전 정답팀(선택팀)에게 선택권이 부여된다.
