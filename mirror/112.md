## 2. 국내 번호 ¶

해당 지역의 경찰서`[1]`에 국번없이 연결해주는
[전국연결번호](%EC%A0%84%EA%B5%AD%EC%97%B0%EA%B2%B0%EB%B2%88%ED%98%B8.md). 이곳에
전화혹은 문자`[2]`로써 각종 범죄신고를 할 수 있다.
[장난전화](%EC%9E%A5%EB%82%9C%EC%A0%84%ED%99%94.md)해선 안되는 [양대산맥](119.md) 중
하나.`[3]`  
그리고 이건 **확실히 하지 말아야 할 금기사항**인데, 112에서 문자로 법이나 처벌에 관련하여 질문을 할 수는 있다. (자주하면 다른
번호를 알려주며 그 다른 번호에 전화를 걸어 물어보라고 한다.) 근데 만약 [어른의사정](%EC%96%B4%EB%A5%B8%EC%9D%98%20%EC%82%AC%EC%A0%95.md)이 개입된 사건에 대해 함부로
질문을 하거나 명령하면 5분 이내에 112에서 전화가 온다. 그러니 애초에 112는 범죄신고때만 이용하자. 민원상담 회선이182로 따로
존재하니, 민원상담을 비롯한 비긴급신고는 182로 전화하자. 가끔 112로 전화해서 궁금한거 물어보는 사람이 있다. 생각보다 많다.

  

우리나라나 일본 뿐만 아니라, 주로 유럽이나 영연방 국가에서도 긴급전화 번호로 사용된다. 달리 말 하면, 각 국가마다 경찰서 전화번호가
다르므로, 여행이나 출장가는 해당 국가의 경찰서, 응급신고 번호 등은 미리 숙지하고 있어야 한다. 말이 통할지 어떨지는 나중 문제지만...

  

위치추적 시스템이 119에만 이루어진다고 119에 신고하라는 이야기가 있는데, 이는 사실이 아니다. 2012년 11월 이후로 112신고도
신고자의 동의없이 자동적으로 위치추적 시스템이 사용된다. 안심하고 신고하자.

### 2.1. 신고 ¶

[[edit](http://rigvedawiki.net/r1/wiki.php/112?action=edit&section=4)]

#### 2.1.1. 순서 ¶

  * 신고 : 말그대로 신고자가 신고한다. 위치추적 시스템이 사용된다.
  * 접수 : 종합상황실에서 신고를 접수한다. 위치, 상황과 가해자에 대한 정보를 비롯해 사건처리에 필요한 정보를 물어본다.
  * 접수 내용 분석 : 접수된 내용을 분석하고 긴급성에 따라 등급을 부가한다.
  * 지령 : 경찰조치가 필요하다고 여겨질 때, 현장 경찰관에게 무선으로 지령을 내린다. 이 때 부상자가 있을 경우 지령과 동시에 119에 출동을 요청한다.
  * 현장대응 및 상황보고 : 현장에 도착한 경찰관은 초동조치`[4]`를 취한다.
  * 사건인계 후 종결 : 전문부서로 사건을 인계하고 이후 추가적인 조치 이후 사건이 종결된다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/112?action=edit&section=5)]

#### 2.1.2. 등급 ¶

신고에도 등급이 존재하는데, Code로 등급을 나눈다.  

  * Code 1 : 긴급출동. 다음의 사유로 최우선적으로 출동이 필요한 경우다 - 범죄로부터 인명, 신체, 재산 보호, 심각한 공공의 위험 제거 및 방지, 신속한 범인 검거
  * Code 2 : 비긴급출동. 현장조치의 필요성은 존재하나(다시말해 출동은 해야하나) Code 1에 준하지 않는 경우. Code 1을 비롯한 주요 경찰 업무에 지장을 주지않는 선에서 출동한다.
  * Code 3 : 비출동. 출동하지 않는 경우다. 주로 민원 상담이나 장난전화 등이 여기에 포함된다. 제발 신고하기 전에 딱봐도 Code 3처럼 보이는 신고는 112로 신고 하지 말자. 경찰 민원 상담은 해당 회선인 182로 전화하자.
