### 1.1. 컴퓨터에서의 물음표 ¶

**?**  
아스키 코드 63(16진법 3F)(유니코드 U+003F).

  

윈도우에서는 파일 이름에 사용할 수 없다. 파일 검색시 와일드카드 역할을 하는 문자로, *와 다른 점은 글자 수까지 맞춰서 찾는다는 것이다.
예를 들어, `DIR *.H?P` 이렇게 쓰면 H와 P 사이에 어떤 글자든 정확히 1글자가 들어가는 파일(예를 들어
[HWP](%EC%95%84%EB%9E%98%EC%95%84%20%ED%95%9C%EA%B8%80.md))을 찾아준다.  
[Mac OS X](Mac%20OS%20X.md)에서는 시스템이 윈도우와 다르기 때문에 파일명에 사용할 수 있다.

  

**？ (Fullwidth Question Mark)**  
유니코드 U+FF1F.

  

윈도우에서 파일 이름에 사용할 수 있다. 다른 글자. 위 물음표와 비교해서 이 물음표 주위는 여백이 좀 더 넓다. 프로그래밍이나 명령에서 별
기능이 없기 때문에, 물음표 모양의 그림을 가진 일반 문자라고 생각하면 편하다.

  

여담으로, 윈도우에서 사용 불가한 문자는 \ / : * ? " < > | 인데, 굳이 파일명에 적고 싶다면, \는 U+005C →
U+FF3C(역슬래시) U+FFE6(원)로, /는 U+002F → U+FF0F로, :는 U+003A → U+FF1A로, *는 U+002A →
U+FF0A로, ?는 U+003F → U+FF1F로, "는 U+0022 → U+FF02로, <는 U+003C → U+FF1C로, >는
U+003E → U+FF1E로, |는 U+007C → U+FF5C로, 그리고 공백은 U+0020 → U+FF00로 적어 그럴싸하게 바꿔줄 수
있다.

  

그 외의 예:  

  * 깨진 문자를 물음표로 표시하기도 한다.
  * [BASIC](BASIC.md)에서 행 첫 머리(행 번호 제외)에 물음표를 쓰면 [PRINT](PRINT.md)로 자동 변환된다.

### 1.2. 리그베다 위키에서의 물음표 ¶

[리그베다 위키](%EB%A6%AC%EA%B7%B8%EB%B2%A0%EB%8B%A4%20%EC%9C%84%ED%82%A4.md)는
[모니위키](%EB%AA%A8%EB%8B%88%EC%9C%84%ED%82%A4.md)를 사용하기 때문에, 링크할 항목 앞에 물음표를
쓰면 (예: ?****물음표) 항목을 링크해 주는 기능을 갖고 있었다. ?물음표 이런 식으로. 현재는 사용할 수 없으므로 보이는 대로 대괄호
문법으로 바꿔주자.  
따라서 직접적으로 물음표 자체를 링크하거나 검색하는 것은 일반적인 방법으로는 불가능하며, 한때 항목이 만들어지기도 했으나 기술적 문제를
이유로 삭제된 바 있다. 정 보고싶다면 검색창에 물음표를 **2번**입력해서 가보자. 1번 입력하면 검색창 옆의 버튼이 물음표가 되어서 검색
자체가 불가능하지만, 2번 입력하면 Go로 되돌아가기 때문에 검색이 **된다.**

  

다만 이것은 앞 말과 조사를 붙여 쓰는 [한국어](%ED%95%9C%EA%B5%AD%EC%96%B4.md)에는 그다지 적합하지 않다.
'?****물음표는'과 같이 쓸 경우 '물음표는'에 링크가 걸리지 '물음표'에만 링크가 걸리지는 않는다. '물음표'에만 링크를 걸고 싶을
경우 '?****물음표 는'과 같이 띄어 쓰는 수밖에 없는데, 이는 앞 말과 조사를 붙여 쓴다는 한국어 띄어쓰기 원칙에 어긋난다. 따라서
물음표보다는 대괄호(`[[ ]]`)를 사용하여 링크하는 것을 추천한다.

  

만약 리그베다 위키에서 물음표를 조사에 붙여 쓰고 싶다면 { 3번에 물음표를 넣고 } 3번으로 닫으면 된다. 예시 : `{{{?}}}`

  

리그베다 위키에서 항목명에 물음표가 포함된 문서가 몇몇 있는데, 링크를 걸어 다른 문서에서 클릭하여 들어갈 경우 물음표 뒤의 문자들이 모두
사라져 버린다. 이런 경우는 링크된 제목의 항목을 생성해서 원래의 항목으로 리다이렉트시켜 줘야 한다.

  

예: [아빠! 어디가?](%EC%95%84%EB%B9%A0%21%20%EC%96%B4%EB%94%94%EA%B0%80.md) → 클릭시
[아빠! 어디가](%EC%95%84%EB%B9%A0%21%20%EC%96%B4%EB%94%94%EA%B0%80.md)로 이동 →
[아빠! 어디가?](%EC%95%84%EB%B9%A0%21%20%EC%96%B4%EB%94%94%EA%B0%80.md)로 리다이렉트  
예: [진짜 화났다? 나 잘못한 거 없는데](http://rigvedawiki.net/r1/wiki.php/%EC%A7%84%EC%A7%9C
%20%ED%99%94%EB%82%AC%EB%8B%A4?%20%EB%82%98%20%EC%9E%98%EB%AA%BB%ED%95%9C%20%E
A%B1%B0%20%EC%97%86%EB%8A%94%EB%8D%B0) → 클릭시 [진짜화났다](%EC%A7%84%EC%A7%9C%20%ED%99%94%EB%82%AC%EB%8B%A4.md)로 이동 → [진짜 화났다? 나
잘못한 거 없는데](http://rigvedawiki.net/r1/wiki.php/%EC%A7%84%EC%A7%9C%20%ED%99%94%E
B%82%AC%EB%8B%A4?%20%EB%82%98%20%EC%9E%98%EB%AA%BB%ED%95%9C%20%EA%B1%B0%20%EC%
97%86%EB%8A%94%EB%8D%B0)로 리다이렉트

  

조금 번거롭지만 링크를 깔끔하게 걸고 싶을 경우 `[[아빠! 어디가%3F|아빠!어디가?]]`, `[[정말 화났어%3F 나 잘못한 거
없어|정말 화났어? 나 잘못한 거 없어]]`처럼 ?를 %3F로 바꾸어 적용할 수도 있다. 이 경우 리다이렉트를 이용하지 않아 역링크를 보기
쉬운 장점이 있다.

  

[엔하위키 미러](%EC%97%94%ED%95%98%EC%9C%84%ED%82%A4%20%EB%AF%B8%EB%9F%AC.md)에서는
기술적인 문제로 ?을 누르고 이동 버튼을 누르면 이 문서로 오지 않는다. 물음표라고 적고 이동하거나 **%3F**를 적고 이동 버튼을 눌러야
한다.`[2]`

### 1.3. 한국 방송에서의 물음표 ¶

만 유재석" 식으로. [방통위](%EB%B0%A9%ED%86%B5%EC%9C%84.md) 경고를 피하려는 속셈이지만 너무 뻔하고 시청자
