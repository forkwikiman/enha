## 1.1. 평가 ¶

**사기캐** 현재[갓일표](%EB%B0%95%EC%9D%BC%ED%91%9C%28%EB%A1%9C%EC%8A%A4%ED%8A%B8%EC%82%AC%EA%B0%80%29.md)가 래더를 지배하고있지만.박일표가 너프되면 최상의 성능을 자랑할것이라고 유저들이 입을모아 말하고있다,기본적인 사거리가 매우 길뿐더러 모든 공격중에 방향키2회입력으로 원하는방향으로 이동해 공격캔슬기인 착검을 사용할수있으며 착검을3회 발동시 다음지상공격이 적을 공중으로 띄우며 땅긁이가능한 낙화로 변경된다.여기까지만했었다면 성능이 준수한편의 개념캐였겠지만 다크나이트의 진짜 사기적인면모를 보여주는 점공3타는 그냥 휘두르기만해도 맞는 괴랄한 판정범위를 지니고있으며 기본3타째에는 착검을 사용할수없지만 점공은 3타를 다때리고도 착검이 발동가능하다!이외에도 각종 바운드기술과 낙화를 조합하여 반무콤을 가지고있는것은 덤이고 거기에 각성상태때도 반무콤이 있는것역시 덤이다.하지만 무시못할 단점도있는데,각성상태때는 무기스킬,패시브스킬을 제외한 모든 스킬이 사용불가능하며 유미라의 차력상태등과 다르게 공격을받아 게이지가 0이되어도 자신이 완전히 기립하기전까진 각성상태가 풀리지않는다,쉽게말해 각성중에 콤보걸리면 골로간다는소리다.

# 2. 기본기 ¶

  * 기본으로100까지 차는 **광폭화**게이지가 찬다.적을공격하면5,적이공격을방어하면2가차며 공격을받을시2.공격방어시5가찬다.
  * DDD : 마검으로3번밴다.피격시 무꿇상태가되며,그상태에서 다시한번 평타를 피격시 멀리 날아가며 바운딩된다.
  * D꾹,대쉬D꾹 : 끌려오라<del>[오라](%EC%98%A4%EB%9D%BC%EC%98%A4%EB%9D%BC.md)</del> 말그대로 검으로 상대를 끌어온다. 공중에서나 무끓상태에서 피격시 바운딩이되며.그후평타3번으로 추가타를 집어넣을수있다.
  * 대쉬중DDD : 전진하며검으로 3번밴다.1타의 전진거리가 길어 선타잡기 편하다.3타피격시 평타와마찬가지로 무꿇상태가되며,그상태에서 대쉬공격을 할시 멀리 날아가며 바운딩된다.
  * ADDD : 공중에서 검으로3번밴다. 판정은 별로지만 리치가 상당하고 전진거리가 길다.착검으로 캔슬이된다.
  * AD꾹 : 공중에서 검으로 찍는다.다운판정을 가지고있으며,공중공격과 함께하면 무꿇상태로 만든다.넉백이 상당하여 쓸만하다.
  * 기본공격,대쉬공격,점프공격중 방향키2번입력 : **착검** <del>[?](%EB%82%AD%EC%9D%B8%28%EB%A1%9C%EC%8A%A4%ED%8A%B8%EC%82%AC%EA%B0%80%29.md)</del> **다크나이트 핵심기**방향키 입력한쪽으로 대쉬하며 공중공격중사용시 그즉시 내려오며,지상에서도 상당한 리치와더불어 정신나간 심리전을 가능캐해주는 기술이다.
  * 착검3회사용후 평타,대쉬공격 : **낙화**말그대로 검으로 찍어버린다.평타와 대쉬공격은 바뀌지만 공중공격은 그대로이므로 공중공격ㅡ착검ㅡ낙화로 다시 공격을 이어갈수있다.
  * 가드후공격 : 검으로 띄운다.공중공격으로 이어가기 편하다.
  * 게이지100 : **광폭화**[게이지100이되면 자동으로 변신이된다](%EB%82%98%EB%A5%B4%28%EB%A6%AC%EA%B7%B8%20%EC%98%A4%EB%B8%8C%20%EB%A0%88%EC%A0%84%EB%93%9C%29.md).기본기의 성능이 장난아니게 상향되지만.**용병교체가 불가능하며,무기스킬을 제외하고 사용이 불가능하다.**이떄문에 변신상태에서 선타잡히고 콤보먹이면 그대로 당해야한다. 말그대로 양날의검.이 단점이싫으면 게이지가 다차기전에 다른용병으로 교체하여 게이지를 날릴수있다.적공격시10씩 게이지를 사용하며,공격을받을경우15,공격을 막을경우10씩게이지가 떨어진다.`[1]` 참고로 패시브 사용 가능이다.
  * 광폭화후 낙화 : 총3번공격한다.1~2타는 무꿇,3타는 띄우기공격으로,1~2타만치고 캔슬이 가능하다.
  * 광폭화후 대쉬공격 : 전진거리가 대폭상향되며. 마지막타떄 상대를 스턴시킨다.대쉬공격으로선타잡고ㅡ상대반대방향으로 1~2타를 하다가 상대를 보며3타로 스턴-무한반복으로 간단히 반무를 넣을수있는 기술.
