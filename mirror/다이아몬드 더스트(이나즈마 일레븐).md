## 1. 개요 ¶

  

게임 2편 블리자드 버전의 숨겨진 히든 팀.
[가이아](%EB%8D%94%20%EC%A0%9C%EB%84%A4%EC%8B%9C%EC%8A%A4.md)와 어깨를 나란히 하는 마스터
랭크의 축구팀. 하지만 경쟁에서 밀려 '더 제네시스'의 칭호를 받는 데 실패하고 호시탐탐 칭호를 빼앗으려 하지만 끝내 얻지 못했다.
프로미넌스가 불꽃이라면 이쪽은 얼음. 그리고 선수들이 모두 風속성 아니면 山속성이다.

  

게임에서는 에일리아 학원의 본거지로 찾아가기 직전, 가젤이 혼자서 라이몬중으로 찾아오는데 여기서 팀의 존재가 처음으로 라이몬에게 알려진다.
더 제네시스의 칭호를 빼앗긴 가젤은 자기 팀이야말로 제네시스에 어울린다는 걸 증명하기 위해 단독으로 라이몬 일레븐을 습격하는데, 이때
[아후로 테루미](%EC%95%84%ED%9B%84%EB%A1%9C%20%ED%85%8C%EB%A3%A8%EB%AF%B8.md)가
라이몬의 원군으로 나타나 가젤과 1:1로 붙는다. 그러던 중 [그란](%EA%B7%B8%EB%9E%80.md)과 더 제네시스 멤버가
난입하고 결국 가젤은 어쩔 수 없이 물러난다.

  

이후 게임 클리어 데이터를 이어서 하면 가젤을 찾아서 팀과 싸울 수 있다. 스토리를 보면, 에일리아 학원은 이미 붕괴됐지만 아직도 자신들이
제네시스란 칭호에 어울리는 최강의 팀이란 걸 증명하기 위해 라이몬 일레븐과 시합을 벌인다는 내용. <del>이제와서 뒷북질.</del> 이긴
후에 엔딩을 다시 보면 진엔딩이 나온다. 그밖에도 블리자드 버전을 파이어 버전과 연동하면, [더카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)란 혼성팀과 시합할 수도 있다.

  

애니메이션에서는 [프로미넌스](%ED%94%84%EB%A1%9C%EB%AF%B8%EB%84%8C%EC%8A%A4%28%EC%9D%B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9D%BC%EB%A0%88%EB%B8%90%29.md)보다 더 많이 나온다.
대놓고 라이몬과 시합도 해서 2:2로 비겼을 정도. 이후 혼성팀 [더카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)라는 팀을 짜서 등장했다.

## 3. 선수 프로필 ¶

※ 프로필은 게임판을 기준으로 작성해주세요. 2편에선 다이아몬드 더스트 멤버 전원을 동료로 영입할 수 있다. 3편에서도 특정조건 만족시
2편과의 수퍼링크를 통해 가젤을 제외한 멤버 10명을 영입할 수 있다. [레제](%EB%A0%88%EC%A0%9C.md)나
[데자무](%EB%8D%B0%EC%9E%90%EB%AC%B4.md), [그란](%EA%B7%B8%EB%9E%80.md)과 달리,
가젤은 2편 버전과 3편 버전이 동일인 취급된다.

  

■ 가젤 / 스즈노 후우스케(涼野風介) - 포지션 FW(10) / 속성 風  
다이아몬드 더스트의 캡틴. 얼음을 연상시키는 창백한 머리카락을 하고있다. 고요함의 이면에는 냉철함을 숨기고 있다. 같은 마스터랭크팀
'가이아'의 캡틴인 [그란](%EA%B7%B8%EB%9E%80.md)을 싫어하며, 언젠가 그란으로부터 [더제네시스](%EB%8D%94%20%EC%A0%9C%EB%84%A4%EC%8B%9C%EC%8A%A4.md)의 칭호를 빼앗으려고 항상
벼르고 있다. 게임판에선 에일리아 학원 붕괴 후, 프로미넌스의 캡틴 번과 연합해 [더카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)란 팀을 결성하기도 한다. 3편에선 [아후로테루미](%EC%95%84%ED%9B%84%EB%A1%9C%20%ED%85%8C%EB%A3%A8%EB%AF%B8.md)에게 스카우트되어
한국 대표팀 [파이어드래곤](%ED%8C%8C%EC%9D%B4%EC%96%B4%20%EB%93%9C%EB%9E%98%EA%B3%A4.md)의 일원으로
등장한다.  
-자력기(2편) : 속성강화(skill) / 노던 임팩트(S,風) / 워터 베일(D,風) / 파이어 블리자드(S,風)  
-자력기(3편) : 속성강화(skill) / 워터 베일V2(D,風) / 노던 임팩트(S,風) / 파이어 블리자드(S,風)

  

■ 벨가 / 시라이 잇카쿠(白井一角) - 포지션 GK(1) / 속성 山  
장신에, 푸른 헤드밴드로 눈을 숨기고 있다. 푸른 피부가 특징. 평소엔 쿨하지만, 마음 속에는 뜨거움을 품고 있는 것 같다.  
-자력기(2편) : 아이스블록(C,風) / 싸이클론(B,風) / 토네이도 캐치(C,風) / 프로즌 스틸(B,風)  
-자력기(3편) : 아이스블록(C,風) / 럭키!(skill) / 웜홀(C,林) / 프로즌 스틸(B,風)

  

■ 아이큐 / 토우치 슈지(凍地修児) - 포지션 DF(2) / 속성 風  
회색 머리카락에 안경을 쓰고 있다. 그의 일상은 오직 공부뿐이며, 그 IQ는 1000을 넘었다고 한다. 계산 결과를 여동생인 아이시에게
가르쳐줘서 그녀에게 정확한 플레이를 시키고 있다는 듯 하다.  
-자력기(2편) : 꽃미남UP!(skill) / 프로즌 스틸(B,風) / 수퍼스캔(B,林) / 아이스블록(C,風)  
-자력기(3편) : 꽃미남UP!(skill) / 프로즌 스틸(B,風) / 학습(skill) / 노던 임팩트(S,風)

  

■ 클라라 / 쿠라카케 쿠라라(倉掛クララ) - 포지션 DF(3) / 속성 風  
여자아이. 파란 단발머리에, 머리 양사이드를 노란 띠로 묶고있다. 공허한 웃는 얼굴로 차가운 말을 내뱉는다. 하지만 그점이 좋다는 사람도
있다는 모양. 프로미넌스와의 혼성팀 [더 카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)의
멤버로 등장하기도 한다.  
-자력기(2편) : 포톤플래쉬(B,火) / 워터 베일(D,風) / 웜홀(C,林) / 아이스블록(C,風)  
-자력기(3편) : 프로즌 스틸(B,風) / 아이스블록(C,風) / 절약!(skill) / 갓핸드X(C,火)

거구의 몸체에, 남색 머리카락. '극악 스모 플레이어'란 이명을 가졌으며, 거체로 위력을 발휘해 볼을 빼앗는다. 프로미넌스와의 혼성팀 [더카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)의 멤버로 등장하기도 한다.  
-자력기(2편) : 플래닛 실드(B,火) / 로드롤러 태클(B,山) / 발구르기(B,山) / 프로즌 스틸(B,風)  
-자력기(3편) : 발구르기(B,山) / 디펜스 플러스(skill) / 거대 부채(D,風) / 그랜드 퀘이크(B,山)

  

■ 아이시 / 토우치 아이(凍地愛) - 포지션 MF(5) / 속성 風  
여자아이. 아이큐의 여동생. 보라색의 장발. 몸집이 작지만, 반사 신경이 뛰어나다.

-자력기(3편) : 아스테로이드 벨트(B,林) / 프리즈 샷(S,林) / 색기UP!(skill) / 그랑 펜릴(S,林)

  

■ 바렌 / 토바 렌(鳥羽蓮) - 포지션 MF(6) / 속성 山  
남자이지만 사랑스러운 얼굴이 컴플렉스. 때문에 마스크로 입을 숨기고 메이크업을 해서 무서운 분위기를 연출하고 있는 것 같다.  
-자력기(2편) : 워터 베일(D,風) / 스피닝 컷(B,風) / 포톤플래쉬(B,火) / 쿵푸어택(S,山)  
-자력기(3편) : 워터 베일(D,風) / 스피닝 컷(B,風) / 롤링 슬라이드(B,火) / 황제헹귄 2호(S,林)

짙은 녹색 머리카락에 가면을 쓰고 있다. 결점을 반성하여 곧바로 다음번에 활용하는 성격. 프로미넌스와의 혼성팀 [더카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)의 멤버로 등장하기도 한다.  
-자력기(2편) : 워터 베일(D,風) / 퀵 드로우(B,林) / 질풍대쉬(D,風) / 유니버스 블래스트(S,林)  
-자력기(3편) : 워터 베일(D,風) / 아이스 그라운드(B,風) / 헤븐즈 타임(D,風) / 유니버스 블래스트(S,林)

여자아이. 오렌지빛 머리카락의 포니테일. 얼굴을 숨기고 있는 것은 가면과 같은 화장이다. 웃는 얼굴은 의외로 귀엽다는 모양. 프로미넌스와의
혼성팀 [더 카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)의 멤버로 등장하기도 한다.

-자력기(3편) : 라이트닝 액셀(D,火) / 스파이얼 샷(S,風) / 워터 베일(D,風) / 스페이스 펭귄(S,風)

  

■ 브로우 / 츠무지노 토우지(旋風野冬司) - 포지션 FW(9) / 속성 風
