# 1. [KOF MI2](KOF%20MI2.md) ¶

  

★ 기본잡기  

  * 저녁뜸의 피날레 : 근접해서 ← or → + C
  * 선풍의 론도 : 근접해서 ← or → + D  

★ 필살기  

  * 천사의 애드 리브(전방) : 공중에서 →→
  * 천사의 애드 리브(후방) : 공중에서 ←←
  * 천사의 애드 리브(바깥쪽) : 공중에서 ↓↓
  * 천사의 애드 리브(안쪽) : 공중에서 ↑↑
  * 망각의 콰르텟 : ↓↘→ + A or C  
┠ 샤프 : →↓↘ + A or C  
┠ 플랫 : →↘↓↙← + A or C  
┗ 내추럴 : →↘↓↙← + B or D

  * 호접의 레퀴엠 : →↓↘ + A or C
  * 반역의 발라드 : →↘↓↙← + A or C
  * 암야의 녹턴 : ←↙↓↘→ + B or D
  * 나선의 프렐튜드 : →↘↓↙← + B or D (잡기)
  * 천우의 왈츠 : 공중에서 BD  
┗ 기만의 푸가 : → + D  

★ 초필살기(☆은 기게이지 소모 개수 표시)  

  * 희망의 팡파르(☆) : ↓↙←↙→ + A or C
  * 단죄의 미뉴에트(☆☆) : ←→↓↑↓ + A or C
  * 이별의 오버추어(☆☆☆) : ↓↙←↙→ + B or D (공중 가능)  

★ 스타일리쉬 아트  

  * 01 : A ▶ A ▶ A
  * 02 : B ▶ B ▶ B
  * 03 : → + B ▶ D
  * 04 : → + B ▶ → + D
  * 05 : → + B ▶ ← + D (페인트)
  * 06 : C ▶ C ▶ C
  * 07 : → + C ▶ ↗ + B ▶ D ▶ D
  * 08 : → + C ▶ C
  * 09 : ← + C ▶ C ▶ C
  * 10 : ← + C ▶ C ▶ ↓ + D
  * 11 : D ▶ D ▶ D
  * 12 : → + D ▶ C ▶ D
  * 13 : → + D ▶ D
  * 14 : A ▶ A ▶ (B or C ▶ A or D) 이후 괄호 안을 반복
  * 15 : B ▶ B ▶ (A or D ▶ B or C) 이후 괄호 안을 반복
  * 16 : C ▶ C ▶ (A or D ▶ B or C) 이후 괄호 안을 반복
  * 17 : D ▶ D ▶ (B or C ▶ A or D) 이후 괄호 안을 반복
  * 18 : 공중에서 A ▶ CD
  * 19 : 공중에서 B ▶ CD
