## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 던지기 : 근접해서 ← or → + CD  

★ 특수기  

  * 추가 공격 : 상대 다운시 ↘ + B
  * 기술 변환의 자세 : BCD (극모드에서는 사용 불가)  

★ 필살기  

  * 하늘을 뚫는 적색 화살 : ↓↘→ + A or B (모으기 가능)
  * 천진의 죄를 씻는 대불(●) : ↓↙← + A or B (4회 연속 입력)
  * 별천간친의 일도 : ↓↙← + C  

★ 초오의  

  * 주작 ・ 고상한 작열의 화염 : 공중에서 →←↙↓↘→ + A
  * 백호 ・ 하늘을 찢는 맹호의 날뛰는 발톱 : →←↙↓↘→ + B
  * 현무 ・ 근심에 찬 세상의 더러움을 정화하는 물기둥 : →←↙↓↘→ + C
  * 청룡 ・ 천지와 공명해 노호하는 천둥소리 : →←↙↓↘→ + D  

★ 잠재오의  

  * 사신의 힘을 모아 교만한 자를 황천으로 인도한다 : ←→↘↓↙← + B  

★ 난무오의  

  * 종언의 춤 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, ↓↘→ + B  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, BC, ↓↘→ + B
