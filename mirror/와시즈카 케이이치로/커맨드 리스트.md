## 1. [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) ¶

※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 업어치기 : 근접해서 ← or → + B  

★ 특수기  

  * 천주 찌르기 : 상대 다운시 ↘ + B   

★ 필살기  

  * 준살 : ↓↙← + A or B  
┗ 준살 ・ 연 : 준살 히트시 A or B 연타

  * 허공살 : ↓ 모으고 ↑ + A or B 
  * 질공살 : ← 모으고 → + A or B
  * 낭아(●) : ← 모으고 → + C   
┣ 낭아 ・ 직식(●) : 낭아 히트시 ↓↘→ + C  
┗ 낭아 ・ 사식 : 낭아 히트시 ↓↙← + C  

★ 초오의  

  * 진 ・ 랑아 : ↓↙←→ + AB  

★ 잠재오의  

  * 최종 ・ 랑아 : ↓↙←→ + B (모으기 가능, 최대 축적시 [가드 불능](%EA%B0%80%EB%93%9C%20%EB%B6%88%EB%8A%A5.md))

## 2. [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 업어치기 : 근접해서 ← or → + CD  

★ 특수기  

  * 천주 찌르기 : 상대 다운시 ↘ + B
  * 준속 ・ 거리 줄임(●) : 힘모드에서 BC를 모았다 사용   

★ 필살기  

  * 준살 : ↓↙← + A or B  
┗ 준살 ・ 연 : 준살 히트시 A or B 연타

  * 허공살 : ↓ 모으고 ↑ + A or B 
  * 질공살 : ← 모으고 → + A or B
  * 낭아(●)`[1]` : ← 모으고 → + C   
┣ 낭아 ・ 직식(●) : 낭아 히트시 ↓↘→ + C  
┣ 낭아 ・ 사식 : 낭아 히트시 ↓↙← + C  
┗ 낭아 ・ 복식 : 낭아 히트시 ↓↙← + B  

★ 초오의  

  * 진 ・ 랑아 : ↓↙←→ + AB  

★ 잠재오의  

  * 최종 ・ 랑아 : ↓↙←→ + B (모으기 가능, 최대 축적시 [가드 불능](%EA%B0%80%EB%93%9C%20%EB%B6%88%EB%8A%A5.md))  

★ 난무오의  

  * 절 ・ 랑아 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, ↓↘→ + A  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, A, BC, A, ↓↘→ + B

## 3. [네오지오 배틀 컬리시엄](%EB%84%A4%EC%98%A4%EC%A7%80%EC%98%A4%20%EB%B0%B0%ED%8B%80%20%EC%BB%AC%EB%A6%AC%EC%8B%9C%EC%97%84.md) ¶

★ 기본잡기  

  * 업어치기 : 근접해서 ← or → + CD  

★ 특수기  

  * 천주 찌르기 : 상대 다운시 ↘ + C
  * 얼굴 스쳐 올리기 : → + BD
  * 몸통 스쳐 올리기 : ← + BD   
┗ 비껴 상단 베기 : 얼굴 or 몸통 스쳐 올리기 성공시 BD  

★ 필살기  

  * 준살 : ↓↙← + A or C  
┗ 준살 ・ 연 : 준살 히트시 A or C 연타

  * 허공살 : ↓ 모으고 ↑ + A or C 
  * 질공살 : ← 모으고 → + A or C
  * 낭아(●) : ← 모으고 → + B or D   
┣ 낭아 ・ 직식 : 낭아 히트시 ↓↘→ + B or D  
┣ 낭아 ・ 사식 : 낭아 히트시 ↓↘→ + A  
┣ 낭아 ・ 복식 : 낭아 히트시 ↓↘→ + C  
┗ 낭아 ・ 번식(●) : 낭아 히트시 ↓↙← + 공격 버튼

  * 허식(●) : ↓↙← + BC (모으기 가능)  

★ 초필살기  

  * 진 ・ 랑아 : ↓↙←↙↓↘→ + A or C
  * 최종 ・ 랑아 : ↓↙←↙↓↘→ + AC (게이지 2개 소비)
