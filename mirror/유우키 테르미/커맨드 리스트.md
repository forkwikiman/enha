크로노 판타즈마 공식 일러스트

  

★ 드라이브 : 포스 이터  

  * D (하자마 사인아와 비슷, 판정은 더 안좋음)
  * ↓ + D
  * → + D
  * 공중에서 D
  * 공중에서 ↓ + D  

★ 오버드라이브 : 나이트메어 리퍼  

  * 히트게이지 상승량이 증가하고 디스토션 드라이브가 강화된다.`[1]` (※언리미티드는 오버드라이브 효과가 통상효과가 되며 언리미티드 전용 특수능력이 추가)  

★ 특수기  

  * → + A (상단무적)
  * → + B
  * → + C
  * ↘ + C
★ 필살기  

  * 사악(じゃがく 쟈가쿠) : ↓↘→ + D
돌진기. 첫 돌진에 히트시 쳐올린 후 우로보로스로 끌어당겨 내리꽂는다. 첫 돌진타격을 가드하거나 빗맞추면 후속타 없이 바로 중단. 마지막
타격시 다운상태가 일정시간 유지됨으로 이 때 디스토션으로 연계 가능.

  

  * 아천충(がせんしょう 가센쇼우) : ↓↙← + D
상대를 우로보로스 사슬로 포박해 조여 연타한 뒤 띄우는 기술. 잡기판정. 공격력은 평이하나 게이지 수급량이 매우 높다.

  

  * 아겸격(がれんげき 가렌게키) : ↓↓ + C
상대를 쓰러뜨려 혹은 쓰러진 상대를 발로 마구 짓밟고 날려버리는 기술. C 연타시 타격수가 증가한다.

  

★ 디스토션 드라이브`[2]`  

  * 사경멸섬아(じゃきょうめっせんが 쟈쿄멧센가) : ←↙↓↘→ + C (공중에서도 사용가능)
전방으로 빠르게 이동하며 상대를 단검으로 베는 기술  
※ 오버드라이브 - 타격수 2회로 증가  

언리미티드 -  

  * 굉아쌍천인(ごうがそうてんじん 고우가소우텐진) : →↓↘ + B (공중에서도 사용가능)
점프하며 두 번 차올려 상대를 띄우는 기술 `[3]`  
※ 오버드라이브 - 추가타로 땅으로 떨어뜨림  

언리미티드 - 특별한 효과 없음

  * 대사무겸섬(おろちぶれんせん 오로치부렌센) : →↘↓↙← + A, →↘↓↙← + B
A 사용 시 상대를 바닥에 쓰러뜨려 한번 밟은 뒤(2타) 돌려차기로 날려버리는 기술  
B 사용 시 상대를 바닥에 쓰러뜨려(1타) 마구밟고나서 돌려차기로 날려버리는 기술 `[4]`  
※ 오버드라이브 - 밟기 히트수 대폭 증가 <del>[이건 뭐 발칸포도 아니고...</del>](%EC%9D%B4%EA%B1%B4%20%EB%AD%90%20%EB%B3%91%EC%8B%A0%EB%8F%84%20%EC%95%84%EB%8B%88%EA%B3%A0.md),
대사무겸섬(...), 사박봉언진, 굉아쌍천인, 사경멸섬아로 캔슬가능  

언리미티드 - 특별한 효과 없음  

  * 황사참뇌아(おうじゃざんろうが 오우쟈잔로우가) : ↓↘→↓↘→ + A
상대의 공격 타이밍에 맞춰서 사용가능. 상대 앞에서 사라진 뒤 뒤편에서 나타나며 우로보로스 사슬로 조아서 붉은색 X자를 그으며 큰 타격을
주는 기술.  
※ 오버드라이브 - 모션에 차이는 없으나 데미지가 1000증가함 (보통은 3200, 오버드라이브 도중엔 4200). 보통 황사참뇌아는
상단기와 중단기만 반격 가능하지만 (하단과 던지기는 반격 불가능), 오버드라이브 시에는 하단도 반격이 가능하다고 한다.`[5]`  

언리미티드 -  

  * 사박봉언진(じゃばくふうえんじん 쟈바쿠후엔진) : →↘↓↙←→ + D
상대를 두 번 타격 후 우로보로스 2마리로 상대를 포박한 뒤 충격파를 먹여 날리는 기술. 히트게이지가 남아있다면 남아있는 양을 5씩 추가로
사용해서 충격파를 연달아 먹이는게 가능. 또한 남은 히트 게이지 양에 따라 마지막 타의 타격모션과 강도가 달라짐. `[6]`  
※ 오버드라이브 - 마지막 타를 먹이기 전 1회 추가타, 상대의 게이지도 소모, 마지막 타격의 모션 변경`[7]`  

언리미티드 -  

  * 사인연상아(じゃりんれんしょうが 쟈린렌쇼우가) : ↓↘→↓↘→ + D, ↓↙←↓↙← + D (히트게이지 100 소비)
↓↘→↓↘→ + D 사용 시 정면에 서 있는 상대를 우로보로스로 잡아 끌어온뒤 이리저리 날아다니며 상대를 벤후 땅으로 떨어뜨리는 기술. 보통
가드시 우로보로스가 가드 크러시를 일으키므로 배리어로 막아야 한다.  
↓↙←↓↙← + D 사용 시 대각선 상단의 상대를 잡는다. 동선 예측이 필요하므로 대공으로도 쓰기 힘든 편.  
※ 오버드라이브 - 히트수 증가. 오버드라이브 판 사인연상아는 **완전 가드 불능**이다. 즉 피하는 법 밖에 없다는 소리.  

언리미티드 -  

  * 언리미티드 전용기 1 : ↓↘→↓↘→ + C
상대를 → + D로 난타하며 마지막에 하자마의 뒤잡기 같은 발차기(모으기 가능)로 걷어참. 이때 대사무겸섬이나 굉아쌍천인으로 캔슬 가능하다.

  

  * 언리미티드 전용기 2 : →↘↓↙←→ + C
오버드라이브 도중에만 사용 가능. 하자마의 교룡열화참과 비슷한<del>똑같은</del>기술이며 히트후 우로보로스로 끌어와→ + C로
난도질하고 사경멸섬아와 황사참뢰아의 이펙트로 마무리한다.<del>추가 디스토션도 죄다 발칸포돋는 인간발칸포</del>

  

  * 언리미티드 전용기 3 : ←↙↓↘→ + D (공중에서만 사용 가능)
사경멸섬아와 비슷한 대신에 V자 모션으로 돌진한다. [초전자머신 볼테스 V](%EC%B4%88%EC%A0%84%EC%9E%90%EB%A8%B8%EC%8B%A0%20%EB%B3%BC%ED%85%8C%EC%8A%A4%20V.md)에서 출연하는 천공검 V자 베기 에서 따온 것.

  

★ 아스트랄 히트  

  * 신의 귀환 ・ 대사참두열봉아(神帰り・大蛇斬頭烈封餓 카미가에리・오로치잔토렛푸가) : ↓↓↓ + D
우로보로스 사슬과 동시에 붉은 늪을 소환하여 붉은 늪으로 우로보로스 사슬을 던져 상대를 포박 후 무수히 많은 우로보로스를 소환하며 상대를
타격. 그리고 스사노오 유닛의 힘으로 변신`[8]`하여 검을 만들어 낸후 상대를 베어 마무리.
