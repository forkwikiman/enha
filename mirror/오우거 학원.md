[이나즈마 일레븐](%EC%9D%B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9D%BC%EB%A0%88%EB%B8%90.md)에 등장하는 축구팀. 극장판 애니메이션, 게임, 만화, 소설 등에서 등장한다. 국내판 명칭은 오우거 **학교**. 감독은 [히비키감독](%ED%9E%88%EB%B9%84%ED%82%A4%20%EC%84%B8%EC%9D%B4%EA%B3%A0.md)의 후손인 히비키
제독. 캡틴은 버덥.

  

첫 등장은 극장판 이나즈마 일레븐에 등장한 극장판 전용팀. 축구가 악으로 규정된 먼 미래에 소속한 축구팀으로 역사를 바꾸기 위해 과거로
찾아가 [엔도 마모루](%EC%97%94%EB%8F%84%20%EB%A7%88%EB%AA%A8%EB%A3%A8.md)와 [라이몬중학교](%EB%9D%BC%EC%9D%B4%EB%AA%AC%20%EC%A4%91%ED%95%99%EA%B5%90.md)가 축구를 못하게
만들기 위해 **오퍼레이션 썬더브레이크**를 실행한다.

  

작중에서는 [풋볼프런티어](%ED%92%8B%EB%B3%BC%20%ED%94%84%EB%9F%B0%ED%8B%B0%EC%96%B4.md)에 출전,
[제우스중학교](%EC%A0%9C%EC%9A%B0%EC%8A%A4%20%EC%A4%91%ED%95%99%EA%B5%90.md)`[1]`를
36:0으로 갈아마셔버리는 엄청난 실력을 자랑했다. 감독이 제독이라든가, 선수를 "제군"이라고 부르는 등, 어디까지나 축구 선수가 아닌
[군인](%EA%B5%B0%EC%9D%B8.md)이다. 단지 풋볼 프런티어에 출전하기 위해 오우거 학원이라 칭했을 뿐.

  

[이나즈마 일레븐](%EC%9D%B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9D%BC%EB%A0%88%EB%B8%90.md)3 스파크/봄버에서는 [특정 요소를 만족](http://crive.pe.kr/110090775311) 시키면 오우가 학원에 속한
미스트레와 에스카바 두명을 스카우트할 수 있다. 또한 디 오우거에서는 정식으로 등장했으며, 전용 스토리도 있고 스카우트도 가능하다.

  

디 오우거에서는 보스답게 [데스브레이크](%EB%8D%B0%EC%8A%A4%20%EB%B8%8C%EB%A0%88%EC%9D%B4%ED%81%AC.md)같은 강력한
전용 필살기와 정신나간 수준의 능력치를 가지고 있지만, 정작 [리틀기간트](%EB%A6%AC%ED%8B%80%20%EA%B8%B0%EA%B0%84%ED%8A%B8.md)처럼 전체적으로 능력치 테이블이
초기부터 높을 뿐 성장은 거의 안하는 지라 그저 [망했어요](%EB%A7%9D%ED%96%88%EC%96%B4%EC%9A%94.md).

## 3. 선수 프로필 ¶

**※프로필은 게임 3편을 기준으로 작성했습니다.**

  

■ 버덥 스리드 (バダップ・スリード) - 포지션 FW(10) / 속성 林  
최강군단 오우가를 이끄는 정체불명의 남자. 이마에 의미를 알 수 없는 무늬가 새겨져있다.  
-자력기(3편) : 데스브레이크G2(S,山) / 데스스피어V3(S,林) / 혼신!(skill/Lv57) / 킬러필즈(D,林/Lv63)

  

■ 에스카 바멜 (エスカ・バメル) - 포지션 FW(9) / 속성 山  
통칭 「에스카바」. 점수를 따지 못하면 격노한다. 수단과 방법을 가리지 않고 점수를 따내는 강력한 스트라이커.  
-자력기(3편) : 데스레인(S,火) / 엔젤볼V2(D,風) / 시저스봄(D,山/Lv48) / 데스스피어(S,林/Lv67)

  

■ 미스트레네 칼스 (ミストレーネ・カルス) - 포지션 FW(11) / 속성 山  
통칭 「미스트레」. 언뜻 보면 착한 아이인것같지만, 사실은 아주 난폭하다.  
-자력기(3편) : 데스스피어(S,林) / 데스레인(S,火/Lv41) / 저지스루2(D,火/Lv53) / 데블볼V2(Lv54)

  

■ 자고멜 잔데(ザゴメル・ザンデ) - 포지션 GK(1) / 속성 山  
철벽을 자랑하는 오우가의 골키퍼. 노린 슛은 확실하게 잡는다.  
-자력기(3편) : 하이볼티지G3(C,風) / 니들해머改(C,風) / 일렉트랩改(C,風) / 부활!(skill/Lv46)

## 4. 성능 ¶

**※성능은 게임 3편을 기준으로 작성했습니다.**

  

디 오우거가 발매가 되지 전까지는 극장판의 번호로 영입이 가능한 에스카바와 미스트레를 사용할 수 있다. 능력치, 자력기가 준수할 뿐더러
무엇보다 작중에서 정말 보기 힘든 산속성 FW에 TP가 미친듯이 높았기 때문에 [제트스트림](%EC%A0%9C%ED%8A%B8%20%EC%8A%A4%ED%8A%B8%EB%A6%BC.md)을 2번씩 날리는게
가능했다.`[3]`

  

디 오우거가 발매 이후, 전원 영입이 가능하다. 단, 자고멜, 게보, 다이코는 각각 엔도, 쿠리마츠, 카베야마를 전작과 연동을 시켜야만
영입이 가능.

  

전체적인 성능은 DE멤버 수준의 멤버들이 TP가 무식하게 늘었다고 보면 된다. 허나 제일 커다란 문제는 자고멜.

  

자고멜의 속성은 산(...). 가젤이나 속성강화를 달아놓은 DE카제마루에겐 짤없이 뚫린다(...)
