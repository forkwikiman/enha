## 2.1. [Ctrl](%EC%BB%A8%ED%8A%B8%EB%A1%A4.md) ¶

+A : 페이지의 전체나 글의 전체를 선택ⓐ  
+B 또는 I : 즐겨찾기 등록  
+C : 드래그한 부분을 복사ⓐ  
+D : 즐겨찾기에 추가ⓐ  
+F : 찾기ⓐ  
+H : 방문 기록ⓐ  
+M : 이름 바꾸기  
+N : 새 창 또는 새 탭ⓐ  
+P : 인쇄  
+R : 새로고침ⓐ  
+S : 저장  
+T : 새 탭  
+V : 복사한 주소나 문구를 붙여넣는다  
+W : 탭 또는 창 닫기  
+X : 드래그한 내용을 잘라낸다  
+Y : redo. 방금한 작업을 다시 실행한다.  
+Z : 방금 실행한 작업을 취소하고 직전 작업으로 돌아간다  
+Esc : 시작  
+Tab : 오른 쪽으로 이동  
+Backspace : 커서의 오른쪽 글자를 지운다  
+숫자 : 탭이 여러개 열려있을 경우 해당 번호에 해당하는 탭으로 이동  
+0 : 가장 마지막에 실행한 탭 선택  
\+ +/- : 글자 크기 변경  
+마우스 휠 위아래 : 위쪽은 텍스트 확대. 아래는 텍스트 축소  
+Pg up/Pg Dn : 왼쪽/오른쪽 탭으로 이동

## 2.4. [윈도우 키](%EC%9C%88%EB%8F%84%EC%9A%B0%20%ED%82%A4.md) ¶

+C : ⑧ 오른쪽 참 메뉴를 호출한다. 이걸 알고 있으면 마우스를 굳이 오른쪽에 들고 가서 쓸어내릴 필요가 없어진다.  
+D : 바탕화면으로 이동. 한 번 더 누르면 원래 화면으로 돌아간다. Desktop. 바탕화면을 본다. M과 차이가 없는 것 같지만 M과는
달리 다시 한 번 누르면 전에 열어둔 창이 그대로 다시 나온다. M보다 누르기 쉽고 전환도 빠르다(...). 윈도우 8에서는 메트로 UI에서
데스크탑으로 바로 전환되는 기능도 한다. 윈도우의 우측 최하단의 빈 공간을 누르면 같은 기능이 동작된다.  
+E : Explorer. 윈도우 탐색기를 연다.  
+F : Finder. 윈도우 검색기를 연다. ⑧ 에서는 참 메뉴의 파일 검색이 뜬다.  
+H : ⑧ 참 메뉴의 공유를 바로 호출한다.  
+I : 참 메뉴의 설정이 뜬다. 여기에서 제어판 진입이나 컨트롤 변경, 전원 종료 등이 가능하다.  
+L : Lock screen. 컴퓨터 로그오프. 회사같은데서 컴 잠깐 잠그고 나갈때 유용하다. 보안이 중요한 회사의 경우 윈도우 비밀번호를
거는것은 기본인데 업무중에 자리를 비워야 하면 이 단축키를 항상 입력하고 다녀야 한다.  
+M : Minimize. 모든 창을 접는다.  
+N : OneNote로 보내기가 뜬다. OneNote 설치 필요.  
+P : Projector. 프로젝터에 연결. 노트북이 아닌 이상 딱히 쓸 일은 없다.  
+Q : ⑧ 참 메뉴의 전체 검색이 뜬다. 파일뿐 아니라 인터넷 검색 같은 것도 바로 할 수 있다.  
+Prt Sc : 모니터 전체 스크린샷 저장. 스크린샷은 내 사진에 저장. 윈도우 8부터 가능.  
+R : Run. 실행창이 나온다. cmd를 불러오거나 공유폴더 아이피 주소를 입력할 때 등 각종 윈도우 제어도구를 빠르게 실행할 때
유용하다.  
+X  

  * Windows Vista/7 : 윈도우 모바일 센터
  * Windows 8 : 시스템 관리 드롭바가 열림.
+방향키 - ⑦ 활성화된 창을 배치 가능. 윈도우 + ↑는 전체화면, ↓는 최소화, ←과 →는 각각 모니터 왼쪽, 오른쪽에 창 배치. 이
키는 듀얼 모니터를 사용하는 경우 사용중인 창을 마우스를 사용하지 않고 다른 모니터로 창을 옮길 수 있다. 듀얼 모니터를 쓰다가 모니터를
하나만 사용하는 경우 가끔 듀얼 모니터의 위치에 떠있던 창이 현재 싱글 모니터 임에도 불구하고 듀얼모니터를 사용할 때의 위치에 떠있어서
화면상에서는 프로그램 창이 안보이고 창의 위치를 마우스로 옮기지 못해 프로그램을 사용할 수 없는 경우가 생기는데 이를 해결하는 가장빠른
수단이다.  
+Pause/break - 시스템 기본정보창을 연다.  
+Tab  

  * Windows Vista/7 : 에어로 탭. 단순 알트탭보다 원하는 창 찾기가 좀 더 편하다. 에어로가 지원하지 않은 윈도우 버전이거나 윈도우 테마가 고전이면 사용 불가.
  * Windows 8 : 앱간 전환. Alt+Tab과는 달리 데스크톱을 하나의 앱으로 취급해서 스토어 앱 단위로 화면을 전환한다. 데스크탑 모드에서만 놀면 사실상 쓸 일이 없다.
+숫자키 - ⓥ 작업 표시줄의 빠른 실행 탭에 추가한 응용 프로그램들을 빠르게 실행 할 수 있다.
