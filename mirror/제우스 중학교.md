[썬더 일레븐](%EC%8D%AC%EB%8D%94%20%EC%9D%BC%EB%A0%88%EB%B8%90.md)에서 나오는 중학교.
게임판 1편과 [애니판](%EC%95%A0%EB%8B%88%ED%8C%90.md) 1기의 최종보스팀. 최종보스팀 답게 전용BGM도 갖고
있다. [전용BGM - 성지](http://www.youtube.com/watch?v=2wlE39jKqEc) [전용시합 BGM - 신들의
성전](http://www.youtube.com/watch?v=Tkw3kl63Z8E)

  

게임판에서는 팀원 모두의 기술 하나하나가 강력하다. 특히 최종보스로 나오는 <del>수입산 캡틴</del> [아후로테루미](%EC%95%84%ED%9B%84%EB%A1%9C%20%ED%85%8C%EB%A3%A8%EB%AF%B8.md)의 필살기 **갓
노우즈**는 [마신 더 핸드](%EB%A7%88%EC%8B%A0%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)가
아니면 막을 수 없다고 보면 된다. 그러나 TP의 소모가 크다는 것이 약점.

  

원래 별로 축구로 유명한 학교는 아니였으나, [풋볼프런티어](%ED%92%8B%EB%B3%BC%20%ED%94%84%EB%9F%B0%ED%8B%B0%EC%96%B4.md) 예선에서 제국
일레븐이 라이몬 일레븐에 패배하자 항상 제일 강한 팀을 추구한 [제국학원](%EC%A0%9C%EA%B5%AD%20%ED%95%99%EC%9B%90.md) 축구팀의 의 원 감독 [카게야마 레이지](/wik
i/%EC%B9%B4%EA%B2%8C%EC%95%BC%EB%A7%88%20%EB%A0%88%EC%9D%B4%EC%A7%80%28%EC%9D%
B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9D%BC%EB%A0%88%EB%B8%90%29)의 원조를 받고 순식간에
축구 프런티어 내에서는 따라올 팀이 없는 괴물 팀이 되어버린다. 우승하는 방식은 골로 쟁취하는 것이 아닌, 압도적인 힘으로 상대 팀 전원에게
심각한 부상을 입혀 부전승을 얻어내는 타입. 전국대회 1회전에서 [키도유우토](%ED%82%A4%EB%8F%84%20%EC%9C%A0%EC%9A%B0%ED%86%A0.md)를 제외한 [제국학원](%EC%A0%9C%EA%B5%AD%20%ED%95%99%EC%9B%90.md) 팀원 전원에게 심각한 부상을 입히며 10:0으로
승리했다.

  

갑작스레 이 팀이 강해진 비밀은 팀원 전체가 인체를 강화시키는 약물인 [신의아쿠아](%EC%8B%A0%EC%9D%98%20%EC%95%84%EC%BF%A0%EC%95%84.md)를 복용했기 때문. 후에 이
사실이 [라이몬 중학교](%EB%9D%BC%EC%9D%B4%EB%AA%AC%20%EC%A4%91%ED%95%99%EA%B5%90.md)
매니저들과 그곳에 잠입수사를 하고 있던 형사의 의해 발각되며 카게야마는 다시 체포당하지만 에일리어 학원편에서 탈옥하게 된다.

  

전국대회 결승전에서 라이몬 일레븐과 붙게되는데, 초반부터 중반까지 신의 아쿠아에 의한 압도적인 파워로 라이몬 일레븐을 걸레짝으로 만들고
3골을 선취한 후에 엔도 마모루를 심심풀이 삼아 괴롭히는 엄청난 위력을 보여준다. 그러나 후반에 [마신 더핸드](%EB%A7%88%EC%8B%A0%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)를 습득하게 된 엔도에게 갓
노우즈가 막히고 그걸 계기로 사기가 오른 나머지 라이몬 멤버들에게 [트윈부스트F](%ED%8A%B8%EC%9C%88%20%EB%B6%80%EC%8A%A4%ED%8A%B8.md)로 연속 3골을 허용당하고,
마지막에는 [파이널 토네이도](%ED%8C%8C%EC%9D%B4%EB%84%90%20%ED%86%A0%EB%84%A4%EC%9D%B4%EB%8F%84.md)에 의해 4:3으로 역전패 당하고 만다.  
덧붙여 게임판에서는 라이몬 매니저들이 몰래 잠입해 **신의 아쿠아를 맹물로 바꿔치기 해버린다.** 당연히 후반전에 신의 아쿠아를 못 마시게
된 제우스는 제대로[...](%EB%A7%9D%ED%96%88%EC%96%B4%EC%9A%94.md)

  

극장판에서는 더욱 안습. [오우가학원](%EC%98%A4%EC%9A%B0%EA%B0%80%20%ED%95%99%EC%9B%90.md)한테 36:0으로 비참하게
패배해버렸다.

  

게임판의 후속작 '위협의 침략자'에서는 라이몬 일레븐과 연습경기를 하기도 하며 패배한 이후로 신의 아쿠아에 의존하는 습관은 팀원 모두가
버렸다고 한다.

  

이름이 제우스 중학교인 만큼 축구부원 전부 그리스 신화 인물들에 관련 된 이름(별명)을 가지고 있다. 그리고 이유는 모르겠지만 다른 축구부에
비해 예쁘장하게 생긴 선수가 많다. 하지만 겉모습에 절대 속지 말아라. **전부 남자다.**(...)

  

게임 2편 및 3편에선 게임 클리어 후에 히든 팀으로 **"신과 우주"** 란 제우스중+[더제네시스](%EB%8D%94%20%EC%A0%9C%EB%84%A4%EC%8B%9C%EC%8A%A4.md)의 혼성팀으로 등장하기도 한다.
팀에 포함된 제우스중 멤버는 아프로디, 데메테르, 헤라, 헤파이스, 포세이돈, 헤르메스, 이렇게 6명.

## 2. 우라 제우스 ¶

게임 1탄에서 등장하는 제우스의 어나더버전. 1회차를 클리어하면 연습시합에서 대전할수 있다. 제우스의 어나더 버전이지만 레벨은 평균
80인데다가 능력치도 사기적이라 이기기 힘들다. 골기퍼 포세이돈의 TP가 다 떨어져도 웬만한 필살기는 맨몸으로 막아내는등 그야말로 난이도
높은 최강최악의 적. 2탄이나 3탄에서도 싸울 수 있다.

## 4. 선수 프로필 ¶

※ 게임 1편을 기준으로 작성했습니다. 1편에선 클리어 후 제우스중 멤버를 동료로 영입할 수 있게 된다. 다만 아프로디와 포세이돈은 동료로
영입할 수 없다. 그래도 2편에선 이 두사람도 영입하는게 가능해진다. 게임 3편에서 휴대폰 연동으로 파이어 드래곤 팀의 아프로디와는 별개로
**제우스중 버전의 아프로디**를 따로 팀에 영입할 수 있다.

  

■ 아프로디 / [아후로테루미](%EC%95%84%ED%9B%84%EB%A1%9C%20%ED%85%8C%EB%A3%A8%EB%AF%B8.md)(亜風炉照美) -
포지션 MF(10번) / 속성 林  
제우스중 축구부의 캡틴. 2학년. 금색 장발. 마치 예술과 같은 그 플레이는 상대편마저도 넋을 잃어버릴 정도. 3편에선 한국 대표팀 [파이어드래곤](%ED%8C%8C%EC%9D%B4%EC%96%B4%20%EB%93%9C%EB%9E%98%EA%B3%A4.md)의 멤버로
등장한다.  
-자력기(1편) : 갓 노우즈(S,風) / 헤븐즈 타임(D,風) / 심판의 철퇴(B,火) / 디바인 애로(S,風)  
-자력기(2편) : 슛포스(skill) / 갓 노우즈(S,風) / 헤븐즈 타임(D,風) / 갓 브레이크(S,風)  
-자력기(3편) : 슛포스(skill) / 갓 노우즈(S,風) / 갓 브레이크(S,風) / 더 이카로스 (D,火)

  

■ 포세이돈 / 포세이 돈이치(歩星呑一) - 포지션 GK(1번) / 속성 山  
제우스중 3학년. 덩치가 크다. 수많은 공격에도 골대를 지키는 모습으로 인해 "해신"이라고 불리고 있다.  
-자력기(1편) : 츠나미 월(C,風) / 기간트 월(C,山) / 더 월(B,山) / 발구르기(B,山)  
-자력기(2편) : 초기술!(skill) / 발구르기(B,山) / 기간트 월(C,山) / 츠나미 월(C,風)  
-자력기(3편) : 초기술!(skill) / 츠나미 월(C,風) / 기간트월(C,山) / 스톤 프리즌(B,山)

  

■ 아폴론 / 아포로 히카루(阿保露光) - 포지션 DF(2번) / 속성 林  
제우스중 2학년. 몸집이 작고, 연두빛 머리카락에 머리띠를 쓰고 있다. 평상시엔 밝은 성격이지만, 승부에 이기기 위해선 아무렇지도 않게
냉혹한 플레이를 하기도 한다.  
-자력기(1편) : 심판의 철퇴(B,火) / 샤인 드라이브(S,火) / 매직(D,林) / 대시 스톰(D,風)  
-자력기(2편) : 심판의 철퇴(B,火) / 샤인 드라이브(S,火) / 분신 디펜스(B,林) / 대시 스톰(D,風)  
-자력기(3편) : 심판의 철퇴(B,火) / 샤인 드라이브(S,火) / 듀얼 스톰(B,林) / 대시스톰(D,風)

  

■ 헤파이스 / 헤파이 엔(部灰炎) - 포지션 DF(3번) / 속성 火  
제우스중 3학년. 웨이브결의 보라빛 장발과 갈색피부가 특징. 텐션이 오르면 자신의 플레이에 도취해버린다.  
-자력기(1편) : 대시 스톰(D,風) / 사이코 슛(S,林) / 파이어 토네이도(S,火) / 불꽃 풍향계(S,火)  
-자력기(2편) : 대시 스톰(D,風) / 심판의 철퇴(B,火) / 파이어 토네이도(S,火) / 불꽃 풍향계(S,火)  
-자력기(3편) : 대시 스톰(D,風) / 심판의 철퇴(B,火) / 헬파이어(S,火) / 불꽃 풍향계(S,火)

  

■ 아레스 / 아레스 란(荒須乱) - 포지션 DF(4번) / 속성 山
