울산광역시 중구의 마을버스. 위 그림의 보라색 노선. 노선은 다음과 같다. (굵게 표시된 곳에서 시내버스로 환승 가능)

  * 백양사노선(서부) : **성안청구** ↔ 방통대 후문 ↔ 신안골드빌 ↔ G마트앞 ↔ 지적공사앞 ↔ **성안금호아파트앞** ↔ 지적공사앞 → 백양사
  * 라이온스노선(동부) : **성안청구** ↔ 성안청구앞 ↔ -<del>성안농협</del>-`[2]` ↔ 백련과학유치원 ↔ 애니원고등학교 ↔ 장애인복지회관 ↔ 시립요양원 ↔ 편의점앞 ↔ 라이온스앞
  * 공통구간 : → **경찰청(백양사사거리)** → 궁도장 → **성안입구(북정교차로)** → **향교** → **우정선경2차정문** → 우정시장 → 강원마트 → 목살골목 → 양사초 → 중부도서관(동헌) → 소방서사거리 → <del>시계탑사거리</del>`[3]` → **박병원** → 중앙동주민센터 → 복산초 → **성안입구(북정교차로)** → 궁도장 → 경찰청(백양사사거리) →   

이 노선의 뿌리는 하루 **다섯 번** 운행되던 비정규노선. 명색이 회사명이 **성안**마을버스인데 성안행 노선이 부실했던 아이러니가
발생했으나, 2007년 전후로 정규노선으로 승격되어 운행중이다. 이후 2010년대 들어 성안 순환 고리를 서/동부 두개로 쪼개서 운행중.
(정확한 시기 추가바람.)

2호선이 우정동 구간에서 장춘로를 이용하는 반면, 3호선은 동헌길로 운행중이다. 또한 등교/하교시간 각 2회씩 함월고등학교 지원운행중이다.
