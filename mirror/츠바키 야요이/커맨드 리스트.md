컨티뉴엄 시프트 공식 일러스트

  

★ 드라이브 : 인스톨`[1]`  

  * D
  * ↓ + D
  * 공중에서 D  

★ 특수기  

  * B or C 입력후 같은 버튼 추가입력(공중 B or C, 앉아 B or C, → + B or C 입력으로 나가는 특수기에도 적용됨)
  * → + A or B or C
  * ↘ + C  

★ 필살기  

  * 심기 - 번뜩이는 일격 : ↓↘→ + A or B or C or D `[2]`
  * 심검 - 빛을 베는 검 : ↓↙← + A or B or C or D
  * 심검 - 바람을 재우는 검 : ↓↓ + A or B or C or D (모으기 가능)`[3]`
  * 심창 - 하늘을 뚫는 창 : →↓↘ + A or B or C or D
  * 심기 - 하늘에 번뜩이는 빛 : 공중에서 ↓↘→ + A or B or C or D `[4]`
  * 심기 - 하늘을 나는 날개 : 공중에서 ↓↙← + A or B or C or D  

★ 디스토션 드라이브  

  * 심벌 - 하늘을 베는 불꽃 : ↓↘→ ↓↘→ + C or D `[5]`
  * 심벌 - 땅을 품는 옷 : ↓↙←↓↙← + D  

★ 아스트랄 히트  

  * 심성 - 인간을 처단하는 신 : →↘↓↙←→ + C

## 2. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

★ 드라이브 - 인스톨(インストール / Install) : 필살기를 강화하는 에너지를 채웁니다.  
★ 오버 드라이브 - 세컨드 인스톨(セカンドインストール / Second Install)

  

★ 필살기  

  * 심기 번뜩이는 일격(審技・閃ク一ノ撃 / Sanctum Aequum) : 앞으로 돌진하는 공격입니다.
  * 심검 빛을 베는 검(審剣・光ヲ断ツ剣 / Sanctum Veritas) : 대공 공격입니다.
  * 심검 바람을 재우는 검(審剣・風ヲ凪グ剣 / Sanctum Decus) : 옆으로 베는 공격입니다. D 기술은 최대까지 채우면 가드 불능입니다.
  * 심창 하늘을 뚫는 창(審槍・空ヲ突ク槍 / Benedictus Rex) : 뛰어오르면서 대공 공격을 합니다.
  * 심기 하늘에 번뜩이는 빛(審技・空ニ閃ク光 / Aequum Eleison) : 공중에서 날개를 펄쳐 공격합니다.
  * 심기 하늘을 나는 날개(審技・空ヲ翔ル翼 / Lux Aeterna) : 공중에서 대각선 앞으로 낙하하면서 공격합니다.
  * 심기 어둠을 뚫는 등불(審技・闇ヲ穿ツ灯 / Lux Macto) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)
  * 심쇄 무로 이끄는 사슬(審鎖・無ヘ誘ウ鎖 / Sanctus Aerolata) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 심벌 땅을 품는 옷(審罰・地ヲ抱ク衣 / Macto Maledictus) : 인스톨 게이지가 0이 될 때까지 D 필살기를 게이지 소비 없이 사용할 수 있습니다.
  * 심벌 하늘을 베는 불꽃(審罰・天ヲ刈ル焔 / Confutatis Maledictus) : 상대를 벽에 매달고 장풍으로 공격합니다. D 필살기는 인스톨 게이지의 남은 양에 따라 위력이 달라집니다.
  * 심벌 땅을 뒤덮는 칼날(審罰・地ヲ屠ル刃 / Requiem Maledictis) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 아스트랄 히트  

  * 심성 인간을 처단하는 신(審聖・人ヲ裁ク神 / Requim Aeternam) : 변신해서 돌진 공격으로 강력한 일격을 가합니다.  

◎ 기타 - 게임에서 기술 사용 시에 따로 외치는 이름들  

  * 심기 번뜩이는 일격 : 호법극(護法戟)
  * 심검 빛을 베는 검 : 화휘참(華輝斬)
  * 심검 바람을 재우는 검 : 휘쇄섬(輝砕閃)
