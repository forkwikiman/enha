  * 힐링포션B : 200골드, 체력 400 회복

  * 회복포션A : 520골드, 초당 130의 체력을 10초간 회복

  * 마나포션B : 300골드, 마나 250회복  

  * 마나포션A : 460골드, 마나 350회복

  * 민첩의 룬 : 1 나무, 민첩 1 증가.  

  * 지능의 룬 : 1 나무, 지능 1 증가.  

  * 프로텍트 스크롤 : 1 나무, 방어력 70 증가, 쿨다운 20초  

  * 헤이스트 스크롤 : 1 나무, 공/이속 80 % 증가, 쿨다운 5초

아이템을 찾을때도 사용된다.

## 3 조합 아이템 ¶

### 3.1 장착 아이템 ¶

  * 프라가라흐 = 아스카론 + 바리사다 + 신속의 반지 + 헤이스트 글러브   
올스텟 15 감소 ,공격속도 1000% 증가  

  * 갓 브레이커 = 듀랜달 + 아스가론 + 알자드의 단검 + 홀리 소드   
공격할 때마다 20% 확률로 목표물 주변 250 범위 안의 적들에게 공격자의 레벨 x 60 만큼의 피해를 줌 ,공격력 +10%  

  * 그람 = 카산드라 + 드래곤 슬레이어 + 홀리 소드   
공격할 때마다, 30%확률로 현재 남아있는 마나량만큼의 피해를 자신이 입음, 공격력 1000 공격속도 15% 증가  

  * 그라디우스 = 그룬가르드 + 가룬가르드   
민첩 20감소 / 15% 확률로 3배 크리티컬  

  * 랑그릿사 = 영웅의 검 + 엑스칼리버 + 듀랜달 + 드래곤 슬레이어 + 세계수의 나뭇가지   
힘, 민, 지의 합이 180을 넘으면 공격시 아이템이 떨어짐,공격력 1500, 공격속도 50% 증가  

  * 궁니르 = 바쥬라 + 토르의 망치   
데미지 1500의 전격 발사가능 / 공격력 +50  

  * 아슈켈론 = 드래곤 슬레이어 + 드래곤 스케일   
드래곤류 공격 시, 3% 확률로 모든 스탯 +1, 드래곤류에 피격시 5% 확률로 아이템 파괴됨, 마법 방어력 50% 증가  

  * 파마의 활 = 그룬가르드 + 달의 목걸이   
10%확률 2배 크리티컬 지능 10 증가, 키쿄우가 사용 시 마법 능력 강화  
영력 폭발, 파마의 화살의 데미지가 75 증가, 봉인의 화살 사용 시 대상 주변 275 범위 안의 적들에게 60%의 데미지  

  * 그랜드 스태프 = 똑똑이 마법서 + 백과사전 + 달의 목걸이   
마법을 사용 시 총 마나의 5% 만큼 체력 회복 지능 15증가  

  * 마나 스태프 = 그랜드 스태프 + 달의 목걸이 + 원서클   
마법을 사용 시 총 마나의 10% 만큼 체력 회복, 마법효과가 두 배로 되는 버프(배니시)를 대상에게 시전 (5초 지속)  
지능 15 마나 500증가  

  * 레이징 하트 = 그랜드 스태프 + 궁니르 + 세계수의 나뭇가지   
엠리젠4 지능 50증가 / 나노하의 판넬 지원확률이 디바인 버스터 사용시 50% 스타라이트 브레이커 사용시 30% 증가  

  * 지배자의 갑옷 = 지그프리드의 갑옷 + 문 아머 + 플레이트 실드   
30% 확률로 물리 공격 방어 방어력 13 증가  

  * 가디언 = 자바자 + 드래곤 스케일 + 문아머 + 플레이트 실드   
35% 확률로 150 데미지를 흡수 방어 25증가 피리젠 30증가  

  * 이빌 아머 = 지배자의 갑옷 + 라이프 포스 배슬 + 죽음의 갑옷   
회피(이베이젼) 30% 방어력 50,마법 방어력 25%, 피리젠 10 증가  

  * 샌드웜의 비늘 = 자바자 + 보호의 망토 + 보호 망토 + 가고일의 비늘   
방어력 30 . 피리젠 10 증가  

  * 아이언 메이든 = 자이언트의 갑옷 + 플레이트 쉴드 + 보호 망토   
일반 공격을 할 때 15% 확률로 350 의 피해를 자신이 입음, 방어력 50증가, 20% 확률로 원거리 공격 흡수  

  * 언데드의 생명 = 태양의 목걸이 + 라이프 포스 베슬   
민첩 15,지능10 감소 체력 400 피리젠 55 증가  

  * 문 크리스탈 = 파이브 서클 + 라이프 포스 베슬   
5초 동안 밤으로 만든다. 피리젠 50증가 엠리젠 5감소  

  * 하이퍼 스톤 = 샤이닝 아뮬렛 + 라이프 포스 베슬 +요괴의 탈   
엠리젠 3감소 모든스탯 50증가  

  * 골드 레지스탕스 = 오거 파워 건틀릿 + 아이언 글러브 + 헤이스트 글러브   
40% 확률로 2.5배 크리티컬, 방어력 10증가 공격속도 25% 감소  
악의 성향을 가진 유닛 사용 불가 (ex: 바라이스이쇼,츠바키,이누야샤,샤나,키쿄우)  

  * 샤이닝 크리스탈 = 샤이닝 아뮬렛 + 부서진 샤이닝 아뮬렛 + 거짓된 샤이닝 아뮬렛   
올스텟 +50 / 엠리젠 무한 / HP가 MP보다 많을 시 즉사 / 공격속도 +50%  

  * 판데모니움 = 죽어버린 죽음 + 참살의 저주   
사망 시 HP 1 로 부활(리안카네이션) / 힘, 민첩 +15 / 공격력 +200  
제작시 600개가 만들어진다.  

  * 요정의 반지 = 몬티얼의 거울 + 페어리의 눈물   
35초마다 마법의 기사를 소환 가능 / 마법사만 장비 가능 / 마법방어력 40% 엠리젠 7증가  

  * 식스 서클 = 몽환의 거울 + 투 서클 + 포 서클   
본체의 30% 대미지를 가지는 분신 소환 / 올스탯 5, 힘 10, 방어력 5, 엠리젠 4 증가  

  * 세븐 서클 = 쓰리 서클 + 배신자의 낙인   
모든스탯 20 엠리젠 20 증가  

  * 에잇 서클 = 궁니르 + 세븐 서클   
데미지 2500 의 전격 발사 (쿨다운 8초) ,엠리젠 +20 / 올스탯 +20 / 공격력 +100  

  * 텐 서클 = 원 서클 + 투 서클 + 쓰리 서클 + 파이브 서클 + 나인 서클   
MP 재생력 +20 / 올 스탯 +100  

  * 카무이 = 갓 브레이커 + 하이퍼 포션 + 드래곤 슬레이어 + 세인트 엔젤   
한번 주인이 되면 떨어뜨릴 수 없다. ,공격력 +50% / 반신화 사용가능 (쿨다운 250초)  

  * 플레임 가드 = 카산드라 + 영웅의 검 + 토르의 망치 + 플레이트 실드
