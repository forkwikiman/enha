# 4 운영 ¶

  

예전에는 란슬롯이 워낙 강력한 탱커여서 애매한 위치였지만 란슬이 적당히 너프된 지금은 대마력과 아발론, 검술단련으로 따라오는 회피를 사용해
몸빵형으로 주로 간다. 단, 대마력 속성을 찍는 순간 mp 소모가 엄청나기 때문에 앰회에 신경쓰는게 좋으며 본인의 컨트롤과 카야의 보조가
합쳐질때 진정 빛나는 몸빵이 되는 캐릭터이다. 여담으로 민케이지만 방어력은 갑옷으로 때우고 힘을 찍어 체력을 올리는게 훨씬 잘버틴다.

  

추천 탬은 세븐서클 계열(세븐,에잇,브류나크) 특히 브류나크는 세이버의 전용탬이라 해도 과언이 아닐만큼 효율이 굉장히 좋다. 만들 수 있다면
가급적 차지하도록 하자.

  
  

스킬트리

  
  

1\. 몸빵형

  

스킬 : 슬레쉬/대마력 -> 아발론 -> 엑스칼리버 -> 풍왕결계

  

속성 : 민첩성 증대 -> 대마력강화 -> 검술단련 -> 아발론 -> 엑스칼리버

  
  

가장 일반적인 형태의 몸빵 스킬트리. 대마력의 강력한 회복력과 검술단련으로 따라오는 회피, 돌진 등은 1레벨 아발론으로 버티며 슬레쉬로
야금야금 몹을 잡아들어가는 형태다. 모든 타입을 통틀어 마나 소모가 가장 크니 초당 앰회 20정도는 맞춰야 무난하다.

  

2\. 극사냥형

  

스킬 : 슬레쉬/엑스칼리버 -> 대마력 -> 아발론 -> 풍왕결계

  

속성 : 민첩성 증대 -> 검술단련 -> 대마력강화 -> 아발론 -> 엑스칼리버

  
  

란슬롯 너프 전의 일반적인 트리. 대마력을 과감하게 포기하고 엑스칼리버와 슬레시로 사냥능력을 극대화한 타입이다. 이렇게 찍었을 경우 첫
타니아 전까지 몹 리젠자리 하나를 완벽하게 제압할 수 있지만 대마력을 다 찍을때까지 중반이 굉장히 암울하다. 몸빵 세이버 쪽이 더 지루하지만
효율이 높다.
