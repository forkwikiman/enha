# 4 운영 ¶

  

몹정리에 매우 뛰어난 캐릭터. 딱히 탬을 타는 캐릭터도 아니라 앰회 하나만 덜렁 들고 몹들을 쓸어버리곤 한다. 중갑옷과 요력흡수를 이용해
가끔 탱커 겸 딜러로 나서는 모습도 볼 수 있지만 역시 주로 잡몹정리를 담당한다. 특히 속성으로 배우는 명도잔월파가 데미지도 데미지지만
지속시간이 뛰어나 공중몹을 정리하는데 탁월한 능력을 보여준다.

  

30랩 보너스를 받으면 금강창파가 매우 강력해지며 후반부의 거대몹이 금강창파 한방에 반피 이상 날아가는 모습을 감상할 수 있다. 금강창파
덕에 후반까지 활약하며 유사시 요괴화를 이용해 탬하나 없이 몸빵을 대신하기도 한다(아주 잠깐이지만) 전천후 캐릭터이면서 스킬이 주력이라
초보자가 쓰기에도 좋은 캐릭.

  
  

스킬트리

  
  

1\. 빠른 명도잔월파

  

스킬 : 금강폭류파/붉은 철쇄아 -> 폭류파 -> 금강창파 -> 바람의 상처

  

속성 : 명도잔월파 -> 이후 순서 자유.

  

명도 잔월파에 특화된 스킬트리. 취향에 따라 명도 지속시간을 조금 포기하고 금강창파를 먼저 찍어도 무방하다. 하지만 명도 잔월파의 마나소모가
800이나 되기에 초반에는 좀 부담스러운 트리다. 대신 첫 공중부터 명도를 활용할 수 있다는 장점이 있다.

  
  

2\. 일반

  
  

스킬 : 금강폭류파/금강창파 -> 붉은철쇄아 -> 폭류파 -> 바람의 상처

  

속성 : 금강폭퓨파 -> 이후 순서 자유.

  
  

그냥 무난한 스킬트리.

  
  

3\. 몸빵형

  

스킬 : 붉은철쇄아/금강폭류파 -> 금강창파 -> 폭류파 -> 바람의 상처

  

속성 : 불쥐의 옷 -> 요력강화 -> 요괴화 -> 요력흡수 -> 명도잔월파 -> 이후 자유

  
  

요력흡수로 인한 라이프 회복과 중갑옷, 요괴화를 믿고 버티는 타입. 빨리 때릴수록 피회복이 많이지므로 공속탬이 필수이며 돌진같은 한방기를
폭류파로 받아치는 적절한 센스가 필요하다. (세이버의 아발론과 비슷한 활용) 중반 이후에는 모든 속성을 찍는 이누의 특성상 다른 트리를 탄
경우와 다를 바 없으므로 굳이 처음부터 작정하고 이쪽으로 가지 않아도 상황이 허락하면 언제든지 몸빵으로 전환할 수 있으니 팀에 몸빵이 없는
경우를 제외하면 그냥 빠른 명도 트리를 타는게 낫다.
