# 4 운영 ¶

  

무한의 검제와 무장투척으로 초반 몹정리를 책임지는 캐릭터. 중반에는 무검에 들어갔다가 엄청난 몹의 공세에 비명횡사할 수도 있다. 사용자의
컨트롤이 뛰어나야 중반에 제대로 활약하는 캐릭터이며 보스전에서는 무검+투영으로 데미지를 주고 폴리모프 이후의 후반에서는 몹 리젠자리에서
무검을 사용해 한마리씩 끊어주는 역할을 한다.

  
  

사실 간장막야(데미지 420% 증가, 샤나의 플레임헤이즈 다음으로 높은 데미지 증가률이다)를 이용해 평타딜러로 갈 수도 있다. 단,
간장막야의 페널티가 막대하니 이 경우 탬이 엄청나게 좋아야한다.  

  
  

스킬트리

  
  

1\. 무검형

  

스킬 : 무한의검제/트레이스 온 -> 성해포 -> 로 아이아스 -> 간장막야

  

속성 : 연장투척강화 -> 연속무구 투척 -> 로 아이아스 -> 아발론 -> 무한의 검제

  
  

무한의 검제와 연속무구투척을 활용하는 타입. 로 아이아스는 속성으로 때우고 성해포로 조금이라도 화력을 올리는게 보통이다. 취향에 따라
성해포보다 로 아이아스를 먼저 찍어도 무방하다. 대개의 시로는 이쪽 트리를 탄다.

  
  

2\. 근접딜러형

  

스킬 : 간장막야/무한의검제 -> 성해포 -> 로 아이아스 -> 트레이스 온

  

속성 : 연장투척강화 -> 전투기술 숙달 -> 아발론 -> 로 아이아스 -> 성해포 -> 무한의 검제

  
  

간장막야의 데미지 증가를 무기로 근접딜러를 하는 형태. 샤나보다 데미지 증가율은 떨어지지만 무한의 검제라는 전용 필드와 성해포의 다양한
스킬로 모자란 부분을 때운다. 무검 안에서 성해포와 무검을 모두 활용하려면 손이 아주 빨라야하며 간장막야의 페널티를 때우고 샤나급 DPS를
가지려면 탬이 아주 좋아야하는 단점이 있다.
