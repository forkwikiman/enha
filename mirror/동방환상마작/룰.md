  * 상위항목 : [동방환상마작](%EB%8F%99%EB%B0%A9%ED%99%98%EC%83%81%EB%A7%88%EC%9E%91.md), [동방환상마작 3rdGeneration](%EB%8F%99%EB%B0%A9%ED%99%98%EC%83%81%EB%A7%88%EC%9E%91%203rdGeneration.md), [동방환상마작4](%EB%8F%99%EB%B0%A9%ED%99%98%EC%83%81%EB%A7%88%EC%9E%914.md)  

동방환상마작의 룰을 적어놓은 항목. 탁자에 따라 기본 룰이 약간 바뀔 수도 있다. 탁자 룰에 적혀있지 않는 것은 대부분 기본 룰을
따른다.`[1]`

## 기본 룰 ¶

  * 동풍전 남입, 반장선 서입 없음  

  * 동점일 경우 순위는 기가순  

  * 우마 5-10  

  * 최종집계때 100점대는 반올림  

  * 쿠이탕 있음  

  * 적도라 3개(5만, 5통, 5삭)  

  * 우라도라, 깡도라, 깡우라도라 있음  

  * 깡도라는 즉시 뒤집음  

  * 쿠이카에 있음(치는 패가 바뀌는 경우만, 퐁은 항상 가능)  

  * 동장은 화료연장, 남장은 텐파이연장  

  * 오라스에서 친이 톱일 경우 화료(텐파이유국)시에도 종료  

  * 오라스에서 유국시 공탁금은 톱이 가져감  

  * 더블론, 트리플론은 선하네(아타마하네, 선오름) - **1, R2, 3G**
  * 더블론, 트리플론 있음 - **4**
  * 형식텐파이 있음 (대기패를 자신이 모두 사용 중이여도 텐파이 인정)  

  * 5본장부터 2판묶기  

  * 올림만관 없음  

  * 들통있음 (0점은 속행)  

  * 쿠이핑후 형태는 예외없이 20부  

  * 리치 후 대기가 변하지 않는 안깡 가능 (단 보내기깡은 불가)  

  * 구종구패 있음 (연장)  

  * 사풍연타, 사깡유국 있음 (친 넘어감)  

  * 사가리치유국 없음  

  * 파오(책임지불) 없음  

  * 역만의 경우 4배역만까지 확인 됨  

## 역 관련 ¶

  * 팔연장 있음 (자일때 화료한 것도 카운터 됨, 역없이 팔연장만 화료 불가, 유국만관은 인정, 통상과 아오텐 전부 다른 역만관과 복합되지 않고 팔연장만 인정)  

  * 유국만관 있음 (화료취급, 여러명일 경우 해저가부터 선하네(아타마하네, 선오름))  

  * 삼연각(산렌코), 사연각(스렌코) 있음 (각각 2판, 역만)  

  * 인화 있음 (배만, 삼배만 이상일 시 인화는 계산되지 않음, 아오텐에서는 8판)  

  * 같은 패가 4장인 치또이츠 불가  

  * 안깡의 창깡은 국사무쌍만 가능  

  * 사암각단기, 국사무쌍13면대기, 순정구련보등, 대사희는 따로 기록하지만 더블역만으로 인정하지 않음  

  * 구련보등은 만수패 이외에도 성립  

  * 녹일색은 발이 없어도 성립  

## 3인 마작 룰 ¶

  * 사용하는 패는 2만~8만을 제외한 나머지 패  

  * 적도라 2개 (5통, 5삭)  

  * 1본장당 200점 가산 - **R2**
  * 1본장당 1000점 가산 - **3G, 4**
  * 노텐벌점 총 2000점 - **R2**
  * 노텐벌점 총 3000점 - **3G, 4**
  * 북은 객풍패 취급  

  * 쯔모 화료시 북가가 지불할 분량을 나머지 2명이 반씩 지불  

  * 우마는 1, 3위에게 1, 4위의 우마가 적용되며 2위는 우마 없음

적도라가 없다.
