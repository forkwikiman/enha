# 5. [KOF MIA](KOF%20MIA.md) ¶

  

★ 기본잡기  

  * 브뤼메르: 근접해서 ← or → + AC
  * 프뤽티도르: 근접해서 ← or → + BD  

★ 특수기  

  * 메시도르: → + A
  * 플로레알: ← + B
  * 프레리알: ← + D  

★ 필살기  

  * 방토즈: ← 모으고 → + A or C
  * 니보즈: ↓ 모으고 ↑ + B or D
  * 프리메르 소울라제: 공중에서 ↓↙← + B
  * 프리메르 포르티피에: 공중에서 ↓↙← + D
  * 제르미날: ← 모으고 → + B or D
  * 피에지: ←↓↙ + B or D  
┗ 레코르테: B or D

  * 방데미에르: 근접해서 ←↙↓↘→ + A or C  

★ 초필살기 (☆은 기게이지 소모 개수 표시)  

  * 테르미도르(☆): ↓↘→↓↘→ + A or C
  * 플뤼비오즈(☆): ↓↘→↓↘→ + B or D
  * 상 퀼로트 리베르테(☆): A, B, C, D
  * 상 퀼로트 에갈리테(☆☆): A, C, B, D
  * 레폴트 드 카프리스(☆☆): ↓↙←↙↓↘→ + B or D
  * 상 퀼로트 프라테르니테(☆☆☆): A, B, A, C
  * 에스프와르(☆): 상 퀼로트 프라테르니테 발동 후 ↓↙←↙↓↘→ + C  

★ 스타일리쉬 아트  

  * 01: A ▶ A ▶ C ▶ C ▶ A
  * 02: → + A ▶ C ▶ ← + D ▶ A
  * 03: B ▶ B ▶ D ▶ D ▶ A
  * 04: B ▶ B ▶ D ▶ ← + D ▶ A
  * 05: → + B ▶ D ▶ D
  * 06: B ▶ ↓ + D
  * 07: C ▶ C ▶ D ▶ C
  * 08: C ▶ C ▶ D ▶ D ▶ D
  * 09: C ▶ C ▶ D ▶ ↓ + D
  * 10: ← + C ▶ ← + D ▶ C ▶ D
  * 11: ← + C ▶ ← + D ▶ C ▶ ← + D (페인트)
  * 12: D ▶ D ▶ D ▶ D
  * 13: D ▶ D ▶ D ▶ ← + D ▶ A
  * 14: ↓ + D ▶ ↓ + D ▶ ← + D
  * 15: ← + D ▶ ← + D
