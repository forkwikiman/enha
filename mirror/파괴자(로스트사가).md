# 2. 기본기 ¶

  * DDD : 할버드를 3번 휘두른다. 막타는 적을 무릎꿇린다.
  * D꾹 : **파괴자 충과 장인들을 가려내는 척도.** 아무런 차징도 하지 않았다면 상대를 경직시키며, 1단계는 충격파 1단. 상대를 띄운다. 2단계는 충격파 2단. 충격파가 총 두번 나가고 마찬가지로 상대를 띄운다. 3단계는 드랍존 생성. 자신의 전방에 드랍존을 생성한다. 오묘하게 긴 선딜 때문에 끊길수도 있으니 조심. 숙련된 파괴자 고수들은 디꾹을 전략적으로 사용해 여러가지 변수를 만들어 내거나 사용하지 않지만 초보들은 팀에 기여도 못한채로 디꾹만 모으다가 한두번 떨어트리고 데스셔틀이든 스킬셔틀이든 되기 때문에 문제다.
거기다가 버젓이 콤보를 넣고 있는데 드랍시키는 초보들도 몇몇 존재한다.  

  * D꾹중 대시 :차징구르기.디꾹중 대시해서 자신이 채운 게이지의 효과를 낸다.만약 2단계 였으면 2단계 충격파가 나온다. 익숙해지면 드랍시키기 편해진다.
  * A DD : 위로 살짝 띄웠다가 내려 찍는다.
  * A D꾹 : 상대를 내려 찍으면서 내려온다. D키로 한번 더 띄울 수 있다.
  * 대시 D : 적을 높이 띄우는 공격. 평타나 점프공격으로 이어가기 쉽다.
  * 대시 D꾹 : 2연타로 휘두른다.D를 3번 더누르면 추가공격이 나가며 막타에 스턴이 걸린다.
  * 반격:적을 띄우고 내려치는 반격.
