# 1. [KOF MIA](KOF%20MIA.md) ¶

  

★ 기본잡기  

  * 이씨경성 : 근접해서 ← or → + AC
  * 녹주추루 : 근접해서 ← or → + BD  

★ 특수기  

  * 항아사월 : 초선번신 중 ↘ + B  

★ 필살기  

  * 우희헌배 : ↓↓ + A
  * 초선번신 : ↓↓ + B
  * 서시채시 : ↓↓ + C  

<우희헌배 스타일>  

  * 낙비금수 : ↓↘→ + A or C (연속 3회 입력)
  * 비모각 : ↓↘→ + B or D
  * 사돌아 초귀 : ←↓↙ + A or B or C or D  

<초선번신 스타일>  

  * 금련독보 천산 : ↓↘→ + B or D  
┠금련독보 파산 : ↓↘→ + B  
┃┠금련독보 지소 : ↓↘→ + B  
┃┗금련독보 마천 : ↓↘→ + D  
┗금련독보 붕산 : ↓↘→ + D  

┠금련독보 구풍 : ↓↘→ + B  
┗금련독보 선풍 : ↓↘→ + D

  * 미랑배월 : →↓↘ + A or C
  * 매비유엽 : ↓↙← + A or C  

<서시채시 스타일>  

  * 비연방조 : ←↙↓↘→ + A or C
  * 축융파군 : →↓↘ + A or C
  * 마고철조 : →↘↓↙← + A or C
  * 사돌아 단금 : 서시채시 이행 중 → + A or B or C or D  

★ 초필살기(☆은 기게이지 소모 개수 표시)  

  * 비적오의 난무사화장(☆) : ↓↙←↙↓↘→ + A or C
  * 비적오의 천수나한살 마갈(☆☆) : ↓↙←↓↙← + AC
  * 달기마향살(☆☆☆) : →↘↓↙←→↘↓↙← + AC (잡기)  

★ 스타일리쉬 아트  

  * 01 (우희헌배) : A ▶ A ▶ A ▶ B ▶ C ▶ D
  * 02 (우희헌배) : A ▶ A ▶ A ▶ B ▶ D
  * 03 (우희헌배) : C ▶ C ▶ C ▶ C
  * 04 (우희헌배) : → + C ▶ C ▶ C
  * 05 (우희헌배) : D ▶ D ▶ D ▶ D
  * 06 (우희헌배) : D ▶ D ▶ D ▶ ← + D (페인트)
  * 07 (초선번신) : C ▶ C
  * 08 (초선번신) : → + D ▶ D ▶ D ▶ D
  * 09 (초선번신) : → + D ▶ D ▶ D ▶ ← + D (페인트)
  * 10 (서시채시) : ↓ + A ▶ ↓ + A
  * 11 (서시채시) : C ▶ C
  * 12 (서시채시) : ↘ + C ▶ → + A
  * 13 (서시채시) : ↘ + C ▶ → + B
  * 14 (서시채시) : ↘ + C ▶ → + C
  * 15 (서시채시) : ↘ + C ▶ → + D
  * 16 (서시채시) : D ▶ C ▶ D
  * 17 (서시채시) : ↓ + D ▶ ↓ + D ▶ ↓ + C
  * 18 (서시채시) : ↓ + D ▶ ↓ + D ▶ → + C
  * 19 (우희헌배&초선번신) : ↓ + D ▶ → + D
  * 20 (우희헌배&초선번신) : ↓ + D ▶ ↓ + D
  * 21 (초선번신&서시채시) : A ▶ C
  * 22 (초선번신&서시채시) : B ▶ B ▶ D
  * 23 (우희헌배&초선번신&서시채시) : ↓ + B ▶ ↓ + B
