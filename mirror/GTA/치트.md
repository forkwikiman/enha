## 2. [GTA 2](GTA%202.md) ¶

player 0~7을 고를때 엔터를 눌러서 이름을 지우고 GOURANGA를 치고 아래의 치트를 써야 한다  
예) GOURANGA > EATSOUP > GODOFGTA > ...

  

이미 썼던 치트 지우는법 - 간단하다  
GOURANGA > EATSOUP(이걸 지울때) > GODOFGTA > EATSOUP (다시쓰고) > BUCKFAST (다른것)

  

치트 입력 후 Enter 눌렀을 때 나는 효과음이 그냥 아무렇게나 입력했을 때와는 다르게 나기 때문에 만약 치트치고 나서 나는 소리가 다른
치트와 다른 것 같다면 글자가 틀리지 않았는지 확인해야 한다.

5분간 무적  
XB : → A → ← → RB → ← A Y  
PS : → X → ← → R1 → ← X △

  

무적  
PS : ← X ← → ← R1 ← → X △

  

수배 별 내리기  
XB : RB RB B RT → ← → ← → ←  
PS : R1 R1 ○ R2 → ← → ← → ←

  

수배 별 올리기  
XB : RB RB B RT ← → ← → ← →  
PS : R1 R1 ○ R2 ← → ← → ← →

  

미끄러운 자동차  
XB : Y RB RB ← RB LB RT LB  
PS : △ R1 R1 ← R1 L1 R2 L1

  

술취한 상태  
XB : Y → → ← → X B ←  
PS : △ → → ← → □ ○ ←

  

근접공격시 폭발  
XB : → ← A Y RB B B B LT  
PS : → ← Ⅹ △ R1 ○ ○ ○ L2

  

버자드 소환  
XB : B B LB B B B LB LT RB Y B Y  
PS : ○ ○ L1 ○ ○ ○ L1 L2 R1 △ ○ △

  

코멧 소환  
XB : RB B RT → LB LT A A X RB  
PS : R1 ○ R2 → L1 L2 Ⅹ Ⅹ □ R1

  

리모 소환  
XB : RT → LT ← ← RB LB B →  
PS : R2 → L2 ← ← R1 L1 ○ →

  

Sanchez 소환  
XB : B A LB B B LB B RB RT LT LB LB  
PS : ○ Ⅹ L1 ○ ○ L1 ○ R1 R2 L2 L1 L1

  

스턴트 비행기 소환  
XB : B → LB LT ← RB LB LB ← ← A Y  
PS : ○ → L1 L2 ← R1 L1 L1 ← ← Ⅹ △

  

쓰레기차 소환  
XB : B RB B RB ← ← RB LB B →  
PS : ○ R1 ○ R1 ← ← R1 L1 ○ →

  

Caddy 소환  
XB : B LB ← RB LT A RB LB B A  
PS : ○ L1 ← R1 L2 Ⅹ R1 L1 ○ Ⅹ

  

Duster 소환  
XB : → ← RB RB RB ← Y Y A B LB LB  
PS : → ← R1 R1 R1 ← △ △ Ⅹ ○ L1 L1

  

Rapid GT 소환  
XB : RT LB B → LB RB → ← B RT  
PS : R2 L1 ○ → L1 R1 → ← ○ R2

  

낙하산  
XB : ← → LB LT RB RT RT ← ← → LB  
PS : ← → L1 L2 R1 R2 R2 ← ← → L1

  

빠른 달리기  
XB : Y ← → → LT LB X  
PS : △ ← → → L2 L1 □

  

능력치 충전  
XB : A A X RB LB A → ← A  
PS : Ⅹ Ⅹ □ R1 L1 Ⅹ → ← Ⅹ

XB : X LT RB Y ← X LT → A
