  * 마구 부수기 : 상대 다운시 ↑ + B

  * 금강중추(●) : ↓↘→ + A or B
  * 격진 : →↓↘ + A or B
  * 굉탄충 : 공격 버튼을 눌렀다가 뗀다
  * 부악 : 근접해서 →↘↓↙← + B  
┗ 쥬죠 일루타 : 부악 히트 후 ↓↘→ + B  

┗ 쥬죠 홈런 : 쥬죠 일루타 히트 후 ↓↘→ + B

  * 괄목 ・ 초격동 : 근접해서 →↘↓↙←→↘↓↙← + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 땅 흔들기 : 근접해서 ← or → + CD  

★ 특수기  

  * 마구 부수기 : 상대 다운시 ↑ + B  

★ 필살기  

  * 핫파 : ↓↘→ + A or B
  * 흔들기(●) : →↓↘ + A or B
  * 간지러운 코(●) : ↓↙← + C (모으기 가능)
  * 억지로 버틴다 : ↓↘→ + C
  * 바위 부수기 : → 모으고 ← + C  
┗ 무쇠머리 : 기/극모드에서 바위 부수기 후 ↓↘→ + C (3회 연속 입력)

  * 냅다 던지기 : 근접해서 →↘↓↙← + B  
┣ 멀리 날리기 : 냅다 던지기 히트 후 ↓↘→ + B  

┗ 마구 몰기 : 멀리 날리기 히트 후 ↓↘→ + C

┗ 후려갈기기 : 냅다 던지기 히트 후 ↓↘→ + C  

  * 내동댕이치기 : →↘↓↙← + A  
┣ 마구 갈기기 : 내동댕이치기 히트 중 ↓↘→ + A  
┗ 갈아 으깨기 : 내동댕이치기 히트 중 ↓↘→ + B  

★ 초오의  

  * 대담부적(략) 경악화산탄 : ↓↘→↓↘→ + AB 
  * 괄목! : 마구 갈기기 중 →↘↓↙←→↘↓↙← + AB
  * 홈런! : 냅다 던지기 히트 후 ↓↘→ + AB  

★ 잠재오의  

  * 초격렬보연(략) 앙천대분화 : ↓↘→↓↘→ + B 
  * 초격동이다앗! : 괄목! 중 →↘↓↙←→↘↓↙← + B  

★ 난무오의  

  * 초선렬개선(략) 자재대난타 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C,↓↘→ + B  
┣ 띄우기 루트 : 히트 후 A, B, C,↓ + C, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C,↓ + C, BC, ↓↘→ + B
