## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) ¶

※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 쇄완 ・ 괴격 : 근접해서 → + B
  * 쇄완 ・ 궤격 : 근접해서 ← + B  

★ 특수기  

  * 쇄각 ・ 진 : 상대 다운시 ↘ + C
  * 유리쇄 : 공중에서 근접해서 ↑제외 + B or C
  * 쇄권 ・ 맹호조 : ← + B
  * 쇄각 ・ 돌각 : 대쉬 중 C  

★ 필살기  

  * 백호조(●) : ↓↙← + A or B
  * 열포후 : →↓↘ + A  
┗ 열포후 ・ 열 : 열포후 히트 후 A 연타

  * 절포후 : →↓↘ + B  
┗ 절포후 ・ 멸 : 절포후 히트 후 B 연타

  * 비취쇄 : 근접해서 ←↙↓↘→ + C
  * 금강쇄 : 근접해서 →↘↓↙←→ + C  

★ 초오의  

  * 인과응보 : 근접해서 →↘↓↙←→↘↓↙← + AB  

★ 잠재오의  

  * 불구대천 : 근접해서 →↘↓↙←→↘↓↙← + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 쇄완 ・ 괴격 : 근접해서 ← 또는 → + CD
  * 유리쇄 : 공중에서 근접 + CD  

★ 특수기  

  * 쇄완 ・ 궤격 : 상대 다운시 근접해서 CD
  * 쇄각 ・ 진 : 상대 다운시 ↘ + C
  * 쇄권 ・ 맹호조 : ← + B  

★ 필살기  

  * 백호조(●) : ↓↙← + A or B
  * 열포후 : →↓↘ + A
  * 절포후 : →↓↘ + B
  * 비취쇄 : 근접해서 ←↙↓↘→ + 공격 버튼
  * 금강쇄 : 근접해서 →↘↓↙←→ + C  
┣ 괴 : 금강쇄 중 ←↙↓↘→ + A  

┣ 굉진 : 괴 이후 자동 발동  
┣ 궤 : 괴 중 ↓↙← + A  
┗ 열 : 괴 중 ↓↙← + B

┗ 악 : 금강쇄 중 ←↙↓↘→ + B  

┣ 굉진 : 악 이후 자동 발동  
┣ 승 : 악 중 ↓↙← + A  
┗ 추 : 악 중 ↓↙← + C

  * 백호습 : ↓↙← + C (기/극모드는 3회 연속 입력)  

★ 초오의  

  * 동천경지 : 근접해서 →↘↓↙←→↘↓↙← + AB
  * 인과응보 : ←↙↓↘→← + AB
  * 폭호빙하 : ↓↘→↓↘→ + AB   
┣ 용왕매진 : 폭호빙하 히트 중 ←↙↓↘→ + B  
┗ 경천동지 : 폭호빙하 히트 중 →↘↓↙←→↘↓↙← + B

  * 진천동지 : 괴 중 →↘↓↙←→↘↓↙← + AB  

★ 잠재오의  

  * 불구대천 : 근접해서 →↘↓↙←→↘↓↙← + B
  * 노발충천 : 악 중 →↘↓↙←→↘↓↙← + C  

★ 난무오의  

  * 발산도하 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, ↓↘→ + B  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, BC, ↓↘→ + B
