울산광역시 중구의 마을버스. 위 그림의 검붉은색 노선. 노선은 다음과 같다. (굵게 표시된 곳에서 시내버스로 환승 가능)

운행노선은 동덕현대A → 신한그린 → 장미아파트 → 평창아파트 → 성신고등학교 → 울산교회 → 중앙동주민센터 → 크레죤 → 국민은행 →
목살골목 → 일신스토어 → 향교삼거리 → 남운아파트 → 빅세일마트 → 21세기마트 → 우정아이파크 → 우정성당 → 선경1차 →
**우정초등학교앞삼거리** → 향교삼거리 → 일신스토어 → 목살골목 → 신협 → 국민은행 → 소방서사거리 →
<del>시계탑사거리</del>`[2]` → **박병원** → 안국한의원 → 역전시장 → 울산중고등학교 → 서울마트 → 동덕현대A.

'성안마을버스'사 출신 노선중 그 역사가 가장 오래된 노선. 2000년대 초반에는 15분, 중반부터는 10분 간격(주말엔 15분)으로
운행되었으나, 인수 직후에는 차 한대로 굴리느라 35~40분 간격으로 운행되었다.(편도 30분 가량 소요 + 휴식시간) 2012년 현재에는
1대가 추가되어 15~20분 간격으로 다소 줄어들었다.

등교시간 2회, 하교시간 1회에 한하여 함월고등학교 지원운행중에 있다.
