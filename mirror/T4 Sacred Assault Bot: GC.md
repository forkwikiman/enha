등장하는 유닛.

Code Name: Galactic Colossus  
Tech Level: Secret(비밀제작기체)  
Classification(분류) : Military Vehicle

에이언 일루미네이트의 T4 유닛인 거대한 2족보행병기인 갤러틱 콜로서스는 사람의 얼굴에 해당되는 부분에서 강력한 레이저를 발사하며, 사람의
손에 해당되는 부분은 적의 공군기체들이 빠른 속도로 달라붙게 만들어 충격을 줘 박살을 내 추락하게 만든다. 키가 굉장히 크기 때문에, 바다
속에 들어가도 상체부분이 보여서 이동하고 있다는 것을 알 수 있을 정도다. 잠수 상태에도 보이는 건 사이브런 네이처의 몽키로드 스파이더봇
정도다. 에이언 일루미네이트의 T4 유닛인 CZAR도 물속에 들어가 있는 갤러틱 콜로서스는 본무장이 아닌 보조무장인 미사일로 공격하게 된다.
벽은 밟아버리고, 물속에서 나올 때 옆에 어뢰가 있으면 그냥 쓱하고 지나가면서 날려버린다.
