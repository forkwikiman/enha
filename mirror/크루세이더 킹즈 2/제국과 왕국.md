## 2. 디시전으로 생성 가능한 작위 ¶

  * [로마 제국](%EB%A1%9C%EB%A7%88%20%EC%A0%9C%EA%B5%AD.md)(Roman Empire) : 비잔티움 황제위를 가지고 있는 상태에서 Latium + Venice + Ferrara + Sicily + Genoa + Capua + Apulia + Thrake + Antioch + Jerusalem + Alexandria + Athens + Tunis + Croatia 공작령을 남작단위까지 완벽하게 지배하고 있는 경우 활성화되는 로마 제국 재건 디시전을 통해 생성. 데쥬레는 재건 당시의 플레이어의 메인 작위릐 데쥬레를 빼았는다.
  * [신성 로마 제국](%EC%8B%A0%EC%84%B1%20%EB%A1%9C%EB%A7%88%20%EC%A0%9C%EA%B5%AD.md)(Holy Roman Empire) : 독일 왕위를 가진 상태에서 군주가 게르만인-가톨릭이고, 휘하 홀딩수(모든 성,교회,도시의 합)가 220 이상이고, 16살 이상의 성인이며 수행불능(incapable)상태가 아니고 명성(prestige)이 1000이상일 때 디시전이 활성화된다. 디시전 실행시 왕국법은 최하 왕권, 남성 선거제로 생성된다.
  * 인도 제국(Empire of India) <del>이 [인도 제국](%EC%9D%B8%EB%8F%84%20%EC%A0%9C%EA%B5%AD.md)은 당연히 아니다</del> : 라자스탄 제국, 벵골 제국, 데칸 제국의 황제위를 모두 가진 상태에서 [전륜성왕](%EC%A0%84%EB%A5%9C%EC%84%B1%EC%99%95.md)(Samrat Chakravartin) 디시전으로 생성.`[31]`
  * 이스라엘 왕국(Kingdom of Israel) : 유대교를 믿는 캐릭터가 Jerusalem + Ascalon + Galilee + OultreJourdain 공작령을 완전히 지배하고 예루살렘 백작령이 유대교로 개종되어 있는 상태에서 활성화 되는 이스라엘 왕국 재건 디시전을 통해 생성
  * 헝가리 왕국(Kingdom of Hungary) : 867년도로 시작해서 마자르 왕위를 가지고 군주가 헝가리인일 경우 Pest 공작령이나 Transylvania + Ungvar + Nyitra 지역을 완전히 점령하면 디시전을 통해 헝가리 왕국으로 바꿀 수 있다.
  * 레온 왕국(Kingdom of Leon) : 867년도로 시작해서 아스투리아스 왕위를 가지고 있는 상태에서 Leon + Asturias 공작령을 완전히 지배하고, Leon 백작령을 수도로 하는 상태에서 디시전이 활성화된다.
