**X-2 Antenna**

풀 내구도 공격력 : 65  
치명타 추가 공격력 : 23  
연사력 : 1.1408/초  
크리티컬 확률 배수 : 1배  
특수 능력 : EMP 공격. 로봇에겐 50 파워 아머에겐 20의 추가 데미지를 준다.  
무게 : 15 pound  
[V.A.T.S.](V.A.T.S..md) 소모 AP : 22 AP -> 근접 무기 스킬 50 이상일 경우 소모 AP가 35로
증가하면서 2배의 공격력으로 공격하는 특수 V.A.T.S. 공격인 고통 주파수(Pain Frequency) 공격이 추가  
내구도 : 125  
수리 가능 아이템 : 동종 무기  
응급 수리 대응 아이템 : 두손 근접무기  
요구 힘 : 7  
요구 스킬 : 근접 무기 75  
가격 : 6,000 캡  
가능한 개조 : 없음

  
[Old World Blues](Old%20World%20Blues.md)에서 추가된 자체 유니크 무기.

이름 그대로 원래는 X-2 송신 안테나 시설 (X-2 Transmitter Antenna Array)에 위치한 평범한 송신 안테나이다. 올드
월드 블루스의 메인 퀘스트 도중 싱크 탱크의 일원들이 평화주의장을 해제하기 위해 이것을 회수해오라고 시켜서 배달부는 이것을 말 그대로
뽑아오게 되는데 그걸 그대로 둔기로 휘두르는 것이다(...).

우람한 크기도 크기지만, 상당히 단순 무식하게 남자다운 방법으로 얻는 무기라서 남자다운 파괴력을 기대하게 되는 무기. 하지만 실제론 크리티컬
확률도 안습하고 무게도 무거운지라. 훨씬 쉽게 얻을 수 있고
[로보토마이트](%EB%A1%9C%EB%B3%B4%ED%86%A0%EB%A7%88%EC%9D%B4%ED%8A%B8.md)들이 자주
가지고 나오는 [양성자 도끼](%EC%96%91%EC%84%B1%EC%9E%90%20%EB%8F%84%EB%81%BC.md)에 크게
밀리는 편이다. 내구도도 양성자도끼보다 낮은데다 자체 유니크라서 응급 수리 퍽이 없으면 수리키트나 비싼 돈 주고 수리해야 한다. 특수
V.A.T.S 공격과 EMP 효과가 있다곤 하나 그 성능은 양성자 도끼와 동일하고, DT와 DR을 무시하는 EMP의 특성상 연사력이 더 빠른
양성자 도끼가 로봇 뚜껑을 따기엔 훨씬 적합하다.

양성자 도끼와 비교했을 때 기본 대미지가 약간 더 높은 것과 [고중량(Heavyweight)퍽](Perk/%ED%8F%B4%EC%95%84%EC%9B%83%20%EC%8B%9C%EB%A6%AC%EC%A6%88.md)으로 무게를 감소시킬 경우 양성자 도끼보다 가벼워 진다는 점
<del>그리고 정신나간 외관</del>이 장점이라고 할 수 있는 무기.
