## 1. 어간이 바뀌는 불규칙 활용 ¶

모음으로 시작되는 어미 앞에서 어간이 바뀐다.

  

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%95%9C%EA%B5%AD%EC%96%B4/%EB%B6%

### 1.1. ㄷ 불규칙 활용 ¶

어간의 [ㄷ](%E3%84%B7.md) 받침이 모음으로 시작되는 어미 앞에서 ㄹ 받침으로 바뀌는 활용이다. 동사에만 나타나고 형용사에는
나타나지 않는다.  

  * 깨**닫**\- + -아 → 깨**달**아
  * **눋**\- + -어 → **눌**어 cf)**눌**어붙다 = 눌어 + 붙다
  * **붇**\- + -어 → **불**어 cf)**불**어나다 = 불어 + 나다
  * **묻**〔問〕- + -어 → **물**어`[1]`
  * **싣**\- + -어 → **실**어
  * (규칙) **믿**\- + -어 → **믿**어  

위 ㄷ 불규칙 활용 용언 중 눋다, 붇다, 싣다는 각각 *눌다(*눓다), 불다(*붏다, *뿛다), 싫다 등으로 잘못 쓰이는 예가 많다.
‘붇다’는 ‘붙다’와 헷갈려서 그렇다고 생각할 수도 있는데, ‘불다’라는 동사도 이미 존재하는 데다, ‘눋다’처럼 안 그런 경우에도
헷갈리고, ‘싣다’처럼 오히려 잘못 쓰는 쪽이 혼동의 여지가 있는 경우에도 헷갈리고, 위 ‘묻다’는 혼동의 여지가 있는데도 불구하고 안
헷갈려서 그것만으로는 설명이 안 된다.  
한편, 잘못 쓰이는 경우에 /눌는/, /뿔는/, /실는/ 등으로 발현되거나 /눌타/(←눋다), /뿔키/(←붇기), /실치/(←싣지)와 같이
마치 ㅀ받침인 것처럼 뒤에 오는 어미의 자음을 거센소리로 만드는 현상이 나타나는 것은 ㄹ탈락을 시키지 않기 위함으로서 원래 ㄷ받침이었던 것을
언중이 아직 인식하고 있기 때문으로 보인다. 이영보래와도 상통하는 면이 있다.

### 1.2. 르 불규칙 활용 ¶

어간의 끝소리 '르'가 어미 '-아', '-어' 앞에서 ㄹㄹ로 바뀌는 활용이다.  

  * 구**르**\- + -어 → **굴러**
  * 모**르**\- + -았다 → **몰랐**다
  * 벼**르**\- + -어 왔던 → **별러** 왔던
  * 어간이 '-르'로 끝날 경우에는 러 불규칙(아래 참조)과 르 불규칙 활용이 동시에 존재하므로 둘 다 불규칙 활용이 된다. 
다만, '치르다', '따르다', '다다르다', '우러르다', '들르다' 등의 극히 일부의 용언은 예외적으로 어간이 ㅡ로 끝나는 다른 용언들과
똑같이 활용한다(치**르**\- + -어 → 치러, 다다**르**\- + -아 → 다다라, 들**르**\- + -어 → 들러).

  

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%95%9C%EA%B5%AD%EC%96%B4/%EB%B6%
88%EA%B7%9C%EC%B9%99%20%ED%99%9C%EC%9A%A9?action=edit&section=4)]

### 1.3. ㅂ 불규칙 활용 ¶

어간의 [ㅂ](%E3%85%82.md) 받침이 ㅜ로 바뀌는 활용이다. 어간이 ㅂ으로 끝나는 용언은 '굽다'〔曲〕, '꼽다'(plug
in), '뽑다', '씹다', '업다', '입다', '잡다', '접다', '좁다', '집다'와 같은 일부 예외를 제외하고는 거의 이렇게
활용된다.  

  * 아니꼽- + -어 → 아니**꼬워**
  * 무**덥**\- + -었다 → 무**더웠**다
  * (규칙) **잡**\- + -아 → **잡**아
대부분 '워'가 되나, 아래 두 경우는 예외적으로 '와'가 된다.  

  * **곱**\- + -아 → **고와**
  * **돕**\- + -아 → **도와**
'추위나 중노동으로 인해 손발의 감각이 둔하고 굳어서 잘 움직이지 않는다'는 뜻의 '곱다'는 규칙활용이다.  

  * 영희는 손이 **고왔**다 : 손이 예쁘고 맵시가 있다는 뜻.
  * 찬물에 빨래를 하고 나니 손이 **곱았**다 : 추워서 손이 굳었다는 뜻.  

원인은 중세 국어에서 순경음 비읍(ㅸ)의 음가가 사라진 것이다. 예를 들면, '덥-'의 종성 ㅂ은 본디 ㅸ이었고, 종성에서는 ㅂ으로
발음했으나 활용에 따라 초성으로 넘어가면 ㅸ으로 발음했다. 그러나 ㅸ의 음가가 사라지면서 초성의 ㅸ은 종성의 ㅸ이 ㅂ이 된 것과는 달리
복모음 ㅜ/ㅗ로 바뀌었다.  
여담으로 동남 방언에서는 복모음 ㅜ/ㅗ가 아닌 ㅂ으로 남아있다. <del>하이고 드브라</del>

  

'돕다', 혹은 '귀엽다' 의 경우 '도우니', '도우면'과 같은 활용형으로부터
'[도우다](%EB%8F%84%EC%9A%B0%EB%8B%A4.md)', '귀여우다'라는 형태가 새로 만들어지는 중으로 보인다.

88%EA%B7%9C%EC%B9%99%20%ED%99%9C%EC%9A%A9?action=edit&section=5)]

### 1.4. ㅅ 불규칙 활용 ¶

어간의 [ㅅ](%E3%85%85.md) 받침이 모음으로 시작하는 어미 앞에서 탈락하는 활용이다. '벗다', '솟다', '씻다' '뺏다'
등을 제외한 거의 대부분의 동사가 이렇게 활용되며 형용사로는 '낫다'가 유일하게 ㅅ 불규칙 활용이다.  

  * **낫**\- + -아 → **나**아
  * **긋**\- + -었다 → **그**었다
  * (규칙) **벗**\- + -어 → 벗어
원인은 중세 국어에서 반치음([ㅿ](%E3%85%BF.md))의 음가가 사라진 것이다. 예를 들면, '긋-'의 종성 ㅅ은 본디
ㅿ이었다. ㅜ/ㅗ로 바뀐 ㅸ과는 달리, 초성에서 음가가 완전히 소멸해 버렸기 때문에 ㅅ 받침이 탈락하는 효과가 되었다. 일부 방언에서는
아직도 ㅅ 발음이 남아서 /그서/와 같이 발음하기도 한다.

### 1.5. 우 불규칙 활용 ¶

어간의 끝 '우'가 '어' 앞에서 탈락하는 활용이다. '푸다'가 유일하다.  

  * **푸**\- + -어 → **퍼**
  * (규칙) **주**\- + -어 → 줘

## 2. 어미가 바뀌는 불규칙 활용 ¶

어떤 특정한 어간 뒤에서 어미가 바뀐다.

### 2.1. 거라 불규칙 활용 ¶

명령형 어미 '-아라/-어라'가 '-거라'로 바뀌는 활용이다. 동사 '가다' 또는 '-가다'로 끝나는 동사에서 나타난다.  

  * 가- + -**아라** → 가**거라**
  * 삼가- + -**아라** → 삼가**거라**
  * 들어가- + -**아라** → 들어가**거라**  

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%95%9C%EA%B5%AD%EC%96%B4/%EB%B6%
88%EA%B7%9C%EC%B9%99%20%ED%99%9C%EC%9A%A9?action=edit&section=9)]

### 2.2. 너라 불규칙 활용 ¶

명령형 어미 '-아라/-어라'가 '-너라'로 바뀌는 활용이다. 동사 '오다' 또는 '-오다'로 끝나는 동사에서 나타난다.  

  * 오- + -**아라** → 오**너라**
  * 돌아오- + -**아라** → 돌아오**너라**
  * 실제 구어에서는 **와라** 꼴도 나타나긴 하지만 **너라** 꼴로 나타나는 동사는 '오다' 계통 뿐이다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%95%9C%EA%B5%AD%EC%96%B4/%EB%B6%
88%EA%B7%9C%EC%B9%99%20%ED%99%9C%EC%9A%A9?action=edit&section=10)]

### 2.3. 러 불규칙 활용 ¶

'르'로 끝나는 어간 뒤에서 '-어'가 '-러'로 바뀌는 활용이다.  

  * 이르〔至〕- + -**어** → 이르**러**
  * 누르〔黃〕- + -**어** → 누르**러**
  * 푸르- + -**어** → 푸르**러**  

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%95%9C%EA%B5%AD%EC%96%B4/%EB%B6%
88%EA%B7%9C%EC%B9%99%20%ED%99%9C%EC%9A%A9?action=edit&section=11)]

### 2.4. 여 불규칙 활용 ¶

어미 '-아'가 '-여'로 바뀌는 활용이다. '하다' 또는 '-하다'로 끝나는 용언에서 나타난다.  

  * 하- + -**았**다 → 하**였**다
  * 영원하- + -**아** → 영원하**여**  

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%95%9C%EA%B5%AD%EC%96%B4/%EB%B6%
88%EA%B7%9C%EC%B9%99%20%ED%99%9C%EC%9A%A9?action=edit&section=12)]

### 2.5. 오 불규칙 활용 ¶

어미 '어라/아라'가 '오'로 바뀌는 활용이다. 달다(말하는 이가 듣는 이에게 어떤 것을 주도록 요구하다는 동사)가 유일하다.  

  * 달- + -**아라** → 다**오**  

'달' 다음에 '아라'가 오면서 '달아라'가 아니라 '다오'로 변하는데, 어미만 변화하는 것으로 취급된다. 받침 'ㄹ'이 'ㄴ, ㅂ, ㅅ,
오'와 결합할 경우, ㄹ이 탈락하기 때문이다. 따라서 어미인 **아라**만 **오**로 변화한 것으로 본다.

### 3.1. ㅎ 불규칙 활용 ¶

어간의 [ㅎ](%E3%85%8E.md) 받침이 탈락하고 어미가 축약되거나 탈락하는 활용이다. 주로 '파랗다'와 같은 형용사에서
나타난다.  

  * 그**렇**\- + -**아** → 그**래**
  * **놓**\- + -**아** → **놔**
  * 동그**랗**\- + -**은** → 동그**란**
  * 파**랗**\- + -**으**면 → 파**라**면
  * 하**얗**\- + -**았**다 → 하**얬**다
  * 허**옇**\- + -**어** \+ 지다 → 허**예**지다
  * (규칙) **좋**\- + -**아** → **좋아** (발음은 [조아](%EC%A1%B0%EC%95%84.md)로 한다.)
  * (규칙) **낳**\- + -**아** → **낳아** (**나아**가 아니다!!)
