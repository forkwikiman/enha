점프는 공중에서 형태가 고정되고 명확해야 하며, 높이가 충분해야 한다.  
스플릿 계열 점프를 수행할 시 다리 각도가 135˚ 이상이 되어야 한다.  
모든 점프는 한 발, 혹은 양 발로 도약 가능하다.`[1]`

  * 공중에서 몸을 180˚ 회전할 경우 : +0.1
  * 링 : +0.1
  * 백벤드 : +0.2

## 1. 버티컬 점프 ¶

매트와 수직을 이루는 점프로, 신체의 회전을 동반한다.  

  * 0.1 : 180˚ 회전, 양 다리를 앞으로 굽힘/빠세/허리 높이의 프론트, 사이드, 백 자세
  * 0.2 : 360˚ 회전, 기본 버티컬 자세/양 다리를 앞으로 굽힘/허리 높이의 프론트, 사이드, 백 자세  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=2)]

## 2. 카브리올 점프, 아치 점프 ¶

카브리올 점프는 공중에서 발바닥끼리 맞부딪치는 동작을 수반한다. 발레의 [카브리올](http://youtu.be/cI2Qr8_8Ytc)에서
따온 기술이다.

  

  * 0.1 : 카브리올`[2]`  

아치 점프는 공중에서 몸을 활꼴로 젖히는 점프이다.

  

  * 0.1 : 아치 점프  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=3)]

## 3. 가위뛰기 ¶

가위뛰기는 다리를 앞뒤로 교차하는 동작을 하며, 다리를 교차하는 높이나 교차 동작에 따라 점수가 달라진다.  
[동작의 이해를 위한 유투브 영상](http://youtu.be/MO1mBCGP9UU?t=33s)

  

  * 0.1 : 허리 높이와 수평한 위치에서 가위뛰기
  * 0.2 : 허리보다 높은 위치에서 가위뛰기
  * 0.3 : 링 자세로 가위뛰기  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=4)]

## 4. 스태들 점프 ¶

사이드 스태들이라고도 하며, 양발로 도약해 다리를 옆으로 찢는 동작이다.  
(점프하는 힘은 중심에서 시작하여 양쪽 발끝으로 이동하여, 이 힘으로 체공한다)

  

  * 0.2 : 기본 자세
  * 0.3 : 상체를 앞으로 숙임  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=5)]

## 5. 코작 점프 ¶

공중에서 앉은 자세를 만든다. 이때 다리는 다양한 자세로 바꿀 수 있다. 보통 180˚ 회전이나 360˚ 회전을 동반하는 경우가 많다.

  

  * 0.1 : 기본 자세
  * 0.2 : 신체를 앞으로 숙임/편 다리를 위로 올림
  * 0.3 : 굽혔던 다리로 링 자세`[3]`  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=6)]

## 6. 링 점프 ¶

한 쪽 다리, 혹은 양 쪽 다리를 뒤로 굽혀서 머리에 닿게 하는 링 자세를 만들어야 한다. 스플릿 링 립과 다르게 몸이 수직에 가깝게 놓이게
된다.

  

  * 0.1 : 한 쪽 다리만 링
  * 0.4 :양 쪽 다리 모두 링  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=7)]

## 7. 포에테 점프 ¶

도약하면서 신체를 180˚ 회전한다. 도약한 다리와 착지한 다리가 같은 점이 앙트레라세와의 차이점이다.  
[이해를 돕기 위한 영상](http://youtu.be/DzqQNlaahjs)

  

  * 0.2 : 기본 자세/공중에서 스태그
  * 0.3 : 공중에서 스플릿/한 발 링/스태그 링/공중에서 신체 회전
  * 0.4 : 공중에서 백벤드 스태그
  * 0.5 : 공중에서 양 발 링  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=8)]

## 8. 앙트레라세 ¶

신체를 회전한다는 점에서는 포에테 점프와 비슷하지만, 도약한 다리와 착지한 다리가 다르다는 점이 포에테 점프와의 차이점이다.

  

  * 0.4 : 기본 자세
  * 0.5 : 공중에서 스플릿/한 발 링  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=9)]

## 9. 스플릿/스태그 립 ¶

턴 점프와 함께 선수들이 제일 보편적으로 구사하는 점프 종류이다. 스플릿 시 다리 각도가 135˚ 이상 되어야 하며, 그보다 각도가 작으면
인정되지 않는다.

  

  * 0.1 : 스태그 립
  * 0.2 : 스플릿 립/스태그 링
  * 0.3 : 스플릿 링/백벤드 스태그 립
  * 0.4 : 백벤드 스플릿 립  

한편, 2연속 스플릿 립과 3연속 스플릿 립이 있는데, 이는
[멀리뛰기](%EB%A9%80%EB%A6%AC%EB%9B%B0%EA%B8%B0.md)의 3단뛰기와 비슷한 매커니즘이라 이해하면 된다.
착지한 발로 그대로 다시 도약하는 방식으로 점프해야 한다.

  

  * 0.4 : 2연속 스플릿 립
  * 0.5 : 3연속 스플릿 립  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=10)]

## 10. 다리 교차 점프 ¶

공중에서 다리를 펴거나 구부린 상태로 엇갈리는 동작을 수반한다. 보통 리듬체조 보다는 기계체조에서 자주 쓰인다.  
[이해를 위한 유투브 영상](http://youtu.be/t67LeuYJIL0)

  

  * 0.2 : 기본 자세/스태그
  * 0.3 : 스플릿/스태그 링
  * 0.4 : 한 쪽 링/백벤드 스태그 립
  * 0.5 : 백벤드 스플릿  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=11)]

## 11. 턴 점프 ¶

뒤를 본 상태에서 도약해 몸을 180˚ 회전하면서 앞으로 착지한다. 스플릿 종류와 함께 리듬체조 선수들이 많이 구사하는 점프 종류.

  

  * 0.3 : 스태그 턴 립
  * 0.4 : 스태그 턴 링/다리교차가 들어간 스태그 턴 립
  * 0.5 : 턴 립/다리교차가 들어간 턴 립/다리교차가 들어간 스태그 턴 링  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%A6%AC%EB%93%AC%EC%B2%B4%EC%A1%B
0/%EC%A0%90%ED%94%84%20%EB%82%9C%EB%8F%84?action=edit&section=12)]
