# 1. [KOF XI](KOF%20XI.md) ¶

★ 잡기  
밭다리 후리기 : (근접시) → or ← + C or D  

★ 특수기  
장저 훅 : → + C  
미들 킥 : → + D  
슬라이더 : 맞짱 승부 사용중 ↘ + B  
니 어퍼 : 맞짱 승부 사용중 ↘ + D  

★ 필살기  
\- 장저 훅 or 미들 킥 -  
라이트 훅 : 장저 훅 or 미들 킥 사용중 →  
장저 훅 : 라이트 훅 or 백 너클 사용중 →  
가이 휠 : 라이트 훅 or 백 너클 사용중 ↑  
가이 드릴러 : 라이트 훅 or 백 너클 사용중 ↗  
슬라이더 : 라이트 훅 or 백 너클 사용중 ↓  
백 너클 : 라이트 훅 사용중 ←  
프런트 스텝 : 백 너클 사용중 ↘  
가이 드릴러 : 프런트 스텝 사용중 ↑  
니 어퍼 : 프런트 스텝 사용중 ↗  

\- 미들 킥 -  
백 토네이도 : 미들 킥 사용중 ← + D  
프런트 스텝 : 미들 킥 사용중 ↘  
가이 드릴러 : 프런트 스텝 사용중 ↑ 또는 미들 킥 사용중 ↗  
가이 휠 : 미들 킥 사용중 ↑  
슬라이더 : 미들 킥 사용중 ↓  
프로그 다운 : ↓↓ + A or C  
가이 드릴러 : 프로그 다운 사용중 ↓  
브릿지 스웨이 : ↓↙← + E  
프런트 스텝 : 브릿지 스웨이 사용중 → + 아무버튼  
슬라이더 : 브릿지 스웨이 사용중 ↓ + B or D  
백 너클 : 브릿지 스웨이 사용중 ← + A or C  
백 토네이도 : 브릿지 스웨이 사용중 ← + B or D  
브릿지 스웨이 어퍼 : 브릿지 스웨이 사용중 ↑ + A or C  
가이 휠 : 브릿지 스웨이 사용중 ↑ + B or D  
프로그 다운 : 브릿지 스웨이 사용중 ↓ + A or C  
가이 드릴러 : 프로그 다운 사용중 ↑  
가이 스파이더 : 브릿지 스웨이 사용중 (근접시) ←↙↓↘→ + E  

\- 맞짱 승부 -  
가이 드릴러 : 맞짱 승부 사용중 →↓↘ + A or C  
가이 휠 : 맞짱 승부 사용중 ↓↙← + B or D  
사이 스파이더 : 맞짱 승부 사용중 (근접시) ←↙↓↘→ + E  

★ 초필살기  
러시 : ↓↘→↓↘→ + B or D  
러시 추가 공격 : 러시 사용중 →, →, ←, ↘, ↑, →, ↓, →  
러시 피니시 : 러시 추가 공격 사용중 ↑ or ← or ↓  
고독의 링 : ↓↘→↓↘→ + A or C  

★ 리더 초필살기  
맞짱 승부 : ↓↘→↓↘→ + E
