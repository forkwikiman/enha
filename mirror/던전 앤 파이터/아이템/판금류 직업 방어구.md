### 2 [아수라](%EC%95%84%EC%88%98%EB%9D%BC%28%EB%8D%98%EC%A0%84%20%EC%95%A4%20%ED%8C%8C%EC%9D%B4%ED%84%B0%29.md) ¶

  * 블루 프로미넌스 (Lv 25)  

머리어깨 : 수 속성 저항 + 4~6  
상의 : 파동각인 LV +1  
하의 : HP MAX + 110~115  
신발 : 이동속도 +4%,화 속성 저항 + 4~6  
벨트 : 인벤토리 무게 한도 +3kg  
세트 효과 :빙결내성 + 17, 피격시 3%확률로 적을 빙결

상의에 파동각인이 붙어있기 때문에 도움이 많이 된다. 초합금 고르아투스 하의와 같이 입어서 파동각인을 극대화하는 용도로도 쓴다.  
  

  * [초합금 고르아투스](%EC%B4%88%ED%95%A9%EA%B8%88%20%EA%B3%A0%EB%A5%B4%EC%95%84%ED%88%AC%EC%8A%A4.md)  

  * [칠흑의 베르테스](%EC%B9%A0%ED%9D%91%EC%9D%98%20%EB%B2%A0%EB%A5%B4%ED%85%8C%EC%8A%A4.md)  

  * 고요의 정령 (Lv 40)  

머리어깨 : 파동검 지열 Lv+1  
상의 : 진공참 LV +1  
하의 : MP MAX + 72~86,HP MAX + 72~86,  
신발 : 이동속도 +5%,점프력 +30  
벨트 : 인벤토리 무게 한도 +3kg,회피율 + 2%  
세트 효과 :MP 분당 30회복,캐스트속도 + 6%, 파동 해제, 수라진공참 Lv+1

엠회복, 캐스트 속도를 보고 크루세이더가 입었던 시절도 있었다. 수라진공참의 레벨이 오르면 데미지가 20%에서 40% 증가되고 크기도 소폭
커져서 끼기도 했던 것. 이후엔 그런거 없다.  
  

  * 비전 리퍼 (Lv 45)  

머리어깨 : 공격속도 + 2%  
상의 : 진공참 LV +1,HP MAX + 98~105  
하의 : 히트리커버리 + 50  
신발 : 이동속도 +5%,점프력 +20  
벨트 : 인벤토리 무게 한도 +3kg,화속성 저항 + 5~6, 수속성 저항 + 5~6  
세트 효과 파동검 폭열, 작열 파동진 Lv+1

고요의 정령이랑 용갑 밸러 퀴러스 사이에 껴서 참 애매한 세트. 파동각인도 없다. 덤으로 45제라 숄더 벨트를 만들어야한다.  
이걸 만들어서 쓰느니 고요의 정령 세트로 45~50레벨까지 버티는 게 훠어어어어어어얼씬 이익.  
  

  * 용갑 밸러 퀴러스 (Lv 50)  

머리어깨 : 진공참 Lv+1  
상의 : 파동각인 LV +1  
하의 : HP MAX +  
신발 : 이동속도 +5.5%,둔화내성 + 10  
벨트 : 인벤토리 무게 한도 +3kg,히트리커버리 + 50  
세트 효과 : 회피율 + 3%,히트리커버리 + 100, 부동명왕진, 작열 파동진 Lv+1

  

  * 타오르는 파동 (Lv 55)  

머리어깨 : MP 분당 18회복  
상의 : 파동검 폭염 LV +1  
하의 : HP MAX + 140~158,MP MAX + 141~160  
신발 : 이동속도 +5.5%,MP MAX + 45~55  
벨트 : 인벤토리 무게 한도 +3kg,캐스트속도 + 2%,MP MAX + 49~52  
세트 효과 : 지능 + 34~42,캐스트속도 + 3%,마법 크리티컬 히트 + 2%,부동명왕진, 진공참 Lv+1

아수라 세트에 드디어 지능이 붙었지만 파동각인이 안붙어서 뭔가 애매한 세트.

  * 순백의 갑옷 : 셀레스타 (Lv 40)

축복도 붙어있다.

  * 신야빙의 (Lv 50, 머리어깨와 벨트는 레시피로 제작가능)

세트 효과 :공격속도 + 2%,적중률 + 2%,물리 크리티컬 히트 + 2%,대회전격 Lv+1

숄더, 거들, 신발을 입을 수 있기 때문.
