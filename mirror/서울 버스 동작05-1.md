## 1. 개요 ¶

[2012년](2012%EB%85%84.md) [6월](6%EC%9B%94.md) 1일부로 신설된
[서울특별시](%EC%84%9C%EC%9A%B8%ED%8A%B9%EB%B3%84%EC%8B%9C.md)
[동작구](%EB%8F%99%EC%9E%91%EA%B5%AC.md)의 마을버스인 동작 05-1번은 대방역에서 출발하여 대방역 ->
대방동현대아파트 -> 대방주공아파트 -> 대림아파트 후문 -> 7호선
[보라매역](%EB%B3%B4%EB%9D%BC%EB%A7%A4%EC%97%AD.md) -> 보라매공원 -> 대방중학교 -> 2호선
[신대방역](%EC%8B%A0%EB%8C%80%EB%B0%A9%EC%97%AD.md) -> 롯데백화점 -> 7호선 [신대방삼거리역](/
wiki/%EC%8B%A0%EB%8C%80%EB%B0%A9%EC%82%BC%EA%B1%B0%EB%A6%AC%EC%97%AD) ->
코오롱하늘채아파트 -> 대림아파트 후문 -> 대방동주공아파트 -> 공군회관 -> 대방역으로 돌아온다.

  

운행업체는 보라매운수`[1]`로, 배차간격은 13~20분 정도이다.

  

분리 전의 자세한 정보는
[동작05](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%20%EB%8F%99%EC%9E%9105.md)번
문서를 참고바란다.

## 2. 역사 ¶

  

2012년 5월 31일까지는 동작 05번 단일 노선으로 운행하였으나, 2012년 6월 1일부로 동작 05번과 동작 05-1번으로 분리하여
현재와 같이 운행중이다. 안 그래도 복잡하게 이리저리 쑤시고 다니던 노선인데 노선 변경 이후 노선파악이 더 어려워져서 헛갈려하는 승객들이
많다. 단적인 예로 대방주공아파트 정류장에는 05번 대방역 방향, 신대방역 방향, 05-1번 대방역 방향, 신대방역 방향 이렇게 4종류의
05번 버스가 정차한다. 하지만 도로구조상 반대편에는 [하나빼고](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%20%EB%8F%99%EC%9E%9103.md) 전혀
정차하지 않는다. 결론적으로는 기존 동작05번 단일노선에서 보라매공원 - 보라매역 구간을 연장한 형태라서 기존에 다니던 곳에 전부 가기는
**한다만** 신대방삼거리역 이남 구간이 각각 일방순환 형태이기 때문에 방향을 잘못 타서 한 바퀴 돌게 되면 슬퍼지게 되고 분리 이전보다
배차간격도 늘어난 느낌이다. 지금은 그나마 6대나 다녀서 그렇지, 초창기때에는 4대로 20분 배차를 겨우 유지할 정도였다... 그러나
2014년 5월 19일 부로 다시 한대를
[동작05번](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%20%EB%8F%99%EC%9E%9105.md)에
보내서 결국 5대로 운행하게 되었다.

  

또한, 2013년 1월 24일 첫차부터 **기상청, 수도여고, 대방중을 경유하도록 노선이 변경**되었다.

  

그렇게 해서 대방역 -> 대방주공아파트 -> 대림아파트후문 -> 대방교회 -> 보라매역 -> 보라매공원 -> 대방중학교 -> 신대방역 ->
롯데백화점 -> 신대방삼거리역 -> 대림아파트후문 -> 대방주공아파트 -> 공군회관 -> 대방역 으로 운행된다.

  

참고로 대방역 출발 막차는 [회차지점](%ED%9A%8C%EC%B0%A8%EC%A7%80%EC%A0%90.md)인 신대방역에서
종착한다.`[2]``[3]`
