## 4. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

★ 드라이브 - 참신(斬神 / God Slash) : 상대의 공격을 막아내는 반격기입니다.  
★ 오버 드라이브 - 귀신(鬼神 / Demon God)

  

★ 필살기  

  * 홍련(紅蓮 / Crimson Lotus) : 돌진 공격을 합니다. 곡옥을 1개 사용합니다.
  * 귀신차기(鬼蹴 / Demonic Leg) : 앞으로 고속 이동합니다.곡옥을 1개 사용합니다.  
┗ 염마(閻魔 / Hades) : 상대를 쳐올리는 공격입니다. 곡옥을 1개 사용합니다.

  * 연꽃(蓮華 / Lotus) : 2단 공격입니다. 곡옥을 2개 사용합니다.
  * 반딧불이(火蛍 / Firefly) : 상대를 쳐올리는 공격입니다. 곡옥을 2개 사용합니다.
  * 잔철(残鉄 / Rending Steel) : 2단 베기입니다. 곡옥을 3개 사용합니다.
  * 동백(椿祈 / Camellia) : 공중에서 상대를 공격합니다. 곡옥을 3개 사용합니다.
  * 봉마진(封魔陣/ Earthbind) : 장풍을 베면 상대가 날린 장풍을 무력화시키는 방패인 봉마진을 발생시킵니다.
  * 악도(咢刀) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 허공진 질풍(虚空陣·疾風 / Empty Sky Form: Summer's Advance`[13]`) : 앞쪽으로 강력한 공격을 합니다. 버튼을 누르고 있으면 공격 타이밍을 바꿀 수 있습니다. 곡옥을 4개 사용합니다.
  * 허공진 설풍(虚空陣·雪風 / Empty Sky Form: Winter's Reposte`[14]`) : 강력한 반격기입니다. 곡옥을 4개 사용합니다.
  * 허공진오의 몽환(虚空陣奥義·夢幻 / Empty Sky Form Forbidden Art: Dream State`[15]`) : 사용 후에 일정 시간 동안 공격력이 오르고 필살기를 마음껏 사용할 수 있습니다. 곡옥을 8개 사용합니다.  

★ 아스트랄 히트  

  * 허공진오의 악멸(虚空陣奥義·悪滅 / Empty Sky Form Forbidden Art: Perish Evil`[16]`) : 아주 강력한 반격기입니다.
