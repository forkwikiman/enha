# 1 개요 ¶

madballs라는 고무공 장난감 캐릭터들이 나와서 전쟁을 벌이는 아케이드 슈팅게임.  
2009년 7월 [XBLA](XBLA.md)로 발매. 2009년 9월에는 [스팀](%EC%8A%A4%ED%8C%80.md)을
이용한 PC판이 출시되었다. 지원하는 언어는 프랑스어 영어, 스페인어, 독일어, 포루투갈어 그리고 <del>번역기</del>**한글**이다.
놀랍게도 완전 한글화가 되어 있다.(PC판 이야기 XBLA판은 확인 바람)

# 2 스토리 ¶

강력한 힘을 지닌 선조들의 유물을 찾던 오큐러스 오르버스가 스코치드 외인부대인 혼헤드의 공격을 받고 한 행성에 불시착하게 된다. 그곳에서
B*D*I 대원들에게 사정을 설명하고 함께 유물을 찾게 되는데 또 다른 세력의 존재를 알게 된다. B*D*I와 스코치드 중 한 진영을 골라
10스테이지로 구성된 스테이지를 클리어 하는 싱글 플레이가 있다. 하지만 싱글은 그저 락을 풀고 게임 감각을 익히는 연습일뿐. 이 게임의
진가는 멀티플레이.  

# 3 등장 무기 ¶

12종류의 무기가 등장한다. 무기를 사용하기 위해서는 싱글이나 멀티플레이에서 점수를 얻으면 사용가능 하고 그 무기로 일정 수준이상의 피해를
입히면 보조 모드가 사용가능하다. B*D*I와 스코치드의 무기는 구분되어 있어 각 진영에 맞는 무기만 사용할 수 있지만 역시 일정 점수를
얻게되면 진영에 구분없이 모든 무기를 사용할 수 있게된다.  

  

무기는 모두 충격, 에너지, 냉기, 화염 4가지 속성 중 두개의 속성을 지니는데 상성은 충격 <-> 에너지, 냉기 <-> 화염으로 어느 한
속성에 내성을 지닐경우 반대 속성이 약점이 되며 나머지 두개에는 보통의 피해를 받는다. 예를 들어 냉기에 내성이 있는 젠킨스는 화염이
약점이고 에너지와 충격에 대해서는 일반적인 피해를 받는다. 충격은 맞은 적을 밀어내는 효과, 에너지는 관통 공격, 냉기는 잠시 이동속도가
느려지고 화염은 잠시 동안 체력이 조금씩 깍이는 추가 효과가 있다. 약점 속성이나 내성 속성의 공격을 명중 시킬 경우 해당 이펙트와 사운드가
존재한다.  

  

직업, 진영 관계 없이 사용가능한 투척 무기인 수류탄과 화염병도 존재한다. 수류탄은 3개 화염병은 2개 까지 소지가능하다.  

  

  * B*D*I의 무기  

    * **스티처** : 주 모드는 충격, 보조 모드는 에너지 속성의 머신건. 처음부터 선택 가능한 기본 무기.
    * **샷티** : 주 모드는 화염, 보조 모드는 냉기 속성의 샷건틱한 무기로 양쪽 모드 다 사정거리가 굉장히 짧지만 위력도 좋고 연사도 빠르다.
    * **비머** : 주 모드는 에너지 속성의 레이저빔으로 강력한 레이저 광선을 직선으로 발사한다. 연사가 느린것이 단점. 보조 모드는 작은 미사일을 여러 발 발사한다. 이쪽은 탄속이 느린것이 단점.
    * **버처** : 주 모드는 충격, 보조 모드는 에너지 속성의 발칸. 스티처와 비교하면 화력은 더 강하지만 사정거리가 짧고 발사시에 이동속도가 느려진다.
    * **로버** : 주 모드는 냉기속성의 곡사포를 발사한다. B*D*I 무기 중에서 최고의 위력과 넓은 스플래시 범위, 그리고 발사버튼을 유지할 경우 사정거리가 늘어나서 먼곳까지 쏠 수 있지만 근접 공격에 취약하고 탄속과 연사가 느리다. 그리고 곡사인 관계로 천장이 있는 곳에서는 무용지물. 보조 모드는 화염 속성의 5연발 로켓. 역시 곡사로 날아가지만 5연발인 점을 이용해서 뿌리듯이 견제하는 무기. 주 모드와 달리 사정거리 조절이 안된다.
  * **슬라이서** : 주 모드는 화염속성으로 바닥면을 타고 직선으로 계속 나아가며 벽이 있으면 벽을 따라 계속 직진한다. 즉, 벽을 관통한다. 위력도 상당하고 벽을 관통하기 때문에 주로 침략모드에서 터렛을 부술 때 유용하다. 보조 모드는 냉기 속성 어뢰(?)를 발사하는데 벽이나 적에게 부딪히거나 발사버튼을 누르면 폭발한다. 위력은 강한편.  

  * 스코치드의 무기  

    * **럽터** : 주 모드는 충격 속성의 표창. 스티쳐에 비해서 연사는 느리지만 사정거리가 길다. 보조 모드는 에너지 속성의 빔 공격.
    * **스토머** : 주 모드는 에너지 속성의 연사 공격. 버처와 비슷하지만 위력이 떨어지고 사정거리가 길다. 보조모드는 충격 속성의 발칸 공격으로 발사전에 약간 준비시간이 필요하고 발사 중에는 이동이 불가능하다. 물론 그만큼 위력은 발군.([헤비웨폰가이](%ED%97%A4%EB%B9%84%EC%9B%A8%ED%8F%B0%EA%B0%80%EC%9D%B4%28%ED%8C%80%20%ED%8F%AC%ED%8A%B8%EB%A6%AC%EC%8A%A42%29.md)?)
    * **리퍼** : 주 모드는 냉기, 보조 모드는 화염 속성의 빔 공격. 3연발 공격을 한다. 사정거리와 위력 모두 표준적인 무기.
    * **크래클러** : 주 모드는 화염, 보조 모드는 냉기 속성의 화염, 냉기방사기. 
    * **주카** : 주 모드는 화염, 보조 모드는 냉기 속성의 로켓. 주 모드는 연사가 느리고 보조 모드는 연사는 빠르지만 위력이 떨어진다.
  * **스트라이커** : 주 모드는 충격 속성의 작살을 발사한다. 비머의 충격 버전. 보조모드는 위성에서 발사되는 에너지 속성 레이저로 보조모드시 이동이 불가능해지고 조준모드가 된다. 폭격할 지점을 클릭하면 1초후 폭격이 떨어진다.  

# 4 등장 캐릭터 ¶

  

진영별로 5개의 공격병, 지원병, 파워병, 은신병, 비행병이 존재한다. 각각 두개의 스킬이 있으며 스킬은 체력옆 게이지가 가득차야
사용가능하다. 게이지 회복속도는 사용한 스킬에 따라 다르고 스킬 버튼을 눌러서 사용하는 탭 스킬과 1초 이상 버튼을 유지해서 발동시키는 차지
스킬이 있다.  

  

직업마다 체력과 이동속도의 차이가 있는데 이동속도는 파워병 -> 공격병 -> 지원병 -> 비행병 -> 은신병 순서로 빠르고 체력은 반대이다.  

  

  * B*D*I  

    * **오큐러스 오르버스** : 공격병. 탭 스킬은 빠르게 짧은 거리를 대시하는 스피드질주. 차지 스킬은 몸통박치기로 공격하는 스피드돌격. 약점은 없다. 물론 내성도 없다.
    * **젠킨스** : 지원병. 탭 스킬은 자신과 주변 아군을 회복시키는 필드를 생성하는 라이프 버블. 차지 스킬은 스위치 폭탄을 설치하는 사이언스 폭탄. 내성은 냉기 약점은 화염.
    * **로보** : 파워병. 탭 스킬은 가까운 적의 무장을 해제시키는 전기 채찍. 차지 스킬은 범위안 모든 적의 무장을 해제시키는 매그노펄스. 내성은 충격 약점은 에너지.
    * **에이돌론** : 은신병. 탭 스킬은 수류탄을 던져 가까이 있는 적의 눈을 멀게 하는 화이트 아웃. 차지 스킬은 일정 시간 동안 투명 상태가 되는 고스트. 내성은 에너지 약점은 충격. 다른 직업보다 크기가 작기 때문에 공격에 잘 안맞는 숨겨진 장점(?)이 있다.
    * **보베** : 비행병. 탭 스킬은 점프를 하는 제트 점프. 차지 스킬은 엄청난 높이로 점프하는 로켓 봄. 내성은 화염 약점은 냉기. 점프로 다른 캐릭터들은 갈수 없는 곳에 올라간다던지 로켓 봄으로 높이 뛰어서 로켓류 무기로 폭격하는 전법이 유효하다.  

  * 스코치드  

    * **혼헤드** : 전투병. 탭 스킬은 근처 적에게 피해를 주는 사이코 슬래시. 차지 스킬은 넓은 범위에 피해를 주는 사이클론. 약점과 내성은 없다. 오큘러스와 비교하면 스킬 자체의 위력은 강하지만 어디로 튈지 알 수 없다는것이 단점. 그야말로 사방팔방으로 튄다.(...)
    * **에스큐** : 지원병. 탭 스킬은 적의 체력을 흡수하는 라이프 서커. 차지 스킬은 범위내 아군을 회복시키고 적 체력과 이동속도를 저하시키는 바일 바일. 내성은 냉기 약점은 화염. 바일 바일의 경우 눈에 뻔히 보이기 때문에 적이 맞아주는 일은 별로 없지만 아군이 모여 있을 때 사용해주면 상당히 좋다.
    * **매그머** : 파워병. 탭 스킬은 방어력을 올리는 스톤 클래드. 차지 스킬은 거대하게 변해서 모든 상대를 **한방**에 깔아뭉게는 콜로서스. 내성은 화염 약점은 냉기.
    * **페트** : 은신병. 탭 스킬은 땅속으로 파고드는 더트 다이브. 그런데 경사가 진 지역은 이동이 불가능하고 땅속으로 파고 들어도 다 보이는데다가 때리면 피해까지 입는다. 도대체 어디에 쓰라는 스킬인가.(...) 차지 스킬은 가스를 살포해서 적을 비틀거리게 만드는 디지 스포어. 내성은 에너지 약점은 충격.
    * **디머낙** : 비행병. 탭 스킬은 점프하는 디먼 도약. 차지 스킬은 디먼 활공. 보바와 비교하면 점프 높이는 낮지만 활공을 하기 때문에 체공 시간이 길고 먼 거리를 이동할 수 있다. 내성은 충격 약점은 에너지.  

# 5 멀티플레이 ¶

CTF, 데스매치, 팀데스매치, 침략, 코옵 등등 다양한 멀티플레이 모드를 지원하며 10가지 직업과 10가지 무기를 조합하여 자신만의
개성적인 조합이나 상황에 맞춘 특화형 조합으로 게임을 진행 할 수 있다.
