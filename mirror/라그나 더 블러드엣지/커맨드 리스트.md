매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요. 여담이지만 개그성 드라마CD에 따르면 기술명 전부
[쥬베이](%EC%A5%AC%EB%B2%A0%EC%9D%B4.md)가 지어준 거라고(...). <del>스승도 중2병이었나</del>

  

★ 드라이브 - 소울 이터 (ソウルイーター / Soul Eater) : 상대의 체력을 흡수합니다.  
★ 오버 드라이브 - 블러드 카인 이데아 (ブラッドカインイデア / Blood Kain Idea)

  

★ 필살기  

  * 헬즈 팽(ヘルズファング / Hell's Fang) : 앞으로 돌진하는 공격입니다.  
┗ 추가 공격 : 상대의 체력을 흡수하는 추가 공격입니다.

  * 인페르노 디바이더 (インフェルノディバイダー / Inferno Divider) : 올라가면서 대공 공격을 합니다.  
┗ 올려치기 : 인페르노 디바이더 후에 추가로 공격합니다.  

┠ 옆으로 날려보내기 : 상대를 옆으로 날려버립니다.  
┗ 내려찍기 : 상대를 땅에 내려찍는 공격입니다.

  * 건틀렛 하데스(ガントレットハーデス / Gauntlet Hades) : 앉아서 가드할 수 없는 돌진 공격입니다.  
┗ 추가 공격 : 상대의 체력을 흡수하고 높이 차 올립니다.

  * 데드 스파이크(デッドスパイク / Dead Spike) : 상대의 체력을 흡수하는 장풍을 눈앞에 만들어냅니다.
  * 아직 끝이 아니야(まだ終わりじゃねぇぞ / Not Over Yet) : 다운된 상대를 일으켜 공격합니다.
  * 베리얼 엣지(ベリアルエッジ / Burial Edge) : 공중에서 대각선 아래쪽으로 돌진하는 공격입니다.
  * 블러드 사이드(ブラッドサイズ / Blood Scythe) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 카니지 시저(カーネージシザー / Carnage Scissor) : 앞으로 돌진 공격을 한 다음 상대의 체력을 흡수하는 추가 공격을 합니다. 버튼을 누르고 있으면 공격을 도중에 멈출 수 있습니다.
  * 블러드 카인(ブラッドカイン / Blood Kain) : 일정 시간 동안 드라이브 버튼에 의한 공격력이 향상됩니다. 단, 발동 중에는 시간이 지날수록 체력이 감소합니다.
  * 어둠의 탐식(闇に食われろ / Devoured by Darkness) : 상대의 체력을 많이 흡수하는 가드 불능 공격입니다.  

★ 아스트랄 히트  

  * 블랙 온슬로트(ブラックオンスロート / Black Onslaught) : 앞쪽에 무기를 쳐올리는 공격을 하고 명중했을 때만 연출이 시작됩니다.
