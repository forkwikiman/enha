# 1 개요 ¶

이름 : [아야사키하야테](%EC%95%84%EC%95%BC%EC%82%AC%ED%82%A4%20%ED%95%98%EC%95%BC%ED%85%8C.md)

클래스 : 산젠인 가의 집사  
사정거리 : 100(근거리)  
주 속성 : 힘

궁극기를 활용한 적진형 무너뜨리기와 두 개의 하야테처럼을 활용한 테러에 능한 탱커.  

# 2 기술 ¶

  

## 2.1 일반기 ¶

  

### 2.1.1 신출귀몰 ¶

사거리 내 아군에게 이동하며 300범위에 100/200/300/400의 데미지와 영웅에겐 1.5초 스턴 비영웅에겐 2초스턴을 줍니다.  
사거리2500 시전시간 1초 마나 소모 100/125/150/175 쿨다운 25초

  

도주용으로 적절하게 쓸수있으며 아군중 추격 스킬이 있는 캐릭이 있다면 적절히 써서 추격용으로도 쓸만하다.  

### 2.1.2 하야테처럼 ¶

90/180/270/360 의 데미지를 준다.  
길이 800 폭 200 마나 소모 60/90/120/150 쿨다운 12초

### 2.1.3 무적의 신체 ¶

패시브 스킬 하드스킨.  

  

하야테가 받는 물리데미지가 15/30/45/60 감소 합니다.  

  

하드스킨은 서로 중첩되기 때문에 물리데미지를 줄여주는 타워 실드같은걸 몇개 껴주면 물리공격으로는 안죽는 불사의 하야테가 된다.  

## 2.2 궁극기 ¶

  

### 2.2.1 진 하야테처럼! ¶

길이1000 폭250 데미지 700/1200 2초 스턴  
마나 소모 300/550 쿨다운 180초

  

임페일 기반이라 다른 임페일 기반과 마찬가지로 정지트리거로 잡아둔후 데미지를 주는 아군 스킬이 있다면 조심해야한다.  

# 3 운영 ¶

이동기가 3개(1,2,4)나되고 남은 한개도 생존을 늘려주는 하드스킨 스킬이라 어지간해서는 추격하기 힘들다.  
1스킬로 적을 맞추는건 아군 스턴캐릭이 없으면 조금 힘들지만 맞추게되면 2킬로 추가 피해를 줄수있다.
