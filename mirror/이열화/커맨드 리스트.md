## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) ¶

※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 침투장 : 근접해서 ← or → + B  

★ 특수기  

  * 용추각 : 상대 다운시 ↑ + C
  * 식취 : 스타트 버튼 지속
  * 하 : AB  
┣ 화영 : 하 중 공격 버튼  
┣ 염군 : 하 중 ↓↙← + B  
┗ 선풍 : 하 중 ↓ + C`[1]`  

┣ 선풍 : 선풍 중 ↓ + C  
┗ 용쇄낙 : 선풍 중 → + C  

┣ 열선풍 : 선풍 중 ↓ + C  
┗ 용쇄낙 : 선풍 중 → + C  

★ 필살기  

  * 용추선 : ↓↙← + C (3회 연속 입력)
  * 염선상 : →↓↘ + B  
┗ 봉락종 : 염선상 중 ↓ + B

  * 염룡파미(●) : →↘↓↙← + A or B  
┗ 사자포후 : 강 염룡파미 히트시 ←↙↓↘→ + B

  * 무영각 : 공중에서 ↓ + C  

★ 초오의  

  * 오의 ・ 염룡전신 : ↓↙←→ + AB   

★ 잠재오의  

  * 비오의 ・ 창천무영각 : ↓↙←→ + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 침투장 : 근접해서 ← or → + CD  

★ 특수기  

  * 용추각 : 상대 다운시 ↑ + C
  * 식취 : 스타트 버튼 지속
  * 하 : AB  
┣ 화영 : 하 중 A  
┣ 염군 : 하 중 B  
┣ 선풍 : 하 중 C  

┣ 선풍 : 선풍 중 ↓ + C  
┗ 용쇄낙 : 선풍 중 → + C  

┣ 열선풍 : 선풍 중 ↓ + C  
┗ 용쇄낙 : 선풍 중 → + C

┗ 발경 : 하 중 D  
★ 필살기  

  * 용추선 : ↓↙← + C (3회 연속 입력)
  * 염선상 : ↓ 모으고 ↑ + B  
┗ 봉락종 : 염선상 중 ↓ + B

  * 염룡파미(●) : →↘↓↙← + A or B  
┗ 사자포후 : 강 염룡파미 히트시 ←↙↓↘→ + B

  * 무영각 : 공중에서 ↓↘→ + C
  * 용상선 : 공중에서 ↓↙← + C (2회 연속 입력)  

★ 초오의  

  * 오의 ・ 염룡전신 : ↓↙←→ + AB   

★ 잠재오의  

  * 비오의 ・ 창천무영각 : ↓↙←→ + B   

★ 난무오의  

  * 절명오의 ・ 천지여명각 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, A, ↓↘→ + C  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, A, → + BC, A, B, ↓↘→ + B
