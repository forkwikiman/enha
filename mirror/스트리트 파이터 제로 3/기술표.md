# 1. 읽기 전에 알아둘 사항 ¶

  * '계열'은 해당 레버 방향을 중심으로 대각선 양쪽 2방향을 포함한 3방향을 뜻한다. (예: ↓ 계열 = ↙ or ↓ or ↘)
  * 계열 표기는 레버 단일 방향 커맨드를 지닌 기술들 중 대각선 방향 입력으로도 나가는 기술과 그렇지 않은 기술을 구분하기 위한 용도이므로 섬머솔트류 커맨드 등에 대해서는 생략.
  * 지상 ← or → 계열 커맨드는 ↖ or ↗ 입력으로도 사용 가능하다.`[1]`
  * 레버 1회전 커맨드는 3/4회전만 시켜도 커맨드가 성립된다.
  * 레버 2회전 커맨드는 1 + 3/4회전만 시켜도 커맨드가 성립된다.
  * 버튼을 연타하면 데미지가 올라가는 슈퍼 콤보들은 버튼 연타와 함께 레버를 좌우로 흔들어주면 더욱 데미지가 증가한다.
  * 제로 카운터의 커맨드는 가드 중 → + 같은 위력의 PK로 변경되었고 X-ISM에서는 사용 불가능. 제로 카운터는 본 문서에 별도로 기재하지 않는다.
  * Z-ISM과 V-ISM의 제로 카운터가 각각 다르다. 대체로 전작에서 P로 사용하던 제로 카운터는 Z-ISM에, K로 사용하던 제로 카운터는 V-ISM에 배정되었다.
  * X와 Z, V 표기는 각각 X-ISM, Z-ISM, V-ISM에서 사용 가능한 기술. 아무 표기가 없다면 모든 ISM에서 쓸 수 있다.

## 2.1. 류 ¶

★ 기본잡기  
업어치기 : ← or → + PP (잡기)  
배대되치기 : ← or → + KK (잡기)  
공중 업어치기 : 공중 ← or → + PP (공중 잡기)

  

★ 특수기  
쇄골 깨기 : → + 중P  
정중 이단 찌르기 : → + 강P <X, V>  
선풍각 : → + 중K <Z, V>

  

★ 필살기  
파동권 : ↓↘→ + P  
작열파동권 : ←↙↓↘→ + P  
승룡권 : →↓↘ + P

파동의 자세 : ↓↘→ + 스타트 <Z, V>

  

★ 슈퍼 콤보  
진공파동권 : ↓↘→↓↘→ + P <X, Z>  
진공용권선풍각 : ↓↙←↓↙← + K <Z>  
멸 승룡권 : ↓↘→↓↘→ + K (Lv3 전용) <Z>  
┗ 진 승룡권 : 특정 타이밍에 히트시키면 자동 발동`[2]`
