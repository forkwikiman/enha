# 4. [KOF XIII](KOF%20XIII.md) ¶

※ EX 대응기(●)는 A or C로 나가는 기술은 AC로, B or D로 나가는 기술은 BD로 EX 필살기를 쓸 수 있다.

┗ 격권 (페인트) : 격권 (모으기) 중 B or D

┗ 강용격(●) : ↓↘→ + A

  * 전련권(●) : ↓↘→ + B or D
  * 탄권(●) : →↘↓↙←→ + C (잡기)
  * 탄권 (튕기기) : ↓↙← + C  

★ 초필살기  

  * 절! 격권(●) : ↓↘→↓↘→ + A or C
  * 폭진 : C, A, B, C (기게이지 2개 소비)

  * 천장폭진격 : ↓↘→↘↓↙← + AC
