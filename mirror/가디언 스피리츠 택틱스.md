## 1. 개요 ¶

맵 제작 커뮤니티인 4RUM에서 만든 추리 계열의 유즈맵이다. 같은 4RUM에서 만든 유즈맵인 [가디언 스피리츠 2](%EA%B0%80%EB%94%94%EC%96%B8%20%EC%8A%A4%ED%94%BC%EB%A6%AC%EC%B8%A0%202.md)의 스토리와 [타뷸라의늑대](%ED%83%80%EB%B7%B8%EB%9D%BC%EC%9D%98%20%EB%8A%91%EB%8C%80.md) 제작으로 쌓인
노하우를 기반으로 카이(4rum_Kai)와 리츠베른(4rum_Lizberne)이 제작하였으며, 이름을 감춘 플레이어가 서로의 정체를 추리하며
적 진영의 지휘관을 추리하여 살해하는 것을 목적으로 한다.

  

현재 한국의 추리 계열 유즈맵 중에서는 [데스노트](%EB%8D%B0%EC%8A%A4%EB%85%B8%ED%8A%B8.md),
[타뷸라의 늑대](%ED%83%80%EB%B7%B8%EB%9D%BC%EC%9D%98%20%EB%8A%91%EB%8C%80.md)와 함께
가장 인기있는 유즈맵에 속한다.

## 2. 공통 스킬 ¶

모든 모드에서 공통적으로 등장하는 기본적인 스킬들이다. 동맹/동맹 파기와 공표 스킬은 모든 캐릭터가 가지고 있다.

  

  * 공표 : 자신의 이름을 표시한다. 진짜 이름을 표시할 경우 매턴마다 진실의 조각을 얻을 수 있고, 마나를 10 더 받는다. 쿨타임 40초.
  * 동맹 : 특정 플레이어와 동맹을 맺는다. 동맹을 맺게 되면 매 턴마다 10의 마나가 추가로 들어온다. 쿨타임 5초. 마나 10 소모.`[1]`
  * 동맹 파기 : 특정 플레이어와 맺은 동맹을 파기한다. 쿨타임 5초. 마나 10 소모.
  * 공격 : 특정 플레이어를 선택 한 후 적군의 이름을 선택한다. 정체와 일치하면 공격에 성공하지만 이름이 틀렸거나 보디가드가 있는 경우(카이-아린 단테스-켈후)에는 자신이 사망한다. 쿨타임 60초. 마나 50 소모.
  * 상급 공격 : 공격과 비슷하지만 두 번 실패하면 사망한다.
  * 최상급 공격 : 이름이 틀렸거나 보디가드로 공격에 실패해도 사망하지 않는다.
  * 아군 확인 : 대상이 아군이고 진명(진짜 이름)을 공표 하였을 경우, 대상의 정체를 알아낸다. 반복 사용 가능. 쿨타임 45초. 마나 15 소모.
  * 적군 확인 : 대상이 적군이고 진명을 공표 하였을 경우, 대상의 정체를 알아낸다. 쿨타임 60초. 마나 25 소모.
  * 스캔 : 대상의 정체를 하나 선택하며(단, 지휘관 캐릭은 불가능하다) 일치 할 경우 정체를 알아 낼 수 있다. 보통 한 진영당 한명에게 주어진다. 마나 20 소모.
  * 상급 스캔(아군 확인/적군 확인) : 스캔(아군 확인/적군 확인)과 동일하지만 쿨타임이 좀 더 짧다.
  * 아군 스캔 : 스캔과 동일하지만 아군만 알아 낼 수 있다.
  * 적군 스캔 : 스캔과 동일하지만 적군만 알아 낼 수 있다.
  * 진실의 보석 : 자신이 진명을 낸 상태로 턴이 지나가면 1/3 조각을 얻을수있으며, 2/3 조각은 1회성 소모품, 완성품은 지속적 사용이 가능한 아이템이 된다. 사용하면 대상의 정체를 알 수있다. 단 지휘관 캐릭터(카이/단테스,카이/샤이닝,라엘/엘타스)는 정확하게 알아낼 수 없고 "대상은 카이 혹은 단테스 입니다." 이런 식으로 표기된다.
  * 채팅시 -플레이어 숫자(예를 들어 1p에게 할경우 -1 "할말")를 사용하여 그 플레이어에게 귓말을 보낸다. 스킬은 아니지만 게임 진행에서 매우 중요한 요소이다.

## 3. 진행 방식 ¶

진행은 실시간으로 되며, 턴은 마나를 회복하거나 진실의 조각을 모을 때 사용된다. 마나와 쿨타임만 된다면 자신의 스킬은 언제든지 사용 할 수
있다.

[[edit](http://rigvedawiki.net/r1/wiki.php/%EA%B0%80%EB%94%94%EC%96%B8%20%EC%8
A%A4%ED%94%BC%EB%A6%AC%EC%B8%A0%20%ED%83%9D%ED%8B%B1%EC%8A%A4?action=edit&sect
ion=4)]

### 3.1. 턴 ¶

1분 30초마다 턴이 지나며, 매 턴마다 마나 20을 획득한다. 동맹(내가 건 것이 아니라 상대가 나에게 건)수x10 만큼 추가 회복, 턴이
지날 때 진명을 내고 있다면 마나 10이 추가로 회복되고 진실의 조각(1/3)을 획득한다.

[[edit](http://rigvedawiki.net/r1/wiki.php/%EA%B0%80%EB%94%94%EC%96%B8%20%EC%8
A%A4%ED%94%BC%EB%A6%AC%EC%B8%A0%20%ED%83%9D%ED%8B%B1%EC%8A%A4?action=edit&sect
ion=5)]

### 3.2. 진실의 보석 ¶

매 턴이 지날때 진명이면 얻을 수 있다. 진실의 조각(1/3)은 아무런 능력도 없지만, 진실의 조각(2/3)을 가지고 있다면 소모하여 대상의
정체를 알아 낼 수 있다. 쿨타임 180초. 마나 50 소모. 합쳐진 진실의 보석은 무한으로 사용 가능하다. 쿨타임 360초. 마나 75
소모.`[2]`

## 4. 게임 모드 ¶

왕자들인 단테스와 카이의 대립을 다룬 '왕자들의 내전', 그리고 1차 가디언 vs 다크니스 전쟁을 다룬 '리델루트 황야', 얼음 트롤 부족의
분열을 다룬 '트롤 부족의 반란'의 4개 모드이다.

#### 4.2.2. 단테스 측(붉은색)`[5]` ¶

  * 단테스 (석세스 오브 다크니스) : 공격, 아군 확인 / 단테, 단테스`[6]`  

    * 마황자의 명령 : 대상이 카이이면 즉시 살해한다. 카이가 아니면 단테스가 사망한다.
    * 후계자 임명 : 대상이 메르츠키엘이면 후계자로 임명한다.(단테스가 죽어도 게임이 끝나지 않게 된다.) 성공시 모두에게 알려진다. 1회 사용 가능.
    * 전체 채팅 : "-전체 할말" 로 정체를 숨기고 모두에게 말할 수 있다. 사용시 단테스 : 할말 로 표기된다. 마나 35 소모.  

  * 메르츠키엘 (세컨드 프린스) : 상급 공격 / 멸치, 멜츠, 메르  

    * 쉐도우 자일 : 대상을 45초간 무적+행동불가 상태로 만든다. 반복 사용 가능.
    * 배우자 : 대상이 레인딜라이면 알아낸다. 반복 사용 가능.
    * 후계자 : 단테스가 후계자 임명 스킬에 성공하면 획득한다. 단테스가 사망하더라도 메르츠키엘이 살아있으면 패배하지 않는다.  

  * 켈후 (오우거 로드) : 상급 공격 / 켈후, 케루  

    * 충복 : 대상이 단테스이면 알아낸다. 반복 사용 가능. 마나 25 소모.
    * 보디가드 : 켈후가 생존하는 동안 단테스는 공격으로 죽지 않는다.
    * 하드 스킨 : 켈후는 3번 공격받아야 사망한다.
    * 전사의 후각 : 대상이 상급/최상급 공격을 가지고 있는지 알아낸다. 단 켈후나 투마의 이름을 내고 있는 대상에게는 사용이 불가능하다.
    * 냉정함 : 소엔의 조언을 받으면 습득한다. 투마의 용맹한 돌진에 사망하지 않는다.  

  * 프레이아 (드로우 프리스티스) : 스캔 / 프레  

    * 신탁 : 게임 시작 3분 후 사용이 가능하다. 대상의 진실의 보석 소유 여부를 알아낸다. 소유하고 있을 경우 등급을 알아 낼 수 있다.  

  * 세피 (카오스 위치) : 아군 확인 / 세피  

    * 버닝 매직 : 대상의 마나를 70 태운다.
    * 교란 : 대상을 45초동안 행동 불가 상태로 만든다. 반복 사용 가능.`[7]`
    * 저주 : 자신이 진명을 내고 있을 때만 사용 가능. 대상의 마나가 50 이하이면 정체를 알아낸다. 쿨타임 300초. 반복 사용 가능.  

  * 소엔 (드로우 어쌔신) : 적군 확인 / 소엔  

    * 백스텝 : 대상이 자신에게 동맹을 맺었다면 사용하여 대상을 죽일 수 있다. 백스텝에 성공하면 연쇄 살인을 습득한다.
    * 변장 : 카이측의 이름으로 공표 시 카이측 아군 확인과 크레이트의 스캔에 성공으로 표시된다.
    * 연쇄 살인 : 백스텝 성공시 습득한다. 공격과 같은 기능을 가졌으나, 적을 죽이는데 실패하면 스킬이 사라진다.(투마 공격에 성공하였더라도 살해에 실패하면 사라진다.) 연쇄 살인으로 적을 죽이면 다시 사용이 가능하다. 연쇄 살인이 사라져도 백스텝이 성공 할 경우 다시 습득한다.
    * 조언 : 대상이 켈후이면 서로 동맹이 되고, 투마의 용맹한 돌진으로부터 켈후를 보호한다.  

  * 레인딜라 (소울 팔로어) : 아군 확인 / 레딜, 레인, 딜라  

    * 배우자 : 대상이 메르츠키엘이면 알아낸다.
    * 영혼의 회복 : 대상이 공격에 실패해도 패널티를 입지 않는다. 반복 사용 가능.
    * 리비도의 여사제 : 자신이 진명을 낸 상태에서만 사용이 가능하다. 프레이아의 정체를 알게 된다. 1회 사용 가능. 마나 100 소모.
    * 영혼의 벽 : 대상은 카이의 소울 리버에 죽지 않는다. 사용 시 레인딜라는 사망한다. 마나 60 소모.
