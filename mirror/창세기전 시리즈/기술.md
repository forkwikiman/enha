## 1. 필살기 ¶

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8

### 1.1. [창세기전 1](%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%84.md) & [창세기전2](%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%84%202.md) ¶

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=3)]

#### 1.1.1. 검 ¶

  * [연](%EC%97%B0.md)(連): 가장 기본적인 필살기. 바로 앞의 적에게 연속 공격을 한다.
  * [제](%EC%A0%9C.md)(制): 주위 적들을 견제하여 적들의 TP를 감소시키고 움직임을 둔화시킨다. <del>적 전용 필살기</del>
  * [반](%EB%B0%98.md)(反): 한턴 간 적에게 공격받으면 반격한다. 통상적인 반격 커맨드는 전방에서 공격받았을 때만 가능하지만 반(反)은 어느 방향에서 공격받았든 무조건 반격한다.
  * [비](%EB%B9%84.md)(飛): 원거리 공격기. 전방의 적에게 검기를 날리며 공중에 떠 있는 적도 공격할 수 있다. 검기에 맞은 적은 뒤로 일정거리 밀려나간다. 
  * [살](%EC%82%B4.md)(殺): 다음 턴이 돌아올 때까지 모든 공격(반격 포함)을 치명타(크리티컬)로 명중시킨다. 때에 따라서는 일정 확률로 적을 즉사시킨다.
  * [혼](%ED%98%BC.md)(魂): 돌격형 필살기. 적을 돌파하며 전방으로 일정 거리를 뚫고 지나간다. 다수의 적도 돌파할 수 있다. 주로 적의 포위망을 뚫고 탈출할 때나 적진 내부로 깊숙히 난입할 때 이용.
  * [파](%ED%8C%8C.md)(破): 검기술의 최종 진화단계. 360도 회전하면서 거대한 검광을 휘둘러 주변의 일정 범위 내에 있는 적들을 공격한다. 적에게 포위당했을 때 매우 유용한 필살기.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=4)]

#### 1.1.2. 창 ¶

  * 제(制): 검기술 제와 같지만 범위가 더 넓다.
  * 연(連): 검기술 연과 같지만 창의 긴 리치 판정을 받아 전방 두 칸까지 공격할 수 있다.
  * 투(投): 검기술 비와 똑같다. 검기를 날리는 게 아니라 창을 던진다. <del>줄을 매달아서 회수하나?</del>
  * 혼(魂): 검기술 혼과 똑같다. 최대 4명까지 돌파 가능하지만 검기술보다 공격력은 떨어진다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=5)]

#### 1.1.3. 도끼 ¶

  * 격(擊): 순간적으로 공격력을 올려 강력한 한 방을 날린다.
  * 살(殺): 검기술 살과 같다.
  * 회(回): 원거리 공격기. 도끼의 기를 날려 적을 공격한다.
  * 광(狂): 검기술 파와 같지만 범위는 작고 공격력은 더 강하다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=6)]

#### 1.1.4. 활 ¶

활을 쓰는 궁수 계열은 물론이고, 표창을 집어던져 공격하는 닌자 계열도 아래의 필살기들을 사용한다.

  

  * 살(殺): 공격의 명중률과 크리티컬 확률을 증가시킨다. 원거리 무기의 단점인 낮은 명중율과 공격력을 보완하는 필살기.
  * 연(連): 여러 개의 화살을 적에게 난사한다. 보통은 2~3 발을 발사하지만 때때로 10여개의 화살을 난사한다.
  * 뇌(雷): 전방 일직선 방향으로 화살을 발사한다. 화살은 적을 관통하면서 날아가며 추가로 전격 데미지도 입힌다.
  * 심(心): 눈을 감고 마음의 눈으로 적 한명을 공격한다. 사정거리에 상관없이 맵 상에 있는 어떠한 적이라도 공격할 수 있다. 데미지도 높다. 일종의 저격기술.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=7)]

#### 1.1.5. 주먹 ¶

  * 연(連): 검기술 연과 동일. 주먹과 발로 연타 공격을 날린다.
  * 살(殺): 검기술 살과 동일.
  * 장(掌): 기를 이용하여 원거리의 적에게 장풍을 날려 공격한다.
  * 기(氣): HP를 소모하는 대신 기를 끌어모아 순간적으로 많은 양의 TP를 모은다. TP 소모량이 큰 기술을 급히 써야할 때 유용하다.
  * 붕(崩): 전방의 적에게 강력한 타격을 입히며 뒤로 밀어낸다.
  * 극(極): 적의 방어력을 무시하고 데미지를 입힌다. 방어력이 높은 적들에게 매우 유용한 필살기.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=8)]

내에서 상당히 연비가 좋은 필살기로, [아수라](%EC%95%84%EC%88%98%EB%9D%BC%28%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%84%20%EC%8B%9C%EB%A6%AC%EC%A6%88%29.md)를 소환한 상태에서 사용할 경우

### 1.3. [창세기전 외전 템페스트](%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%84%20%EC%99%B8%EC%A0%84%20%ED%85%9C%ED%8E%98%EC%8A%A4%ED%8A%B8.md) ¶

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=10)]

#### 1.3.1. 검 ¶

  * [진공수라인](%EC%A7%84%EA%B3%B5%EC%88%98%EB%9D%BC%EC%9D%B8.md): 바로 앞에 있는 적에게 검을 빠르게 휘두르며 공격한다
  * 반월참: 검의 기를 날려서 원거리에 있는 적을 공격한다.
  * 무극일섬혼: 앞으로 5칸까지 일직선으로 뚫고 나가면서 적들을 관통한다
  * 비검: 롱소드를 던져 적을 맞추어 대미지를 입히는 기술. (롱소드)
  * 성혼파쇄: 유리검에 온힘을 모아 적에게 내리친다. 무기는 파괴된다 (유리검)
  * 천라연격: 빠른속도로 적을 난도질한다 (에스톡)

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=11)]

#### 1.3.2. 대검 ¶

  * 반월참: 검의 기를 날려서 원거리에 있는 적에게 피해를 입힌다.
  * 단공빙쇄참: 기를 집중하여 적에게 강한 공격을 가한다.
  * 무극일섬혼: 앞으로 5칸까지 일직선으로 뚫고 나가면서 적에게 피해를 준다.
  * 아르마다 스페셜: 기합을 지르며 바로 앞에 있는 적을 3번 연속 내리치면서 공격한다. (아르마다) 
  * 부메랑: 검을 날려서 원거리에 있는 적에게 피해를 준다. (부로바 블레이드)

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=12)]

#### 1.3.3. 채찍 ¶

  * 사이드와인더: 전방 2칸까지의 적을 채찍으로 관통시키며 공격한다. (사이드와인더)
  * 환영편: 적을 채찍으로 내리쳐서 혼돈 상태로 만든다. (미라쥬)
  * 사두금라편: 바로 앞에 있는 적을 채찍으로 내리치고 뱀이 나타나 적을 가격한다. (스네이크 아이즈)
  * 콜미퀸: 채찍을 강하게 내리쳐서 바로 앞에 있는 적에게 피해를 입힌다. (댄싱퀸)

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%8
4%20%EC%8B%9C%EB%A6%AC%EC%A6%88/%EA%B8%B0%EC%88%A0?action=edit&section=13)]

#### 1.3.4. 총 ¶

  * 3연사: 적에게 3번 연속으로 공격을 가한다. (돌격소총)
  * 6연사: 적에게 6번 연속으로 공격을 가한다. (머쉰건)
  * 러시안 룰렛: 1/6확률로 적에게 강력한 타격을 준다. (리볼버)

#### 2.5.1. 전용 필살기 ¶

  * [나인 크루세이더](%EB%82%98%EC%9D%B8%20%ED%81%AC%EB%A3%A8%EC%84%B8%EC%9D%B4%EB%8D%94.md) \- [디에네 라미엘](%EB%94%94%EC%97%90%EB%84%A4%20%EB%9D%BC%EB%AF%B8%EC%97%98.md)  
디에네가 처음부터 달고 나오기 때문에 자주 쓰게 되는 필살기. [창세기전3](%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%84%203.md)의
[지하드](%EC%A7%80%ED%95%98%EB%93%9C.md)와 비슷하게
[광선검](%EA%B4%91%EC%84%A0%EA%B2%80.md)이 날아다니면서 공격한다.
**"[나인](%EB%82%98%EC%9D%B8.md)이란 숫자의 의미를 알고 있나?"**라는 발동대사가 존재하며, 공격 횟수도 9번.

### 3.1. [창세기전 1](%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%84.md) & [창세기전2](%EC%B0%BD%EC%84%B8%EA%B8%B0%EC%A0%84%202.md) ¶

마법별로 숙련도가 높아지면 보다 높은 등급(최고 3등급)의 마법을 사용하게 되며 위력과 효과도 강해진다. 고레벨의 마법을 사용하려면 많은
TP 축적과 MP가 필요하다.

#### 3.1.1. 원소마법 ¶

화염, 빙한, 전격 속성을 가진 마법을 3원소 마법이라 부르며 기본적인 공격마법들이다. 데미지는 화염>전격>빙한 순이며, 거꾸로
빙한>전격>화염 순으로 행동 방해 효과가 강하다. 설정상 게임 내 기상과 지형 효과도 마법에 영향을 미치는데 바람이 불고 일반 평지에서는
화염 마법이, 추운 지역과 눈이 쌓인 지역에서는 빙계 마법이, 비가 오거나 수상 지역에서는 전격 마법이 강해진다. 역시 반대로 각 속성과
대비되는 지형이나 기후에서는 위력이 약해진다. 이런 3원소 마법은 마법 효과와 속성은 다르지만, 아래와 같이 기본적인 틀은 같다.

  

  * 1레벨: 전방의 일정 범위 내에 있는 한명의 적에게 강한 데미지를 입힌다. 
  * 2레벨: 일정 범위 내에 있는 자신 포함 아군에게 속성을 부여하여 공격력을 증가시킨다. 일반무기에 저항이 있거나 반대 속성을 가진 적에게 큰 피해를 입힐 수도 있다.
  * 3레벨: 전방의 넓은 지역에 마법 공격을 가한다. 데미지는 적지만 적은 MP소모로 다수의 적을 공격할 수 있으며 추가 효과로 괴롭힐 수도 있다.
  * 4레벨: 전방의 좁은 범위에 강한 마법 공격을 한다. 1레벨 마법의 대략 3~4배 수준.
  * 5레벨: 속성별 정령을 소환한다.
  * 6레벨: 캐릭터를 중심으로 하여 넓은 지역에 강한 마법 공격을 한다. 1레벨 마법의 5~6배 수준.
  * 7레벨: 지역 내 모든 적을 공격한다. 전체마법.

#### 3.1.2. 회복마법 ¶

  * 3레벨: 큐어 - 지정 범위에 있는 아군들의 모든 상태이상을 치료한다.

#### 3.3.1. 화염마법 ¶

  * Class 1│화이어 애로우: 불의 화살을 만들어서 적을 공격한다.
  * Class 2│화이어 볼: 불의 공을 여러 개 띄워서 적을 공격한다.
  * Class 3│화이어 필드: 특정지역에 불이 붙는 바위들을 떨어뜨려 적에게 피해를 입힌다.
  * Class 4│화이어 웨이브: 전방의 적에게 막대한 피해를 입히는 화염을 발사한다.
  * Class 5│헬 화이어: 화염으로 적 1명에게 막대한 피해를 입힌다.
  * Class 6│프로미넌스: 화면 전체를 태양의 표면과 같은 불로 채워서 적에게 피해를 입힌다.
  * Class 7│메테오 : 우주공간을 떠도는 운석을 소환하여 적에게 피해를 입힌다.

#### 3.3.2. 빙계마법 ¶

  * Class 1│아이스 미사일: 얼음 미사일을 만들어서 적을 공격한다.
  * Class 2│다이아 스피어: 적을 얼음으로 둘러싼 뒤에 깸으로써 공격한다.
  * Class 3│아이스 포그: 얼음안개를 만들어 일정 범위 내의 적들을 공격한다.
  * Class 4│프로즌 웨이브: 전방으로 얼음기둥을 전진시켜 적들을 공격한다.
  * Class 5│아이스 스톰: 얼음 폭풍우를 불러내어 일정 영역의 적들을 공격한다.
  * Class 6│블리자드: 화면 전체의 적에게 강력한 대미지를 입힌다.
  * Class 7│글라시어 퀘이크: 땅 전체를 빙하로 만든 뒤에 거대한 소환수로 하여금 부숨으로써 적 전체에게 피해를 입힌다.

  * Class 4│에너지 월: 방으로 강력한 전기를 발사하여 적을 공격한다.
  * Class 5│에너지 서클: 한 화면 전체에 전기 피해를 입힌다.
  * Class 6│썬더 스톰: 화면 전체에 강력한 번개를 떨어뜨려 적에게 피해를 입힌다.
  * Class 7│기가 썬더: 강력한 전격의 장을 시공에 형성시켜 적 전체에게 피해를 입힌다.
