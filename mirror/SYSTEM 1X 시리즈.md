## 1. 개요 ¶

  

[남코](%EB%82%A8%EC%BD%94.md)에서 사용한
[아케이드](%EC%95%84%EC%BC%80%EC%9D%B4%EB%93%9C.md)용
[게임](%EA%B2%8C%EC%9E%84.md)[기판](%EA%B8%B0%ED%8C%90.md).

  

전체 기계 구조는 [플레이스테이션](%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%98.md)1을 게임 기판에 맞춰 JAMMA규격등을 넣어 커스터마이즈한것이며, 1X라고 써놓은것은 해당 타입의 기판이 3가지가
존재하기 때문이다. 물론 기초 기판은 같지만 세세하게 구조가 다른것이 특징.  
SYSTEM 10 / SYSTEM 11 / SYSTEM 12가 출시되었지만 특이하게도 발매순서는 11 -> 12 -> 10순으로 발매가
되었다.

  

기본적인 기판 구조는 다음과 같다. (전시리즈 공통)  
메인 CPU: R3000A 32-bit RISC 프로세서`[1]` (SYSTEM 12부터 확장 명령어가 사용이 가능해져 더욱 고사양의 게임을
제작할 수 있게 되었다.)  
RAM : 2 MB  
V RAM: 2 MB  
Sound RAM: 512 kB  
그래픽 프로세서 : 초당 360000 폴리곤을 기본으로 스프라이트 / BG 그리기, 개별 스케일링과 회전, 동시 배경 (시차 스크롤)을
지원.  
지원 해상도: 256x224 - 640x480  
색상: 16,700,000 색상  
(SYSTEM 10에서는 [DJ MAIN](DJ%20MAIN.md)처럼 별도의 하드디스크를 달아놓는것이 가능하다.)

  

최초로 발매된 SYSTEM 11을 기본으로 하여 SYSTEM 11의 반도체의 동작 주파수를 1.5 배 빠르게하는 것을 중심으로, 메모리
용량을 추가 증량 파워업시킨 SYSTEM 12가 발매되었으며, 그 뒤 2D게임에 특화된 저가형기판인 10을 출시.  
아이러니하게도 11과 12의 첫 발매 게임은 **[철권 1](%EC%B2%A0%EA%B6%8C%201.md)**과 **[철권3](%EC%B2%A0%EA%B6%8C%203.md)**였다.

  
  

국내에서는 **[철권](%EC%B2%A0%EA%B6%8C.md)시리즈** 하나로 이 기판이 엄청 퍼지는 계기를 마련해주었다. 물론 다른
게임들도 수입되었지만 결국 [철권](%EC%B2%A0%EA%B6%8C.md)으로 바뀌었다
[카더라](%EC%B9%B4%EB%8D%94%EB%9D%BC.md).

  

시스템 11과 12가 국내에 많이 퍼진데 반해 10의 경우는 태고의 달인을 제외하고는 국내에서 히트한 게임이 없기 때문에 잘 찾아보기
힘들다. 그것조차 태고의 달인 7부터는 기판이 바뀌어서 더 찾아보기 힘들어졌다.
