## 2. 화염 계열 스킬 ¶

특수 효과 "화상"으로 극딜에 초점이 맞춰져 있는 트리. 말그대로 미친듯이 데미지를 입힌다. 헬파이어의 추가로 인해 낮은 크리티컬로도 화상에
걸린 적에게 한해 엄청난 크리티컬을 볼 수 있다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=3)]

### 2.1. 파이어 볼 ¶

  * 유형 : 액티브
  * 습득 : 15레벨
  * 쿨타임 : 15초
  * 후행 : 인페르노(엘리멘탈 로드), 파이어볼 EX(셀레아나)  
  
손에 불꽃을 모아 바라보는 방향으로 쏘는 폭발형 마법. 차지가 가능하며 최대 3초간 차지시 안드로메다급의 데미지와 폭발 범위를 가졌다. 단일
공격으로는 엘로의 모든 스킬중 가장 강력하다. 셀레아나는 ex로 강화가능.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=4)]

### 2.2. 플레임 스파크 ¶

  * 유형 : 패시브
  * 습득 : 15레벨
  * 쿨타임 : 8초  
  
그냥 동떨어진 패시브 스킬. 블링크 후 일반공격 버튼을 누르면 불을 내뿜는다. 블링크 후 추가 뎀을 주는데 상당한 폭발 범위를 지녔으나 구체
자체는 작은편. 쏠쏠한 추가뎀과 슈아파괴력을 보여주며 맞은 적에게 화상을 건다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=5)]

### 2.3. 인페르노 ¶

  * 유형 : 액티브
  * 습득 : 17레벨
  * 선행 : 파이어볼 1레벨(엘리멘탈 로드)
  * 쿨타임 : 24초
  * 후행 : 파이어 월(엘리멘탈 로드)  
  
전방에 다단히트의 불꽃을 방사한다. 사거리가 무지 짧다. 6레벨때부터 사거리가 7m로 증가.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=6)]

### 2.4. 파이어 버드 ¶

  * 유형 : 액티브
  * 습득 : 19레벨
  * 쿨타임 : 20초
  * 후행 : 이그니션(엘리멘탈 로드)  
  
3개의 새를 전방에 발사한다. 부채꼴로 뿜어져 나가며 닿은 적은 화상에 걸린다. 과거에는 이그니션의 제물이었지만 현재는 헬파이어의 효과를
보기위해 사용한다. 후에 화염 속성 방어력을 감소 시키는 디버프가 추가되어 [완소](%EC%99%84%EC%86%8C.md)스킬로
등극한다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=7)]

### 2.5. 파이어 월 ¶

  * 유형 : 액티브
  * 습득 : 23레벨
  * 선행 : 인페르노 3레벨(엘리멘탈 로드)
  * 쿨타임 : 33초
  * 후행 : 파이어 실드  
화염기둥을 소환하여 주변의 적에게 데미지를 입힌다. 마지막 타격이 특히 강하다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=8)]

### 2.6. 파이어 실드 ¶

  * 유형 : 액티브
  * 습득 : 26레벨
  * 선행 : 파이어 월 1레벨(엘리멘탈 로드) 
  * 후행 : 플레임 로드(엘리멘탈 로드)  
  
입는 피해를 낮춰주고 주변의 적에게 화상을 거는 버프. 토글형 스킬로 사용시 초당 최대마나의 1%을 지속적으로 소모한다. 사용중에는
슈퍼아머가 상당히 상승된다.

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=9)]

### 2.7. 이그니션 ¶

  * 유형 : 액티브 
  * 습득 : 32레벨
  * 선행 : 파이어 버드 1레벨(엘리멘탈 로드)
  * 쿨타임 : 18초 ~10초  
  
화상상태의 적을 폭발시켜 고데미지를 주는 기술. 상당한 데미지를 지녔으나 화상뎀이 쏠쏠한 스킬들을 강제로 폭발시켜 쓰는 컨트롤이 필요한
기술. 같은 성격의 아이시 프렉션보다 표기 데미지는 높다. 화상 디버프가 모두 날아간다. 레벨이 올라갈 때 마다 스킬 쿨타임이 줄어들어
마스터시에는 10초이다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=10)]

### 2.8. 플레임 로드 ¶

  * 유형 : 액티브
  * 습득 : 32레벨
  * 선행 : 파이어 실드 1레벨(엘리멘탈 로드)
  * 쿨타임 : 45초
  * 후행 : 플레임 볼텍스(엘리멘탈 로드)  
  
지팡이를 타고 날아가 지나가는 자리에 불꽃을 소환한다. 그냥 평범한 잡몹 상대로는 그저 그런 데미지지만 다단히트와 크리티컬과 몸집이 큰
몹이라는 조건이 부합되면 플레임 볼텍스를 관광하는 데미지를 보여준다. 뭐 물론 플레임 볼텍스도 상향이 돼서 그정도는 아니지만 주력기이다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=11)]

### 2.9. 플레임 볼텍스 ¶

  * 유형 : 액티브
  * 습득 : 40레벨
  * 선행 : 플레임 로드 1레벨(엘리멘탈 로드)
  * 쿨타임 : 150초
  * 후행 : 버닝 핸드(셀레아나)  
  
커다란 불꽃의 소용돌이를 생성해서 공격한다. 화염 엘로의 결전병기. 데미지하나만큼은 굴지의 1위를 달리고 있으며 종잇장 슈아랑 보호막이지만
그래도 생성된다. 말이 필요없는 불엘로의 간판기  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=12)]

## 3. 냉기 계열 스킬 ¶

기본적으로 모든 스킬들이 냉기저항을 감소시키며 몇몇은 데미지 뻥튀기가 가능한 빙결을 건다. 그리고 각 빙결스킬은 서로 중첩이 가능하며 최대
4중첩시 들어가는 데미지가 2배로 늘어난다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=13)]

### 3.1. 프리징 소드 ¶

  * 유형 : 액티브 
  * 습득 : 15레벨
  * 쿨타임 : 15초
  * 후행 : 프리징 필드(엘리멘탈 로드), 프리징 소드 EX(엘리멘탈 로드)  
전방 바닥에서 여러개의 칼이 튀어나오게 하여 피해를 입힌다. 이때 적은 공중에 뜨게 된다. 부가적으로 냉기 저항력이 낮아진다. 냉기쪽인
엘레스트라로 간다면 프소 ex가 크고 아름답기에 좋으나 화염엘로는 그냥 칠링 미스트, 프리징 스파이크 쓰기위한 제물일뿐(..). 패시브를
찍은 뒤로는 진짜 좋아진다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=14)]

### 3.2. 프로스트 윈드 ¶

  * 유형 : 패시브 
  * 습득 : 15레벨
  * 쿨타임 : 8초  
  
화염의 플레임 스파크와 같은 성격의 기술. 다만, 전방에 쏘는 플레임 스파크와는 다르게 주변에 쓰기 때문에 좀 더 방어적인 성향과 부가효과로
이속저하와 냉기 저항력 하락이 있다. 냉기 저항력이 내려가면 빙결 지속시간이 조금 더 증가하기 때문에 빙결을 걸기 전에 한번 써주는 것이
좋다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=15)]

### 3.3. 프리징 필드 ¶

  * 유형 : 액티브 
  * 습득 : 17레벨
  * 선행 : 프리징 소드 1레벨(엘리멘탈 로드)
  * 쿨타임 : 28초
  * 후행 : 프리징 스파이크(엘리멘탈 로드)  
  
전방 넓은 범위에 냉기속성 공격을 한다. 시전시간이 꽤 있는 편이며 범위가 상당하다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=16)]

### 3.4. 글레이셜 웨이브 ¶

  * 유형 : 액티브 
  * 습득 : 19레벨
  * 쿨타임 : 24초
  * 후행 : 아이시 프렉션(엘리멘탈 로드)  
  
주변을 순식간에 얼려 데미지를 준다. 빙결 확률이 굉장히 높고 시간도 긴편이다. 마스터 시 7초가량의 빙결을 걸며 이는 다른
빙결스킬(글레이셜 스파이크 , 아이스 스피어 , 블리자드 등..) 과 중첩이 가능하다. 얼음의 내구도도 굉장히 높기에 개편 후 엘로의
필수스킬이라고도 볼 수 있을정도.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=17)]

### 3.5. 프로즌 버스트 ¶

  * 유형 : 토글
  * 습득 : 20레벨  
  
스킬을 쓰는 순간 주변에 걸린 빙결을 모두 제거한다. 물 속성 공격력이 30% 증가하며, 불 속성 공격력은 30% 감소한다. 글레이셜
스파이크, 글레이셜 웨이브, 아이스 스피어의 데미지가 크게 증가하고, 빙결은 동상으로 대체된다. 프로즌 버스트 상태에서도 데미지 변화가 없는
블리자드 스톰, 프리징 필드 EX로는 빙결을 걸 수 있다. 실드와 함께 사용이 불가능하여 종잇장 슈퍼아머로 인해 솔플에서는 프로즌
버스트쓰다가 스킬 캔슬나는 경우가 꽤 있다. 파티에서는 쓸 일이 없다. 빙결 추가데미지가 월등히 좋기 때문  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=18)]

### 3.6. 프리징 스파이크 ¶

  * 유형 : 액티브 
  * 습득 : 23레벨
  * 선행 : 프리징 필드 1레벨(엘리멘탈 로드)
  * 쿨타임 : 15초
  * 후행 : 프리징 실드(엘리멘탈 로드)  
  
전방에 얼음 벽을 생성하여 지속적으로 데미지를 준다. 상당한 데미지와 착한 쿨타임으로 화염엘리던 냉기엘리던 무조건 마스터 해주는 밥줄기.
기술만 보면 프리징 소드의 강화판이라고 볼 수 있다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=19)]

### 3.7. 프리징 실드 ¶

  * 유형 : 액티브 
  * 습득 : 26레벨
  * 선행 : 프리징 스파이크 1레벨(엘리멘탈 로드)
  * 후행 : 칠링 미스트(엘리멘탈 로드)  
  
자기 자신에게 냉기의 방패를 생성하여 슈퍼아머를 크게 상승시키고 입는 피해를 낮추며 주위 적에게 낮은 확률로 빙결 효과를 주는 버프 스킬.
개편 이후 토글스킬로 변화되었다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%97%98%EB%A6%AC%EB%A9%98%ED%83%8
8%EB%A1%9C%EB%93%9C/%EC%8A%A4%ED%82%AC?action=edit&section=20)]

### 3.8. 아이시 프렉션 ¶

  * 유형 : 액티브 
  * 습득 : 32레벨
  * 선행 : 글레이셜 웨이브 1레벨(엘리멘탈 로드)
  * 쿨타임 : 18초
