## 2. 2004년 왕중왕전(2004년 12월 25일) ¶

  * 역대 1대 달인부터 4대 달인들이 펼치는 왕중왕전으로 진행되었으며, 2인 1조 팀의 대표 한 명과 아나운서들이 2인 1조로 팀을 이루어서 진행했다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%9A%B0%EB%A6%AC%EB%A7%90%20%EA%B
2%A8%EB%A3%A8%EA%B8%B0/%ED%8A%B9%EC%A7%91%20%EB%B0%A9%EC%8B%9D?action=edit&sec
tion=3)]

### 2.1. 진행 형식 ¶

  * 1단계 : 2004년 진행형식 2단계와 동일하다. 한 문제당 20만원(개인전으로 진행)
  * 2단계 : 2004년 진행형식 1단계와 동일하다. 한 문제당 10만원(개인전으로 진행)
  * 3단계 : 어느 한 소설을 보고 버저를 눌러 빈 칸을 맞히는 형식, 2문제(한 문제당 50만원)
  * 4단계 : 2004년 진행형식 3단계와 동일하다. 개인전으로 진행되고, 현재까지 적립한 상금의 절반까지 걸 수 있어서 성공하면 건 상금을 가져가며, 실패하면 건 상금을 뺀 일부만 가져간다.
  * 5단계 : 2004년 진행형식 4단계와 동일하다. 개인전으로 진행되고, 현재까지 적립한 상금을 모두 걸 수 있으며, 역시 앞과 동일하다.
  * 6단계 : 2004년 진행형식 5단계와 동일하다. 우승팀이 제한시간 안에 성공하면 획득한 상금의 2배를 가져가며, 실패하면 현재까지 획득한 상금만을 가져간다.

## 3. 2007년 왕중왕전(2007년 12월 31일) ¶

  * 역대 9대 달인부터 12대 달인들이 펼치는 왕중왕전으로 진행되었다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%9A%B0%EB%A6%AC%EB%A7%90%20%EA%B

tion=5)]

### 3.1. 진행 형식 ¶

  * 1-1단계 : 보기 10개 중 제시어와 관련된 정답 6개 찾기, 개인전으로 진행되며 한 문제 당 20점, 모두 맞히면 200점이 주어진다(중간에 오답을 고르면 문제 풀이 종료)
  * 1-2단계 : 사전적 정의 맞히기, 형식은 전과 동일
  * 1-3단계 : 점수가 높은 사람이 상대편을 지목해서 1:1 맞대결을 펼친다  

문제 유형은 2음절 단어 맞히기로, 버저 형식으로 진행되며, 한 문제당 30점이 주어진다(총 10문제)

  * 1-4단계 : 1-3단계와 동일하며, 문제 유형은 3지선다 형식으로, 한 문제당 50점이 주어진다(총 10문제), 각 대결마다 한 문제에는 60만원 상품권이 걸려 있다.
  * 문제를 다 풀고 점수가 높은 2명이 2단계에 진출하며, 그 뒤 형식은 전과 동일하다.

### 4.1. 진행 형식 ¶

  * 1-1단계 : 10개의 낱말 중 정답 5개 찾기(개인전으로 진행, 한 문제당 50점, 오답을 고르면 문제 풀이 종료)
  * 1-2단계 : 세 개의 뜻풀이를 보고 그에 맞는 동음어 맞히기(버저 형식, 한 문제당 100점, 총 10문제)
  * 1-3단계 : 2007년 왕중왕전 1-4단계와 유사하다.
  * 1-4단계 : 사전적 정의 맞히기, 형식은 전과 동일(단, 1단계 점수가 2단계로 이어지지 않음.)
  * 문제를 다 풀고 상위 2명이 결승에 진출하며, 그 뒤 형식은 전과 동일하다(다만, 낱말 맞히기 문제는 20개로 늘었으며, 속담 문제도 들어있음)

## 5. 2008~2009 왕중왕전(2009년 12월 28일) ¶

  * 2008~2009년 달인 4명이 대결을 펼치는 형식이다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%9A%B0%EB%A6%AC%EB%A7%90%20%EA%B

### 5.1. 진행 형식 ¶

  * 1-1단계 : 단어를 보고, 3음절 명사를 만들기(예 : 김치- ㄱ, ㄷ, ㅁ, ㅅ, ㅍ(김칫국, 파김치, 김칫독)) 개인전으로 진행되며, 한 단어 당 50점이 주어진다(제한 시간 60초)
  * 1-2단계 : 단어를 보고, 동사, 형용사 만들기(예 : ㅂㄴ가다, ㅈㄴ가다 - 빗나가다, 지나가다) 개인전으로 진행되며, 한 단어 당 50점이 주어진다(제한 시간 60초)
  * 1-3단계 : 2010년 2월 1일 ~ 2011년 6월 27일 진행형식과 유사하다.(여기서 점수가 낮은 한 사람은 탈락하게 된다)
  * 1-4단계 : 사전적 정의 맞히기(단, 1단계 점수가 2단계로 이어지지 않음)
  * 2단계 : 가로 세로 낱말 맞히기(여기서는 문제 점수가 없으며, 먼저 최종단어를 맞힌 사람이 우승하게 된다.)
  * 달인 도전 문제 : 9개의 2지선다 문제 맞히기(모두 맞히면 3000만원, 실패하면 100만원 획득)

## 6. 400회 특집 방송(2012년 1월 9일) ¶

  * 15명의 역대 왕중왕들이 대결을 펼치는 형식이다.
  * 초대 mc인 정재환 씨가 엄지인 아나운서와 같이 진행을 하였다.

### 6.1. 진행 형식 ¶

  * 1단계 : 국립 국어원에서 선정한 국어 순화어 맞히기. 총 5문제, 한 문제당 50점(공통 쓰기 문제)
  * 2단계 : 맞춤법, 띄어쓰기 문제로 구성. 총 15문제, 한 문제당 50점(공통 쓰기 문제)
  * 3단계 : 10문제 연상문제(형식은 전과 동일, 겹낱말 5문제, 공통낱말 5문제로 구성)
  * 여기서 점수가 높은 3명이 다음 단계 진출  

    * 4-1단계 : 주어진 초성으로 3음절 단어 맞히기(한 문제당 50점, 모두 맞히면 50점) 단, 지명, 인명, 외래어는 제외
    * 4-2단계 : 사전적 정의 맞히기(형식은 전과 동일)
  * 왕중왕 도전 문제 : 2012~2013년 달인 도전 형식과 동일(도전에 성공하면 2000만원, 실패하면 1000만원만 획득)

## 7. 국군의 날 특집(2012년 10월 1일) 2인 1조 진행방식 ¶

  * 예비 : 장병 300명이 o/x퀴즈 3문제를 통해 과반수 이상이 생존하면 전원에게 1만원 상품권이 주어진다.(야외에서 진행)
  * 1단계 : 2인 1조로 우리말과 관련된 스피드 퀴즈 풀기(한 문제당 50점, 제한시간 60초)
  * 2단계 : 현재 진행되고 있는 연상문제 풀이와 동일, 총 7문제이며, 한 문제에는 포상휴가증이 걸려있다.(상위 2팀 결승 진출)
  * 3단계 : 십자말 풀이, 형식은 전과 동일`[1]`
  * 명예 달인 도전 : 병영일기에서 맞춤법과 띄어쓰기 올바르게 고치기, 맞춤법은 총 5문제가 주어지며 한 문제당 100만원이 주어진다(주관식도 있음), 띄어쓰기는 60초 간 문장을 고쳐쓰는 형식으로 진행되며, 성공하면 맞춤법 문제에서 받은 상금의 두 배가 주어진다. 모두 성공하면 1000만원을 획득하면서 명예 달인에 등극한다.

## 8. 한글날 특집(2012년 10월 8일) 진행방식 ¶

  * 5명의 중학생으로 이루어진 4팀이 대결하는 형식으로 진행된다.
  * 1단계 : 끝말잇기로 주어진 단어를 보고 끝말을 이어가는 형식, 명사만 허용되며 지명,인명,외래어는 정답으로 인정되지 않는다. 세 번의 기회가 주어지며, 한 문제 당 50점이 주어진다(최대 300점 획득 가능)
  * 2단계 : 소설 [메밀꽃 필 무렵](%EB%A9%94%EB%B0%80%EA%BD%83%20%ED%95%84%20%EB%AC%B4%EB%A0%B5.md)에서 발췌한 문장 속의 낱말 뜻풀이, 한 팀씩 개인으로 진행되며, 한 문장 당 100점이 주어진다.(세 번의 기회가 주어지며, 첫 번째 시도에서 모두 맞히면 500점 획득)
  * 3단계 : 주어진 초성으로 사자성어 만들기, 한 학생이 문제를 설명하면, 다른 네 사람이 한 음절 씩 답을 적는 형식으로 진행된다.(예 : ㄱㅅㅊㅎ-금상첨화)(총 3문제, 1문제 당 200점, 한 팀씩 진행) 여기서 점수가 높은 2팀이 다음 단계에 진출한다.
  * 4단계 : 소설 [메밀꽃 필 무렵](%EB%A9%94%EB%B0%80%EA%BD%83%20%ED%95%84%20%EB%AC%B4%EB%A0%B5.md)에서 발췌한 문장에서 문제가 출제되며, 맞춤법 5문제(한 문제 150점), 띄어쓰기 1문제(200점)로 구성된다. 여기서 점수가 높은 한 팀이 명예 우리말 달인에 도전할 자격을 얻게 된다.
  * 5단계 : 십자말풀이로 형식은 전과 동일하다. 다만, 한 사람씩 차례대로 문제를 풀게 되며, 한 사람이 탈락하면 다음 사람이 계속 문제를 푸는 형식으로 진행된다. 다섯 명이 모두 탈락할 때까지 문제풀이가 계속된다.
  * 명예 우리말 달인에 등극하면 우승팀에게는 500만원이 주어지며 나머지 3팀에게도 100만원이 주어진다.(실패시 4단계 누적 점수x1000의 상금이 주어진다)
