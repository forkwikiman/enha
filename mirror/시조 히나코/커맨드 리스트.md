  * 뺨 치기 - 천대 : → + A, A
  * 뺨 치기 - 금 : → + A, C
  * 샅바 두드리기 : ↘ + A
  * 밀쳐내기 : → + B, B

  * 안다리 후리기 : →↘↓↙←→ + B or D (잡기)

# 4. [KOF 2003](KOF%202003.md) ¶

  

★ 기본잡기  

  * 끌어당겨 던지기 : 근접해서 ← or → + C or D  

★ 특수기  

  * 앞먼지 털기 : ↘ + A
  * E.H 킥 : ↘ + D  

★ 필살기  

  * 밀어내기 : A or C 연타
  * 찔러 올리기 : ↓↙← + A
  * 찔러 떨구기 : ↓↙← + C
  * 팔잡아 던지기 : ←↙↓↘→ + B or D (잡기)
  * 초 박치기 : ←↙↓↘→ + A or C
  * 붙어 덮치기 : →↘↓↙←→ + A or C (잡기)  
┠ 찔러 내리기 : ↓↘→ + A  
┠ 걸고 던지기 : ↓↘→ + B  
┠ 샅바 잡아던지기 : ↓↘→ + C  
┗ 떠밀어내기 : ↓↘→ + D  

★ 초필살기  

  * 합장 비틀기 : ↓↘→↓↘→ + B or D
  * 대일번 : ↓↘→↓↘→ + A or C  

★ 리더 초필살기  

  * 양손 찌르기 : ↓↙←↙↓↘→ + AC
