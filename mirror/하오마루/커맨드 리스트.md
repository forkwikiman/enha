# 11. 네오지오 배틀 콜로세움 ¶

  * 잡기   
자세 흐트리기 ~ 비연 베기 : 접근해서 레버 중립 or → or ← + CD  

  * 특수기   
무릎 치고 차기 : ↙ + B  
슬라이딩 차기 : ↘ + D

  * 필살기   
오의 선풍 열참 : ↓↘→ + A or C  
오위 선풍 열참 : ↓↘→ + B or D  
오의 호월참 : →↓↘ + A or C  
오의 질풍 호월참 : 스텝 개시 직후 →↓↘ + A or C  
오의 열진참 : ↓↙← + A or C  
오의 참강섬 : →↘↓↙← + B or D  

  * 초필살기   
천패 처황참 : →↘↓↙←→ + A or C(기 1줄 사용)  
천패 봉신참 : ↓↘→↓↘→ + A or C (기 2줄 사용)  
분노 폭발 : ABC (기 3줄 사용)  

  * 어나더 더블 어설트   
일섬 : 파트너가 겐쥬로일 때 분노폭발 사용후 ↘↘ + AE
