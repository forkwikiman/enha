# 1. 가고일 ¶

  

{{|레어도 : S  
ST : 30  
HP : 50  
MHP : 50  
G : 90 + 지속성 영지X1  
제한 : 없음.  
능력 : 선제. 배치 시 ST=50  
설명 : 몸이 돌로 구성된 악마로 석상처럼 있다가 불시에 상대를 공격한다.|}}

  

전형적인 방어에 유리한 크리처. 방어시에 ST가 30 증가하여 50이 된다. 그리고 선제능력을 가지고 있기 때문에 조금만 신경써 준다면
안전하게 영지를 보호해 낼 수 있는 크리처. 기본 능력치도 꽤 쓸만한 편이다.

# 2. 구바 퀸 ¶

  

{{|레어도 : R  
ST : 10  
HP : 40  
MHP : 40  
G : 30 + 지속성 영지X2  
아이템 제한 : 무기.  
능력 : 영지[20](20.md) 대상 빈 토지에 '구바'를 배치한다.  
설명 : 구바들의 여왕. 몸에 알로 가득차 있어서 계속해서 구바를 낳을 수 있다.|}}

  

소환 조건은 아주 까다로우나 능력은 거의 초중반에나 어울릴 수준의 카드. 단 빈 영지가 많은 상황이라면 영지능력으로 선점을 시도해 볼만은
하나, 이 카드를 쓰느니 풍속성의 파우더 이터 쪽이 더 나을 수도 있다.

## 2.1. 구바 ¶

  

{{|레어도 : E  
ST : 0  
HP : 10  
MHP : 10  
G : 0  
아이템 제한 : 무기 / 방어구 / 스크롤.  
능력 : 없음  
설명 : 동그란 생물. 건드리면 터질 정도로 약한 생물. 일시적인 토지 파수꾼 정도로만 쓰인다.|}}

  

컬드셉트2의 카드들 중 유일하게 직접 북에 넣을 수 없으며 오직 구바 퀸의 영지능력을 서야만 처음 배치가능한 크리처. 능력은 최약수준이니
구바 퀸의 능력으로 이녀석을 배치한 영지는 최대한 빨리 다른 크리처 카드로 바꿔줘야 할 것이다.
