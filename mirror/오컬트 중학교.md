## 1. 개요 ¶

[썬더 일레븐](%EC%8D%AC%EB%8D%94%20%EC%9D%BC%EB%A0%88%EB%B8%90.md)에 등장했던 FF팀.
이름부터가 대놓고 [오컬트](%EC%98%A4%EC%BB%AC%ED%8A%B8.md)다.

  

이름답게 유령, [늑대인간](%EB%8A%91%EB%8C%80%EC%9D%B8%EA%B0%84.md) 같이 오컬트에 관련된 특이한
생김새를 한 선수들이 많다. 그래서인지 사용하는 스킬도 원령,
[카미카쿠시](%EC%B9%B4%EB%AF%B8%EC%B9%B4%EC%BF%A0%EC%8B%9C.md),
[매직](%EB%A7%A4%EC%A7%81.md) 등 오컬트틱한 기술들이 많다. 생김새도 생김새고, 상대방에게 저주를 건다는 소문이
돌아서 [라이몬중학교](%EB%9D%BC%EC%9D%B4%EB%AA%AC%20%EC%A4%91%ED%95%99%EA%B5%90.md)의 선수들은
으스스하다고 무서워했다.

  

실질적인 팀의 특기는 팀 단위의 [최면술](%EC%B5%9C%EB%A9%B4%EC%88%A0.md)(!)인 **고스트 록**. 하지만
고스트 록이 간파당하면서 결국 4:3로 [라이몬중학교](%EB%9D%BC%EC%9D%B4%EB%AA%AC%20%EC%A4%91%ED%95%99%EA%B5%90.md)에게 깨지고
만다.`[1]`

  

강팀이긴 했나 본지 [풋볼프런티어](%ED%92%8B%EB%B3%BC%20%ED%94%84%EB%9F%B0%ED%8B%B0%EC%96%B4.md) 지역 예선에도
참가했지만, 첫번째 경기에서 [슈요메이토 학원](%EC%8A%88%EC%9A%94%EB%A9%94%EC%9D%B4%ED%86%A0%20%ED%95%99%EC%9B%90.md)한테 어이없게 1:0으로 패배해 탈락하고 만다.

  

코믹스판에선 본격적으로 라이몬과 연습 시합을 가지기 전에 주장인 유코쿠가 엔도와 한판 대결했었다. 본 시합에서도 고스트 록으로 라이몬을
압도하나, 두번째에는 엔도가 **근성으로** 끊어버린다. 이후 엔도의 격려로 한계를 타파한 소메오카의 드래곤 크래쉬, 고엔지와 소메오카의
드래곤 토네이도로 점수를 뺏겨 결국 2:1로 라이몬에게 패배. 유코쿠는 저런 괴물들과 싸우는 건 무리라며 **얼이 빠졌다.**

## 3. 선수 프로필 ¶

※ 프로필은 1편을 기준으로 작성했습니다. 그리고 2편의 자력기 설명이 없는 캐릭터는 1편의 것과 동일하기 때문에 일부러 쓰지 않았습니다.
1편에선 제2장 클리어 후 모든 선수들을 영입 가능하다. 하지만 2편에선 산즈, 츠키무라, 레이겐, 블라드, 후시, 이렇게 5명만 동료로
영입할 수 있다.

  

■ 유코쿠 히로유키(幽谷博之) - 포지션 FW(9) / 속성 林  
오컬트중 축구부의 캡틴. 1학년. 한국판에서 이름은 **유령후**. 가운데에 외눈이 그려진 반다나 헤어밴드가
트레이드마크.<del>[심안](%EC%8B%AC%EC%95%88.md)이라도 쓰는건가</del> 날카로운 영감을 살려 유명 선수의
영혼으로부터 코치를 받고 있다. 모티브는 [유령](%EC%9C%A0%EB%A0%B9.md). 3편에선 일본 대표 후보팀 [네오재팬](%EB%84%A4%EC%98%A4%20%EC%9E%AC%ED%8C%AC.md)의 멤버로 등장한다.  
-자력기(1편) : 카미카쿠시(S,山) / 팬텀 슛(S,林) / 프리즈 샷(S,林) / 카마이타치(D,風)  
-자력기(2편) : 카미카쿠시(S,山) / 팬텀 슛(S,林) / 프리즈 샷(S,林) / 디바인 애로(S,風)

  

■ 나타 주조(鉈十三) - 포지션 GK(1) / 속성 風  
오컬트중 2학년. 한국판 이름은 **마일삼** [제이슨](%EC%A0%9C%EC%9D%B4%EC%8A%A8.md)과 같은 가면을
장착하고 있다. 그라운드의 풀베기에 손도끼를 사용하는 위험한 녀석. "언빌리버블!"이 말버릇. 참고로 이름이 나타(손도끼)에다가
주조(13).  
-자력기(1편) : 킬러 블레이드(C,林) / 일그러진 공간(C,林) / 블레이드 어택(B,風) / 회오리(C,風)  
-자력기(2편) : 절약!(skill) / 킬러 블레이드(C,林) / 일그러진 공간(C,林) / 화염방사(C,火)

록"을 사용한다. 이름의 모티브는 [팔묘촌](%ED%8C%94%EB%AC%98%EC%B4%8C.md).  
-자력기 : 저주(D,林) / 그림자 묶기(B,林) / 팬텀 슛(S,林) / 카마이타치(D,風)

-자력기(2편) : 매직(D,林) / 팬텀 슛(S,林) / 문설트(D,風) / 트윈 부스트(S,火)
