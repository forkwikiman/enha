# 1. [KOF 2001](KOF%202001.md) ¶

  

★ 기본잡기  

  * 고 고 프텐 걸 : 근접해서 ← or → + C
  * 스타리트 필드 : 근접해서 ← or → + D  

★ 특수기  

  * 센스리스 채터 : → + B
  * 앳 더 웨이스트랜드 : 공중에서 ↓ + D  

★ 필살기  

  * 포멀리슈츠 블루 : ↘ + B or D
  * 레드스카이 오브 야포네시아 : →↘↓↙← + B or D
  * 매드 머더 룰렛 : ←↓↙ + B or D (잡기)  

<언체인 스타트 기술>  

  * 웰컴 투 류쿠 : 포멀리슈츠 블루 사용 중 A or C
  * 시티즌 오브 더 월드 : 포멀리슈츠 블루 사용 중 B or D
  * 레픈 카무이 : ↓↙← + A or C
  * 비욘드 더 플레임즈 : ←↙↓↘→ + B or D  

<언체인 서클 기술>  

  * 1\. 바이바이 류쿠(6 ↔ 1 ↔ 2) : ↑ + A or C
  * 2\. 임포턴트 심프텀(1 ↔ 2 ↔ 3) : ↓ + B or D
  * 3\. 버기 앤 커핀(2 ↔ 3 ↔ 4) : → + A or C
  * 4\. 쉘터 프롬 스톰(3 ↔ 4 ↔ 5) : ↑ + B or D
  * 5\. 트레인 투 시 체리 블러섬즈(4 ↔ 5 ↔ 6) : ↓ + A or C
  * 6\. 위드 어 램프 포 패스웨이(5 ↔ 6 ↔ 1) : → + B or D  

<언체인 피니쉬 기술>  

  * 크라운 언더 더 스카이 : →→ + A or C
  * 스테이트 오브 히트 헤이즈 : →→ + B or D
  * 어 풀 문 이브닝 : ↓↙← + A or C
  * 라이드 유어 사이클 : ←↙↓↘→ + B or D
  * 레드스카이 오브 야포네시아 : →↘↓↙← + B or D
  * 어 가든 투 플레이 위드 고스트 : ←↓↙ + B or D  

★ 초필살기  

  * 로얄리티 테스트 포 더 리벨러리스트 : 언체인 서클 기술 사용 중 ←→↓↘ + A or C
  * 더 어나이텐티파이트 빅팀 컨셔스네스 : ←→↓↘ + B or D  

★ MAX 초필살기  

  * 로얄리티 테스트 포 더 리벨러리스트 : 언체인 서클 기술 사용 중 ←→↓↘ + AC

★ 특수기

  * 센스리스 피스트 : ←→ + B or D
  * 레픈 카무이 : ↓↙← + A or C
  * 비욘드 더 플레임즈 : ←↙↓↘→ + B or D  

<언체인 서클 기술>  

  * 1\. 바이바이 류쿠(6 ↔ 1 ↔ 2) : ↑ + A or C
  * 2\. 임포턴트 심프텀(1 ↔ 2 ↔ 3) : ↓ + B or D
  * 3\. 버기 앤 커핀(2 ↔ 3 ↔ 4) : → + A or C
  * 4\. 쉘터 프롬 스톰(3 ↔ 4 ↔ 5) : ↑ + B or D
  * 5\. 트레인 투 시 체리 블러섬즈(4 ↔ 5 ↔ 6) : ↓ + A or C
  * 6\. 위드 어 램프 포 패스웨이(5 ↔ 6 ↔ 1) : → + B or D

  * 오마가토키 : ← or → or 레버 중립 + BCD

  * 스테이트 오브 히트 헤이즈 : →→ + B or D
  * 어 풀 문 이브닝 : →→ + C
  * 코스믹 프텐 스윙 : ↓↘→ + A
  * 로스트 홈랜드 : ↓↘→ + C
  * 비욘드 더 플레임즈 : ←↙↓↘→ + B or D  

★ 초필살기  

  * 로얄리티 테스트 포 더 리벨러리스트 : 언체인 서클 기술 사용 중 ←→↓↘ + A or C  

★ MAX 초필살기  

  * 로얄리티 테스트 포 더 리벨러리스트 : 언체인 서클 기술 사용 중 ←→↓↘ + AC
  * 윈즈 페어 그라운드 : ←→↓↘ + BD (지상 반격시)
  * 블루 먼데이 퍼레이드 : ←→↓↘ + BD (공중 반격시)  

★ MAX2 초필살기  

  * 서바이버즈 뱅큇 : 블루 먼데이 퍼레이드 공중 반격 성공시 BCD
