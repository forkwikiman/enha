※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 두개골 부수기 : 근접해서 ← or → + B  

★ 특수기  

  * 안개 가르기 : 대쉬 중 B
  * 뼈 꿰매기 : 상대 다운시 ↘ + B   

★ 필살기  

  * 유영진 : ↓↘→ + A or B  
┗ 유영인 : 유영진 중 ↓↘→ + A or B

  * 강철베기 ・ 개 : →←↙↓↘→ + B
  * 오보로 베기 : →↓↘ + C
  * 기공포(●) : ↓↙← + B
  * 수면 숨기 : ←↓↙ + A
  * 천마 떨구기 : 근접해서 →↘↓↙← + C
  * 뼈 꿰매기 : 공중에서 ↓↘→ + A or B
  * 천마각 : 공중에서 ↓ + C  
┗ 전륜 : 천마각 중 ↓ + C  

★ 초오의  

  * 당랑쌍인 : ↓↙←→ + AB  
┗ 유영 ・ 아 : 당랑쌍인 중 ↓↘→ + B  

┗ 유영 ・ 우 : 유영 ・ 아 중 ↓↘→ + B

┗ 천마 ・ 아 : 당랑쌍인 중 ↓↘→ + C  

┗ 천마 ・ 우 : 천마 ・ 아 중 ↓↘→ + C  

★ 잠재오의  

  * 어둠 사냥 : 근접해서 →↘↓↙←→↘↓↙← + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 두개골 부수기 : 근접해서 ← or → + CD  

★ 특수기  

  * 뼈 꿰매기 : 상대 다운시 ↘ + B   

★ 필살기  

  * 유영인 : ↓↘→ + A
  * 기공포(●) : ↓↙← + B
  * 오보로 베기 : →↓↘ + C
  * 영법사 : →←→ + C  
┣ 골파 베기 : 영법사 중 ↓↙← + A  
┗ 안개 가르기 : 영법사 중 ↓↙← + B

  * 수면 숨기 : ←↓↙ + A
  * 천마 떨구기 : 근접해서 →↘↓↙← + C (공중 가능)
  * 뼈 꿰매기 : 공중에서 ↓↘→ + A or B (2회 연속 입력)
  * 천마각 : 공중에서 ↓ + C  
┗ 전륜 : 천마각 중 ↓ + C  

★ 초오의  

  * 단강인 : →←↙↓↘→ + AB (모으기 가능)  

★ 잠재오의  

  * 어둠 사냥 : 근접해서 →↘↓↙←→↘↓↙← + B   

★ 난무오의  

  * 당랑쌍인 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, A, ↓↘→ + C  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, A, → + BC, A, B, ↓↘→ + B
