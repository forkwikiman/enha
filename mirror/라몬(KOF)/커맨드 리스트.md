# 5. [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * 암 휩 : 근접해서 → or ← + C or D  

★ 특수기  

  * 헤드 버트 : → + A
  * 저공 드롭 킥 : ↘ + B  

★ 필살기  

  * 타이거 넥 챈서리 : ←↙↓↘→ + A or C (잡기)
  * 롤링 소버트 : →↓↘ + B or D  
┗ 플라잉 보디 어택 : →↓↘ + B or D  

┗ 일으켜 세우기 : ↓↓ + A or C

  * 서머솔트 : ←↙↓↘→ + B or D (ABC로 캔슬 가능)  
┗ 페인트 대쉬 : 서머솔트 대쉬 중 E

  * 페인트 스텝 : ↓↙← + A or C (모으기 가능)
  * 타이거 로드 : ↓↙← + B or D  
┠ 플라잉 보디 어택 : 타이거 로드 중 벽에 닿는 순간 ↓↘→ + A or C  
┠ 크로스 찹 : 타이거 로드 중 벽에 닿는 순간 ↓↘→ + B or D  
┠ 부파도라 : 타이거 로드 중 벽에 닿는 순간 ↓↙← + A or C  
┗ 페인트 대쉬 : 타이거 로드 대쉬 중 E

  * 로프 흔들기 : ←↙↓↘→ + E (잡기)  

★ 초필살기  

  * 타이거 스핀 : →↘↓↙←→↘↓↙← + A or C (잡기)
  * 엘 디아블로 아마릴로 라몬 : ↓↙←↙↓↘→ + B or D
  * 새비지 파이어 캣 : ↓↙←↙↓↘→ + E  

★ 리더 초필살기  

  * 타이거 콤비네이션 : →↘↓↙←→↘↓↙← + E  
┗ 플라잉 메이어 : 타이거 콤비네이션 1타 중 ↓↘→ + E  

┗ 암 휩 : ↓↙← + E
