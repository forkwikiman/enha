## 1. 설명 ¶

![ERI.jpg](//rv.wkcdn.net/http://rigvedawiki.net/r1/pds/ERI.jpg)

[JPG image (57.42 KB)]

  
[뱅가드 프린세스](%EB%B1%85%EA%B0%80%EB%93%9C%20%ED%94%84%EB%A6%B0%EC%84%B8%EC%8A%A4.md)의 플레이어블 캐릭터.

  

고아원에서 불우하게 살다가 뛰어난 운동신경을 인정받게 되어, 좀더 강해지고 싶다는 마음에 정부군 연구시설로 가서 인체실험 대상이 되었다.
하지만 거듭되는 가혹한 생체실험을 못 견딘 나머지 자유를 찾아, 친구인 [미루마치 쿠루미](%EB%AF%B8%EB%A3%A8%EB%A7%88%EC%B9%98%20%EC%BF%A0%EB%A3%A8%EB%AF%B8.md)의 곁으로 되돌아가기 위해 목숨을 건 탈출을
감행하는데...

  

어두운 배경 스토리와는 딴판으로 시원시원한 옷차림이나, 쩍벌녀 킥이라고밖에 할 수 없는 점프 C 같은 걸 보면 도무지 긴장감이 느껴지지 않는
캐릭터. 에리나 쿠루미 어느 쪽의 엔딩을 봐도 쿠루미의 페이스에 거의 일방적으로 끌려다니는 수준이라 엔딩 후가 더 불쌍해 보인다. 참고로
게임 내 그래픽에 비해 일러스트에서의 머리 숱이 지나치게 [크고아름답다](%ED%81%AC%EA%B3%A0%20%EC%95%84%EB%A6%84%EB%8B%B5%EB%8B%A4.md)(...)

  

참고로 쿠루미와 동갑이다.(14살) 참고로 에리의 복장은 **쿠루미가 가지고있던 응원복이다**. 근데 쿠루미는 공식적으로 **미술부**다.

  

기존의 다단 콤보 위주 격게 스타일에 가장 충실한 캐릭터로 체인에서 주력 기술이 아무렇지도 않게 이어지기 때문에 기선을 잡으면 일방적으로
콤보러시가 가능. 공중 체인까지 있고 기술 발동도 빠르며 상대 스턴 유도도 꽤나 쉽다. 아무리 그래도 중공격이 2번이나 그냥 이어지는 것은
좀 너무하다 싶지만... 대신 기본 화력이 낮으며 히트 수를 많이 뽑는 만큼 데미지 보정도 쉽게 걸리고, 무엇보다 두부살이라 열심히 때려놓고
실수라도 했다가는 몇 번 안되는 상대편 콤보에도 체력이 금새 바닥에 가까워지니 최대한 안맞고 많이 때리는 것에 전념해야 한다.

  

1.01 버전에서는 B 다음 기술 캔슬이 안되고 스매쉬 비즈 파생중 경직 증가, ABA루트 띄우기 시간 축소 등 콤보가 대폭 약화되었다. 더
이상 기절루프는 불가. 그나마 데미지 보정이 적어진 게 위안거리. 대체용으로 AAA루트가 나을 듯. 게다가 피격판정이 넓어져서 별별 콤보가
들어갈 것으로 예상. 체력도 낮은데.. 대신 롤링 페더는 하향이 없어서 여전히 주력기.

  

픽시브에서 현재 팬아트 개수가 가장 많다. 그 다음은 루나나 사키 정도.  
인기가 너무 좋아서 [이 캐릭터만을 위한 전용
스레](http://schiphol.2ch.net/test/read.cgi/gamechara/1247440429/)도 뱅가드 프린세스 캐릭터
중 유일하게 존재한다.  
스매쉬 비즈의 보이스("이치,니노,상!(하나,둘,셋)")가 묘하게 중독성이 있는걸로도 유명.

## 2. 체인 아츠 ¶

  * ↓A -> A -> B  

  * A -> B -> B  
쿠루미에겐 들어가지 않는다.  

  * ↓A -> ↓B  

  * 점프 B -> 점프 C

## 3. 특수기 ¶

  * 무릎차기 (→A)  
리치가 짧은 공격. 필드에서는 쓸 일이 없고 구석 콤보용.  

  * 뛰어차기 (→B)  
점프 C와 같은 자세로 공격. 필드에서 맞으면 바로 상대 다운. 구석에서는 벽바운드로 추가타 가능. 마찬가지로 구석 콤보용.  
구석에서 확정적으로 3번을 맞출 수 있다.  

  * 불명(공중에서 →A)  
공중에서 돌려차기. 필드에서 맞으면 멀리 날아가고, 구석에서는 벽바운드가 일어난다. 막혀도 상대가 엄청나게 밀려난다. 버그인듯.

## 4. 기술 ¶

  * 에어 스파이크 (↓↓+A/B/C)  
뛰어올라 어퍼를 날리는 대공기. 공격력은 보통 수준이지만 판정이 좋고 발동도 빠르다. 상대가 뛰는 게 보였다 싶으면 바로바로 날려주자.  

  * 스매쉬 비즈 (↓↘→+A/B/C 이후 추가입력)  
톤파를 앞세우며 러시를 거는 연속입력 기술. 처음 커맨드만 넣으면 2격부터는 버튼만 눌러도 된다. 버튼에 따라 발동 속도가 달라지는데
실질적으로는 거의 A로만 쓰게 될 기술. 2격째부터는 버튼에 따라 공격 속성이 완전히 달라진다.  
2격 A : 올려치기. 전후 연계 가능.  
2격 B : 전진해 치기. 전후 연계 가능.  
2격 C : 중단기로 종료. 연계 불가.  
2격 ↓C : 슬라이딩. 연계 불가.  
  
3격 A : 돌려차기. 띄우기 판정. B에서 연계된 쪽이 더 높이 띄운다.  
3격 B : 에어 스파이크.  
3격 C : 연계되지 않는 찍기 공격. 벽 바운드.  
3격 ↓C : 슬라이딩. 연계 불가.  
3격 →→ : 빙글 돌며 넘어간다. 위치 바꾸기 기술.  
  
콤보에서 가장 많이 쓰게 될 연계는 ABA (1.01부터는 AAA) 로, 맞춘 다음 상대가 높이 뜨므로 공중 체인을 넣어줄 수 있다.
전진거리가 짧아 보여도 필드 체인에서 다 들어가니 속성에 따라 활용할 수 있도록 익혀둘 것.  

  * 리버스 소배트 (↓↙←+A)  
상대 뒤로 돌아가며 소배트를 날린다. 회피공격성 기술로 히트시 경직이 생기니 추가타 가능. 틈틈이 써줘 상대의 방어를 흔들어 놓는데
유용하다.  

  * 롤링 페더 (↓↙←+B/C)  
깃털로 전신을 감싸고 뛰어올라 회전하는 다단히트 기술. 자체 스턴치가 높으며 히트시 띄우기 판정이 있어 추가타 가능. 콤보에서 조금만 섞어
써줘도 바로 스턴을 뽑아낼 수 있을 정도다. B로 쓰면 후딜레이가 거의 없어서 주력기.

## 5. 리버티 아츠 ¶

  * 스팅거 레이드 (↓↘→+BC)  
[톤파](%ED%86%A4%ED%8C%8C.md) 러시를 날리는 기술. 발동이 상대에게 닿지 않으면 2격째 자세에서 발동이 끝난다.
공중 히트시 전부 맞지 않고 막타는 구석에 가까우면 벽바운드 판정이라 추가타가 가능하지만 그 앞에 콤보를 얼마나 넣었느냐에 따라 바로
낙법가능 판정으로 환원될 수도 있으니 주의. 2타가 모두 헛치면 이후에 3타로 스매쉬 비즈 3타와 같은 기술을 넣을 수도 있다. 하지만 일반
버튼으로 쓰는 기술에 한정된다.

## 6. 콤보 ¶

  

  * 점프 C -> 스매쉬 비즈 AAA -> 점프 B -> 점프 C  

  * 리버스 소배트 -> 스매쉬 비즈 or 리버티 아츠  
리버스 소배트 발동거리에 따라 안 들어갈 수도 있으니 주의.  

  * A -> B -> 에코 →D -> 스매쉬 비즈 or 리버티 아츠  

  * (구석) 점프 B -> 점프 C -> A -> B -> B  
1.0 전용 스턴콤보. 고작 체인 둘만 이어놓은 것 뿐인데 스턴이라니 할 말이 없다... 필드에서는 마지막 B가 안 들어간다.
1.01버전부터는 공중체인 이후 지상 체인 넣기가 어렵고, 스턴도 안 나게 조정되었다.  

  * (구석) 점프 C -> 스매쉬 비즈 AAA -> →B -> →A -> 에코 ↓D -> 중 롤링 페더 -> 점프 B -> 점프 C  

  * (구석) 점프 →A or 롤링 페더B -> →B -> →B -> →B -> 에코 ↓D -> →B -> →B -> 에어 스파이크 or 점프 B-C  
나탈리아에겐 들어가지 않는다.  

  * (구석) 점프 C -> 스매쉬 비즈 AAA -> 점프 B -> 점프 C -> 에코 ↓D -> 중 롤링 페더  
아직도 건재한 스턴콤보... 에코 구겨넣는 타이밍만 익히면 간단.
