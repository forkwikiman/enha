[슈퍼로봇대전](%EC%8A%88%ED%8D%BC%EB%A1%9C%EB%B4%87%EB%8C%80%EC%A0%84.md)에서
[겟타로보](%EA%B2%9F%ED%83%80%EB%A1%9C%EB%B3%B4.md)에게 추가해준 필살기.

분리와 합체를 반복하면서 적을 <del>농락</del>공격한다.

첫 등장시는 대설산 던지기로 띄우고 공중에 따라가서 1호기로 합체, 토마호크로 썰고, 이후 날아가는 상대를 겟타 2로 마무리하는
기술이였지만, 이후 MX에서 대폭 강화되어 마무리가 [겟타드래곤](%EA%B2%9F%ED%83%80%20%EB%93%9C%EB%9E%98%EA%B3%A4.md)의 겟타 빔으로 변경되었다.

이후 3차 알파에서는 게임내 최강급 연출로 등장하며, 겟타 사이클론으로 띄우고, 마하 스페셜을 쳐먹여준 후, 겟타 빔으로 다운 공격을 먹이는
ALL 병기로 등장했다. 연출은 핵폭발.

이후 Z 등에도 빠짐없이 등장하며 겟타의 공식 필살기로 자리매김했다.

그리고 **매 시리즈마다 연출이 변한다.**

특히 Z 에서는 드래곤,라이거,포세이돈이 쓸 경우의 연출이 다르고 각각 다른 무장으로 설정되어 있다.  

  * 겟타 드래곤: 체인지 라이거-> 드릴 미사일 난사-> 체인지 포세이돈-> 스트롱 미사일 연사-> 체인지 드래곤-> 겟타 레이저 캐논 연사
  * 겟타 라이거: 체인지 포세이돈-> 겟타 사이클론-> 체인지 드래곤-> 겟타 토마호크->체인지 라이거-> 마하 스페셜
  * 겟타 포세이돈: 체인지 드래곤-> 킥을 날린 후 토마호크 부메랑-> 체인지 라이거-> 체인으로 토마호크를 잡고 체인을 회전시키면서 난도질-> 체인지 포세이돈->핑거 네트로 적을 잡고 빙빙 돌리다가 내리 꽂으며 스트롱 미사일  

후속작인 2차 Z에서도 진겟타는 각각 다른 연출이다. 단 노멀겟타는 탄수 및 지형 적응은 구분되지만 연출은 동일하다.

  * 노멀 겟타: 체인지 겟타2 -> 드릴 암을 상대에게 박아넣고 돌진 -> 체인지 겟타3 -> 겟타 암으로 상대를 공중으로 집어던진다 -> 체인지 겟타1 -> 겟타 토마호크로 마무리 베기.  

  * 진 겟타1: 체인지 겟타3 -> 해머 펀치로 적을 공중으로 날림 -> 체인지 겟타2 -> 공중에서 적을 붙잡고 상대를 땅에 쳐박아버림 -> 체인지 겟타1 -> 겟타 빔.  

  * 진 겟타2: 체인지 겟타1 -> 다수의 토마호크 랜서 -> 체인지 겟타3 -> 겟타 미사일 연사 -> 체인지 겟타2 -> 드릴 허리케인 난타 이후 드릴을 박아 넣고 플라즈마 드릴 허리케인.  

  * 진 겟타3: 체인지 겟타2 -> 플라즈마 드릴 허리케인으로 적을 공중에 띄움 -> 체인지 겟타1 -> 회오리 안에서 겟타 배틀 윙 -> 체인지 겟타3 -> 떨어지는 적을 받고 대설산 떨구기  

3차 Z [시옥편](%EC%8B%9C%EC%98%A5%ED%8E%B8.md)에서는 오직 진 겟타1만 사용 가능하다. 연출은 체인지
겟타2 -> 드릴 허리케인 -> 체인지 겟타3 -> 대설산 떨구기 -> 체인지 겟타1 -> 겟타 빔.

[천옥편](%EC%B2%9C%EC%98%A5%ED%8E%B8.md)에서는 다시 각각 다른 연출로 변경되었다.

  * 진 겟타3: 체인지 겟타1 -> 겟타 레저 -> 체인지 겟타 2 -> 드릴 암 -> 체인지 겟타 3 -> 대설산 떨구기 이후 미사일 발사
