## 1. 개요 ¶

[제미니 스톰](%EC%A0%9C%EB%AF%B8%EB%8B%88%20%EC%8A%A4%ED%86%B0.md)과 [엡실론](%EC%97%A1%EC%8B%A4%EB%A1%A0%28%EC%9D%B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9D%BC%EB%A0%88%EB%B8%90%29.md)을 능가하는 마스터 랭크의 축구팀. 첫등장시 엡실론이 마지막인 줄 알았던 라이몬 일레븐을 말그대로
충격과 공포에 빠트렸다. 원래 팀 이름은 '가이아'. __더 제네시스__란 건 에일리아 학원 내 최강팀에게 주어지는 칭호으로, 다른 마스터
랭크팀인 [프로미넌스](%ED%94%84%EB%A1%9C%EB%AF%B8%EB%84%8C%EC%8A%A4.md) 및 [다이아몬드 더스트](%EB%8B%A4%EC%9D%B4%EC%95%84%EB%AA%AC%EB%93%9C%20%EB%8D%94%EC%8A%A4%ED%8A%B8.md)와 경쟁해서 얻어냈다.

  

[요카토 중학교](%EC%9A%94%EC%B9%B4%ED%86%A0%20%EC%A4%91%ED%95%99%EA%B5%90.md)에서
처음으로 등장한다. 애니메이션에선 그란이 엔도에게 축구 하자면서 나타나 [라이몬중학교](%EB%9D%BC%EC%9D%B4%EB%AA%AC%20%EC%A4%91%ED%95%99%EA%B5%90.md)와 시합을
벌이는데, 엡실론과는 비교도 안될 정도의 어마어마한 실력으로 라이몬을 20:0으로 발라버린다. 단순한 슛만으로 엔도의 [마신 더핸드](%EB%A7%88%EC%8B%A0%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)를 뚫어버리는데, 이때 그란의
한마디가 진짜 압권. **"어? 들어가버렸네?"** 이 시합으로 인해 [카제마루 이치로타](%EC%B9%B4%EC%A0%9C%EB%A7%88%EB%A3%A8%20%EC%9D%B4%EC%B9%98%EB%A1%9C%ED%83%80.md)와 [쿠리마츠 텟페이](%EC%BF%A0%EB%A6%AC%EB%A7%88%EC%B8%A0%20%ED%85%9F%ED%8E%98%EC%9D%B4.md)가 절망감에 빠져 팀을
이탈해버리고, [후부키 시로](%ED%9B%84%EB%B6%80%ED%82%A4%20%EC%8B%9C%EB%A1%9C.md)는 그란의
유성블레이드를 막으려다 육체와 정신 모두 중상을 입게 된다. 게다가 엔도는 그런 동료들의 이탈에 충격받아 식음을 전폐한채 한동안 폐인 상태로
지내기까지 했다.

  

게임에서는 요카토중을 파괴하기 위해 찾아오는데, 말도 안되는 스피드와 파워로 요카토중 축구부를 완전히 처바른다.
[#](http://www.youtube.com/watch?v=wwXGMSdSizQ) 그러자 라이몬 일레븐이 나서서 상대하지만, 막강한 실력
차이로 결국 라이몬도 제네시스에게 완패하고 만다. 시합 종료 뒤, 학교를 파괴하려는 제네시스를 엔도가 필사적으로 막아서고, 그런 엔도를
보면서 그란이 망설이기 시작한다. 그러자 울비다가 비웃으며 엔도에게 그란이 "키야마 히로토"라는 사실을 폭로해버리고, 이에 당황한 그란은
황급히 팀원들을 이끌고 물러난다. 이 시합에선 카제마루가 좀 활약하는데, 그걸 지켜보던 '에일리아 학원의 보스'가 그를 위험하다고 여겨
제네시스 팀원들에게 명령을 내려서 카제마루를 박살내고, 결국 카제마루는 중상을 입고 팀에서 빠지게 된다.

  

이후 후지산에 위치한 에일리아 학원의 본거지에서 라이몬은 더 제네시스와 마지막 결전을 펼치는데 여기에서 여러가지 충격적인 사실이 밝혀진다.
자세한건 [에일리어학원](%EC%97%90%EC%9D%BC%EB%A6%AC%EC%96%B4%20%ED%95%99%EC%9B%90.md)의 스포일러 항목
참조.

  

[에일리어 석](%EC%97%90%EC%9D%BC%EB%A6%AC%EC%96%B4%20%EC%84%9D.md)에서 에너지를 받아야 하는
하위 랭크팀과는 달리, 더 제네시스 팀은 순전히 재능과 노력만으로 실력을 얻은 강자들이다. 사실 제미니스톰과 입실론도 이들의 연습 상대에
불과했다. 또한 평소에는 본 실력을 숨기고 있지만, [리미터해제](%EB%A6%AC%EB%AF%B8%ED%84%B0%20%ED%95%B4%EC%A0%9C.md)를 하게 되면 육체의 한계를
뛰어넘어 더욱 더 강력한 운동능력을 발휘할 수 있게 된다. 다만 육체가 그걸 버티지 못하기 때문에 매우 위험하다.

  

그리고 제네시스의 멤버들은 모두 [키라세이지로](%ED%82%A4%EB%9D%BC%20%EC%84%B8%EC%9D%B4%EC%A7%80%EB%A1%9C.md)가 후원하는
고아원인 '햇님원'의 아이들로, 어릴 때부터 재능을 인정받아 훈련받았으며, 종종 방문하던 키라 회장을 아버지라고 부르면서 그를 위해서라면
무슨 짓이든 하려고 한다.

  

마스터 랭크팀 다운 시합을 보이면서 시종일관 압도했으나, 역시나 라이몬의 동료 파워에는 당해낼 수 없었던지라, 최후에는 [디어스](%EB%94%94%20%EC%96%B4%EC%8A%A4.md)에 골문이 뚫려 4:3으로 패배. 시합에서 진 뒤로는 다 같이
경찰에 보호되었다.

  

전용 BGM은 [신인류 더 제네시스](http://www.youtube.com/watch?v=9DOZ4hcwpog). 시합 BGM은 [별의
사도](http://www.youtube.com/watch?v=nx6pFjNoAC4).

  

게임 2 및 3편에선 히든 팀으로 **"신과 우주"** 란 [제우스중학교](%EC%A0%9C%EC%9A%B0%EC%8A%A4%20%EC%A4%91%ED%95%99%EA%B5%90.md)+더 제네시스의
혼성팀으로 등장하기도 한다. 팀에 포함된 제네시스 멤버는 그란,울비다,게일,조한,퀼, 이렇게 5명.

## 3. 선수 프로필 ¶

※ 프로필은 게임판을 기준으로 작성해주세요. 2편에선 조한,하우저를 제외한 모든 제네시스 멤버를 동료로 영입할 수 있다. 그리고 3편에선
특정조건 만족시 2편과의 수퍼링크를 통해 멤버 전원을 영입할 수 있다. 특히나 그란은 3편의 히로토와는 별개로 **더 제네시스 버전
그란**을 따로 영입하는게 가능하다.

  

■ 그란 / 키야마 히로토(基山ヒロト) - 포지션 FW(11) / 속성 火  
더 제네시스의 캡틴. 가는 눈에 적색 머리카락의 남자아이, 피부가 하얗다. 에일리어 학원 최강의 전사로, 그 슛은 마치 유성과도 같다.
온화하고 예의 바른 어조로 애기하지만, 어딘가 차가운 분위기를 지니고 있었다. 처음엔 정체를 숨긴채 "키야마 히로토"란 이름으로 엔도 앞에
나타났다. '아버지'로부터 받은 은혜를 갚기 위해 계속 싸워 왔지만, 라이몬과의 최종결전 중반에서 아버지가 전 멤버의 리미터 해제 명령을
내리자, 이에 그란은 동료들을 걱정해 명령을 거부하고 결국 캡틴 자리에서 격하된다. 최후에는 라이몬의 동료를 생각하는 결속의 앞에 패배하고,
"동료의 힘"의 굉장함을 인정한다.  
-자력기(2편) : 유성블레이드(S,火) / 수퍼노바(S,林) / 포톤 플래쉬(B,火) / 사우던크로스 컷(D,林)  
-자력기(3편) : 유성블레이드(S,火) / 시저스 붐(D,山) / 수퍼노바(S,林) / 포톤 플래쉬(B,火)

  

■ 울비다 / 야가미 레이나(八神玲名) - 포지션 MF(10) / 속성 風  
파란 머리카락에 단발머리의 여자아이. 더 제네시스의 숨겨진 우두머리 같은 존재. 냉정한 전사로 감정에 휘둘려 행동하는 일은 없다. 그란처럼
'아버지'를 위해 계속 싸워왔으며, 라이몬과의 최종결전에선 중반에 아버지의 명령을 거역한 그란을 제치고 제네시스의 캡틴으로 임명된다.  
-자력기(2편) : 아스트로 브레이크(S,林) / 미티어 샤워(D,火) / 스페이스 펭귄(S,風) / 지그재그 스파크(D,風)  
-자력기(3편) : 아스트로 브레이크(S,林) / 마타도르 페인트(D,林) / 수퍼 엘라시코(D,山) / X 블래스트(S,火)

  

■ 네로/ 네무로 키미유키(室君之) - 포지션 GK(1) / 속성 風  
초록색 머리카락에, 적안과 흰 피부를 지닌 몸집이 작은 소년. 무서운 점프력으로 모든 슛에 뛰어들을 수가 있다.  
-자력기(2편) : 프로키온 네트(C,風) / 시공의 벽(C,風) / 포톤 플래쉬(B,火) / 일그러진 공간(C,林)  
-자력기(3편) : 프로키온 네트(C,風) / 시공의 벽(C,風) / 크리티컬!(skill) / 속성강화(skill)

  

■ 게일 / 쿠지라이 타카노리(鯨井隆則) - 포지션 DF(2) / 속성 風  
포니테일의 갈색 머리카락에 파충류 같은 눈을 하고 있다. 날카로운 시선으로 상대의 움직임을 가만히 관찰한다.  
-자력기(2편) : 포톤 플래쉬(B,火) / 시그마 존(B,林) / 잔상(D,林) / 라이트닝 액셀(D,林)  
-자력기(3편) : 시그마 존(B,林) / 데저트 스톰(B,風) / 라이트닝 액셀(D,林) / 바바리안의 방패(B,山)

  

■ 키브 / 키이 후미코(紀伊布美子) - 포지션 DF(3) / 속성 火  
여자아이. 웨이브결의 적색 머리카락에 갈색피부. 춤과 같은 이상한 움직임으로 상대를 혼란시킨다.  
-자력기(2편) : 라이트닝 액셀(D,林) / 그래비테이션(B,山) / 블랙홀(C,林) / 프로키온 네트(C,風)  
-자력기(3편) : 라이트닝 액셀(D,林) / 그래비테이션(B,山) / 프로키온 네트(C,風) / 건 샷(S,火)

  

■ 조한 / 이시다이라 한조(石平半蔵) - 포지션 DF(4) / 속성 山  
철 마스크를 쓰고 있다. 격투기로 단련한 몸으로 골 앞에 서서 찬다. 마스크 탓에 무슨 말을 해도 "즈즈즈"라고 밖에 들리지 않지만,
유일하게 코마자와는 그의 말을 이해할 수 있었다. 3편에선 일본 대표 후보팀 [네오재팬](%EB%84%A4%EC%98%A4%20%EC%9E%AC%ED%8C%AC.md)의 일원으로 등장한다.  
-자력기(2편) : 리커버리(skill) / 플래닛 실드(B,火) / 수퍼 아르마딜로(D,山) / 그래비테이션(B,山)  
-자력기(3편) : 리커버리(skill) / 어스퀘이크(B,山) / 하베스트(B,林) / 부스트 글라이더(D,火)
