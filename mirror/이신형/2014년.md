### 1.1. [2014 WCS Korea Season 1·HOT6 GSL Season1](2014%20WCS%20Korea%20Season%201%C2%B7HOT6%20GSL%20Season%201.md) ¶

전 시즌의 아쉬운 모습을 뒤로 하고 좋은 모습을 기대한 팬들이 많았지만 Code A에서 죽음의 조를 배정받는다. 1경기에서
[주성욱](%EC%A3%BC%EC%84%B1%EC%9A%B1.md)에게 0:2 완패. 패자전에서도 [김준호](%EA%B9%80%EC%A4%80%ED%98%B8%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9D%B4%EB%A8%B8%29.md)에게 1:2
완패를 하면서 Code S 진출에 실패한다. 단 2시즌만에 4강에서 Code A 탈락이라는 엄청난 하락을 기록하면서
[SAB라인](SAB%EB%9D%BC%EC%9D%B8.md)에 이름을 올리게 되었다.

### 1.2. [2014 WCS Korea Season 2·HOT6 GSL Season2](2014%20WCS%20Korea%20Season%202%C2%B7HOT6%20GSL%20Season%202.md) ¶

전 시즌의 부진을 딛고 예선을 통과, 코드 A에서 SKT T1의 저그 박령우, CJ 엔투스의 프로토스 김정훈을 꺾고 코드 S 복귀에 성공.

  

5월 24일, 32강 마지막 조 H조에서 [김준호](%EA%B9%80%EC%A4%80%ED%98%B8%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9D%B4%EB%A8%B8%29.md)에게 1:2로 패배했으나 패자전에서
[박현우](%EB%B0%95%ED%98%84%EC%9A%B0.md)를 2:0, 최종전에서
[이동녕](%EC%9D%B4%EB%8F%99%EB%85%95.md)을 2:0으로 압도적으로 꺾고 16강에 합류했다.

  

6월 6일, 16강 D조에서 [주성욱](%EC%A3%BC%EC%84%B1%EC%9A%B1.md)에게 0:2로 지며 패자전으로 떨어졌고,
[이영한](%EC%9D%B4%EC%98%81%ED%95%9C.md)의 올인성 공격을 깔끔하게 막아내 주성욱과의 경기에서 보여준 불안함을
씻어내며 최종전을 맞이했다. 상대는 [김민철](%EA%B9%80%EB%AF%BC%EC%B2%A0.md), 오랜만에 벌어진 이 혁벽대전은
명경기를 만들어냈으나 결국 3세트 프로스트에서 이신형이 준비한 필살의 타이밍 러시가 김민철의 철벽 수비에 막히면서 16강에서 탈락하고
말핬다.

### 1.3. [2014 DreamHack Open:Bucharest](2014%20DreamHack%20Open%3A%20Bucharest.md) ¶

Group Stage 2, 3에서 외국인 선수들을 모두 물리치고 16강에 진출, 16강에서 Welmu, 8강에서 Bunny를 이기고 4강까지

실패했다. 그래도 점차 2014 GSL Season 1 삽라인의 충격에서 벗어나고 있는 중으로 보인다.

### 1.4. [2014 WCS Korea Season 3·HOT6 GSL Season3](2014%20WCS%20Korea%20Season%203%C2%B7HOT6%20GSL%20Season%203.md) ¶

7월 23일, Code A 48강 A조에서 [남기웅](%EB%82%A8%EA%B8%B0%EC%9B%85.md)과
[홍덕](%ED%99%8D%EB%8D%95.md)을 연이어 2:0으로 잡아내고 조1위로 Code S에 복귀했다. 홍덕과의 승자전
2세트에서 경이로운 전투 장면을 보여줬다.

  

코드 S에서도 김명식에게 졌지만 정경두를 패자전에서 잡은 후 다시 김명식에게 설욕에 성공하면서 16강에 일단 진출은 했는데, 경기 내용이 썩
좋지 않은듯한 인상을 남겨 개선해야 할 것이 있음을 파악할 수 있었다. 특히 패자전을 기점으로 경기 질적 차이가 심하다 해서 스갤에서는
이날의 오본과 오막을 다 이신형에게 주기도 했다. 하지만 이 이후 벌어진 온라인대회에서 스누트를 포함한 연속 3연전에서 5승 2패(세트
득실, 참고로 플토 전에서 1:2로 패함.)를 거두고 조 1위 진출을 하면서 일단 저그전 만큼은 그래도 체면치레는 했다.

  

9월 3일, 코드 S 16강 C조에서 [김대엽](%EA%B9%80%EB%8C%80%EC%97%BD.md)을 2:1,
[원이삭](%EC%9B%90%EC%9D%B4%EC%82%AD.md)을 2:1로 잡아내며 오랜만에 8강에 진출했다. 특히 원이삭과의
승자전 2세트를 정말 눈물겹게 이겨냈다.

  

8강에서 [박수호](%EB%B0%95%EC%88%98%ED%98%B8.md)를 3:0으로 격파하면서 오랜만에 다시 4강을 밟았다.
무소속임에도 판짜기, 컨트롤, 물량, 운영 모두 부족함 없는 모습을 보여주면서 명경기를 만들었다.

  

4강에서 [김도욱](%EA%B9%80%EB%8F%84%EC%9A%B1.md)을 접전 끝에 4:3으로 승리하며 [2013 WCSKorea Season 1, MANGOSIXGSL](2013%20WCS%20Korea%20Season%201%2C%20MANGOSIX%20GSL.md) 이후, **483일만 결승
진출**에 성공했다. 그와 더불어 팀의 2연속 팀킬 결승을 기록하게 됐다.

  

그리고 [어윤수](%EC%96%B4%EC%9C%A4%EC%88%98.md)와 맞붙은 결승에서 4:2로 승리하면서 **868일**만에
[정종현](%EC%A0%95%EC%A2%85%ED%98%84.md) 이후의 GSL 테란 우승자가 되었다. 어윤수 4연준의 여파인지
우승자임에도 현장의 관심은 오히려 준우승자에게 쏠렸지만...

  

이 대회 우승으로 WCS 포인트 2000점을 얻어 2014 WCS 파이널 진출이 매우 유력해졌고, 이후 WCS 유럽 8강 결과 이신형보다
WCS 포인트가 높은 선수 2명과 우승해도 이신형보다 WCS 포인트가 높을 수 없는 선수가 진출하면서 파이널 진출이 확정됐다.

### 2.1. Acer TeamStory Cup Season 3 ¶

디펜딩챔피언으로써 출격, 예선을 통과한 [EG](EG.md)팀을 상대로 **선봉올킬**을 기록했다. 이 과정에서 이제동에게 2승을
거두었다. 이로써, 이신형은 이미 기록된 프로리그 선봉올킬, GSTL 선봉올킬에 이어 이 팀리그에서도 선봉올킬을 기록하는 진귀한 기록을
가지게 되었다.

  

결과적으로 눈에 띄는 활약을 했고, 해당 풀리그에서 28승 4패, 올킬 3회`[1]`라는 기록을 보이며 온라인무대로도 강력한 모습을
보여주었다.

  * `[1]` 해당 리그에서 올킬을 다수 한 선수는 2회의 이제동과 그 뿐.
