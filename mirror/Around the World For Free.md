## 1. 개요 ¶

미국의 어메이징 레이스 2시즌의 우승자였던 알렉스 보일런(Alex Boylan)이 카메라맨 졸트 루카(Zsolt Luka), 버튼
로버츠(Burton Roberts)와 함께 웹사이트의 도움을 제외하고 **완전 무일푼**으로 세계일주를 하는 리얼리티 물로 [내셔널 지오그래픽](%EB%82%B4%EC%85%94%EB%84%90%20%EC%A7%80%EC%98%A4%EA%B7%B8%EB%9E%98%ED%94%BD.md) 코리아에서는 "공짜로 세계여행"이라는 제목으로 방영되었다.  
현재 총 3시즌 45화가 방영되었지만 가장 인기있는 것은 아무래도 1시즌. 그리고 더 이상의 연장 계획은 없는듯하다.

### 2.1. 1시즌 ¶

2007년 9월 18일, 알렉스 보일런이 미국 뉴욕 CBS 아침 방송에서 출발하는 것을 시작으로 미국 → 푸에르토리코 → 미국령 버진 제도
→ 영국령 버진 제도 → 도미니카 공화국 → 트리니다드 토바고 → 베네주엘라 → 콜롬비아 → 에쿠아도르 → 페루 → 칠레 → 남아프리카
공화국 → 탄자니아 → 케냐 → 인도 `[1]` → 캄보디아 → 베트남을 거쳐 2008년 2월 23일 다시 미국에 돌아옴으로서 4개 대륙의
16개국, 159일, 70,000Km를 여행했다.

  

이 여행은 155개 웹사이트에 연재되었으며, 250여 시간의 여행기는 2009년 4월, WGW에 재편집되어 11부작으로 발표되었다.

  

웹사이트를 통하든 아니든 무일푼으로 세계를 여행하던 알렉스를 돕는 각 국의 여러 사람들의 이야기와 살아가는 모습들도 감동적이지만 미국에
돌아와 힐사이드 초등학교라는 곳에서 알렉스와 졸트가 아이들 앞에서 자신들의 여행기를 강연하는 모습은 멋지다. <del>이게 바로 진짜
현장교육이지.</del>

### 2.2. 2시즌 ¶

버튼 로버츠와 알렉스 보일런이 프로듀서로 빠지고 어메이징 레이스에서 동료였던 제프 슈로더(Jeff Schroeder)가 졸트 루카와 함께
움직였다.

  

방문한 국가는 미국을 시작으로 → 일본 → 한국 → 캄보디아 → 라오스 → 태국 → 인도 → 루마니아 → 이탈리아 → 독일 → 네덜란드 →
벨기에 → 영국의 13개국에 여행 기간은 100일도 안된다.

### 2.3. 3시즌 ¶

리얼리티 쇼 서바이버의 MC였던 파르바티 스와로(Parvati Shallow)가 참가했다.

  

2011년 7월 7일에 시작해 10월 14일에 끝났으며, 방문한 국가는 미국을 시작으로 → 도미니카 공화국 → 기아나 → 브라질 →
아르헨티나 → 남아프리카 공화국 → 나미비아 → 보츠와나 → 잠비아 → 케냐의 12개국.
