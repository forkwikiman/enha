※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 받아라 비차 : 근접해서 ← or → + B  

★ 특수기  

  * 멈추라구! : 상대 다운시 ↘ + B (모으기 가능)  

★ 필살기  

  * 참새 찌르기(●) : →←→ + A or B
  * 거비차오소리 "아홉수 결판" : A 연타  
┗ 오소리 훈도시 벗기기 : 아홉수 결판 중 ↓↘→ + B

  * 거비차오소리 "세수 결판" : 무기를 놓쳤을 때 A 연타
  * 필지 : →↘↓↙← + B (모으기 가능)  
┗ 필지~ 결판! : 필지를 끝까지 모은다

  * 고비차 : →↘↓↙← + C
  * 장기 쓰러뜨리기 : 근접해서 ↘↘ + C
  * 계마 높이 올리기 : →↓↘ + C  

★ 초오의  

  * 반상차의 한수 "토킨" : →↘↓↙←→↘↓↙← + AB   

★ 잠재오의  

  * 반상차의 한수 "각행" : →↘↓↙←→↘↓↙← + A (모으기 가능) 
  * 반상차의 한수 "비차" : →↘↓↙←→↘↓↙← + B (모으기 가능)

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 받아라 비차 : 근접해서 ← or → + CD  

★ 특수기  

  * 멈추라구! : 상대 다운시 ↘ + B (모으기 가능)
  * 초도발 : ↓↘→ + 스타트 버튼  

★ 필살기  

  * 참새 찌르기(●) : →←→ + A or B
  * 거비차오소리 "아홉수 결판" : A 연타  
┗ 오소리 훈도시 벗기기 : 아홉수 결판 중 ↓↘→ + B

  * 필지 : →↘↓↙← + B (모으기 가능)  
┣ 필지~ 결판! : 필지를 끝까지 모은다  
┗ 필지~ 속이기 : 필지 중 D

  * 고비차 : →↘↓↙← + C
  * 장기 쓰러뜨리기 : 근접해서 ↘↘ + C
  * 계마 높이 올리기 : →↓↘ + C
  * 풍차 : ↓↙← + A  

★ 초오의  

  * 반상차의 한수 "토킨" : →↘↓↙←→↘↓↙← + AB   

★ 잠재오의  

  * 반상차의 한수 "각행" : →↘↓↙←→↘↓↙← + A (모으기 가능)
  * 반상차의 한수 "비차" : →↘↓↙←→↘↓↙← + B (모으기 가능)  

★ 난무오의  

  * 연금 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C,↓↘→ + B  
┣ 띄우기 루트 : 히트 후 A, B, C,↓ + C, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C,↓ + C, BC, ↓↘→ + B
