## 1. [세이버](%EC%84%B8%EC%9D%B4%EB%B2%84%284%EC%B0%A8%2C%205%EC%B0%A8%29.md)
¶

★ 필살기  

  * 퍼스트 에어 : ↓↘→ + a/b/c  
┗세컨트 에어 홀드 : ↓↘→ + a  
┗세컨드 에어 디피트 : ↓↘→ + b  
┗세컨드 에어 브레이크 : ↓↘→ + c  
　┗서드 에어 : ↓↘→ + a/b/c

  * 윙 에어 : →↓↘ + a/b/c  
┗윙 에어 - 쓰러스트 : (b/c 윙 에어 후) 추가로 + c

  * 엘핀 댄스 : ↓↙← + a/b/c  

★ 특수능력  

  * 풍왕결계 : 그냥 마력개방  
엑스칼리버가 모습을 드러내고, 초필살기 풍왕철퇴가 약속된 승리의 검으로 교체된다. 300% 마력개방 시, 상대가 공격을 받았을 때의 넉백
상태 시간이 통상시보다 길어진다. 참고로, 300% 마력개방을 하면 배경음악이 세이버 전용 BGM으로 변경된다.  

★ 초필살기  

  * 풍왕철퇴 : ↓↘→↓↘→ + a  

★ 마력개방 초필살기  

  * 약속된 승리의 검 : ↓↘→↓↘→ + a
  * 모든 게 머나먼 이상향 : ↓↙←↓↙← + c  

★ 성배 초필살기  

  * 약속된 승리의 검 : ↓↘→↓↘→ + c

## 3. [아처](%EC%95%84%EC%B2%98%285%EC%B0%A8%29.md) ¶

★ 필살기  

  * 투영 : ↓↓ + c (공중가능)
  * 학익이련(鶴翼二連) : (검 장비) ↓↘→ + a/b/c  
┗학익이련 인(鶴翼二連・引) : 추가로 + a

  * 학익공구 습(鶴翼攻究・襲) : (검 장비) ↓↙← + a/b/c
  * 학익수구 체(鶴翼守究・薙) : (검 장비) →↓↘ + a
  * 학익수구 승(鶴翼守究・昇) : (검 장비) →↓↘ + b
  * 학익수구 돌(鶴翼守究・突) : (검 장비) →↓↘ + c
  * 링크 샷 : (활 장비) ↓↘→ + a/b/c (공중가능)
  * 트랩 샷 : (활 장비) ↓↙← + a/b/c
  * 블레이드 댄스 : (무한의 검제 발동 중) ↓↓ + c  

★ 특수능력  

  * 무한의 검제 : 영창 6번 후 300% 마력개방  
영창 횟수는 라운드에 상관없이 축척된다. 다만, 무한의 검제를 한번 발동하면 횟수는 리셋된다. 배경이 고유결계로 변화하며, 통상적인 성배개방
효과 외에도, 필살기 "블레이드 댄스"와 초필살기 "무한의 검무"를 쓸 수 있다. 성배개방처럼 제한 시간이 존재. 무한의 검제 발동 중에는
검만 장비할 수 있고, 활은 쓸 수 없다(활 장비 시에 발동하면 강제적으로 검 장비로 바뀜).  

★ 초필살기  

  * 학익삼련 : (검 장비) ↓↘→↓↘→ + a
  * 적원엽견 : (활 장비) ↓↘→↓↘→ + a
  * 거짓 나선검 : (활 장비) 공중에서 ↓↘→↓↘→ + a
  * 무한의 검제 영창 : ↓↘→↓↘→ + b  

★ 마력개방 초필살기  

  * 치천을 뒤덮는 일곱 원환 : 상대의 원거리계 초필살기에 맞춰 abc
  * 무한의 검무(언리미티드 블레이드 댄스) : (무한의 검제 발동 중) ↓↘→↓↘→ + c
