# [아랑 MOW](%EC%95%84%EB%9E%91%20MOW.md) ¶

★잡기  

  * 극한 박치기 : 근접해서 → or ← + C   
  

★특수기  

  * 극한 팔꿈치 : → + A   
  

★콤비네이션 어택  

  * 점프 B - 점프 A or 점프 C   
  

★ TOP기  

  * 잠열권 : CD   
  

★필살기  

  * 호황권 : ↓↘→ + A or C 
  * 호포 : →↓↘ + A or C (브레이킹 가능)
  * 비연 질풍각 : ↓↘→ + B or D 
  * 용섬권 : C 눌렀다 뗌 
  * 호섬권 : D 눌렀다 뗌 
  * 상단 후리기 : ↓↙← + A 
  * 하단 후리기 : ↓↙← + B   
  

★초필살기  

  * 패왕상후권 : ↓↘→↓↘→ + A 
  * 극한 호포 : ↓↘→↓↘→ + B   
  

★잠재 능력  

  * 패왕상후권 : ↓↘→↓↘→ + C 
  * 극한 호포 : ↓↘→↓↘→ + D 
  * 용호난무 : ↓↘→↓↘→↓↘→ + C (연타 가능)
