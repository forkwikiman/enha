지구라트(게임)의 레벨업별 퍽을 소개하는 문서다.  
대부분 강화형의 경우 5까지 찍을수 있으며, 한번만 찍을수도 있고, 1회성 효과가 있는것도 있고, 그중에는 무언가를 대가로 얻어야 하는
것까지 많다.

기본적으로는 랜덤 선택이지만, 이미 플레이중 1~2레벨 이상 찍은 퍼크가 있으면 다음 레벨업시 퍼크선택화면의 둘(책벌레 획득시 셋)중 해당
스킬이 뜰 가능성이 높아진다. 그리고 몇몇 퍼크가 어느정도 보유되고 있는 타 직업의 경우 이러한 반강제 몰빵(?)의 가능성이 커진다.

## 1. 체력/방어력 관련 Perk ¶

  * [드루이드](%EB%93%9C%EB%A3%A8%EC%9D%B4%EB%93%9C.md)(Druid) : 체력포션의 회복량을 늘린다.
  * 생명의식(Life Awareness) : 체력 한계를 늘린다. **유리몸 캐릭터들로 진행시 레벨업 화면에 이게 뜨면 무조건 찍자.**
  * 저주(Curse) : 체력포션의 효과를 2.5배 늘리지만, **피통이 절반이 된다**.
  * 전투함성(War Cry) : 전투가 시작되면 소정의 체력이 회복된다. 찍을수록 그 양은 증가한다. 
  * 백색구(White Orb) : 현재 만피의 절반을 즉시 채운다. 
  * 철구(Wrecking Ball) : 각종 잡동사니를 부숴서 아주 작은 체력을 회복한다. 스킬 레벨 x 1 로 체력이 회복된다. 1정도만 찍으면 조금 부족하지만 2나 3정도를 찍으면 보조 체력회복수단으로도 크게 무리가 없어진다. 멘지로 플레이시 이게 뜨면 무조건 찍는것을 권장한다.
  * 두꺼운 피부(Thick Skin) : 방어력을 증가시킨다. 정확히는 플레이어가 받는 데미지를 감소시킨다. 생존성을 향상시키려먼 반드시 찍어야 하는 스킬.
  * 거북이(Turtle) : 방어력을 비약적으로 증가시키나, **대신 플레이어의 데미지도 줄어든다**는 댓가를 요구한다.
  * 체력 에너지(Vital Energy) : 현재 마나통의 반을 뺏는 댓가로 만피통을 늘린다. 공격보다 생존이 중요하고 마나 수급을 어떻게든 지속할수 있다면 찍어도 크게 나쁠건 없다.
  * 피의 의례(Blood Rites) : 현재 HP의 절반을 깎는 댓가로 만피통을 늘린다. 현재 HP가 빵빵하다면 찍어볼 가치가 있는 스킬.
  * 상처 무시(Ignore wounds) : 적의 공격 일부를 무효화한다. 두꺼운 피부와 함께 찍으면 방어력 증진에 도움이 된다. 
  * 블러드러스트(Bloodlust) : **마나 젬으로도 소정의 체력 회복이 가능해진다**. 멘지를 플레이한다면 반드시 찍어야할 스킬중 하나. 
  * 하이랜더(Highlander) : **체력이 빠르게 재생되게 해주나`[2]`, 피통이 25%로 깎인다.**
  * 마나 실드(Mana Shield) : 적의 공격을 마나를 소모하면서 방어해준다. 마나 수급을 생각하지 않은채로 너무 많이 찍으면 마나 빈곤에 시달리게 되므로 주의.
  * 재생(Regeneration) : 전투 시작시 체력이 채워지나, **체력 포션의 효과가 미미해진다**. <del>멘지 : 어서와 내능력은 처음이지?</del>
  * 부자연스러운 힘(Unnatural Strength) : 피통과 피를 100 늘리는 댓가로 이동속도가 느려진다.

## 2. 마나/무기 관련 Perk ¶

  * 격분(Enraged) : 마나를 즉시 모두 채운다. 다만 1회성이고 마나가 어느정도 쌓이는 중반 이후면 굳이 이걸 찍을 필요는 없다.(마나가 빈곤한 직업이나 상황을 자주 직면한다면 모를까)
  * 기억상실증(Amnesia) : 마나량의 슬롯을 무작위로 바꾸며 마나량이 많은 슬롯이 있을경우 그 슬롯이 무작위로 바뀐다. 마나량이 균일할때에는 효과가 없는듯.
  * 연금술재료 주머니(Alchemic pouch) : 연금술 마나의 용량을 늘린다. 
  * 스펠 수집가(Spell Collector) : 스펠통을 늘린다.
  * 스탭 스캐빈저(Staff Scavanger) : 스탭통을 늘린다.
  * 자경단원(Vigilant) : 전투가 시작되면 소정의 마나가 회복된다. 찍을수록 그 양은 증가한다. 
  * 워록(Warlock) : 마나젬으로 회복할수있는 마나의 양이 늘어난다.
  * 정화(Cleansing) : 데미지를 입을때까지 마나가 무한이 된다. 석상에 빌수 있는 버프효과중 이와 동일한 효과가 있으며 되도록이면 석상에 빌어서 이 능력을 얻는게 그나마 레벨업보다는 원하는 상황에 맞을 수도 있으니 
  * 뒤섞음(Shuffle) : **모든 무기를 버리는대신** 새로운 무기를 얻는다.
  * 장사꾼(Trader) : 마나젬이 해당 색깔뿐 아닌 **다른 색깔의 마나도 채운다**. 
  * 연금술 숙달(Alchemy Proficiency) : 연금술 무기를 쓸때 마나 소모량이 줄어든다. 
  * 대체(Alternative) : 2차 발사시 요구하는 마나량이 줄어든다. 
  * 균형(Equalizer) : 마나 크리스탈로 다른 색깔의 마나를 회복할수 있지만, 그 양이 비약적으로 줄어든다. 장사꾼의 하위 호환.
  * 이단(Heretic) : 갑자기 새로운 스탭이 생긴다. 
  * 내부 지혜(Inner Wisdom) : 갑자기 새로운 스펠이 생긴다.
  * 마나 버스트(Mana Burst) : 데미지를 2배 늘리지만, **마나 소모량이 3배로 늘어난다.**
  * 마나 스틸 : **지식젬으로도 소량의 마나 충전이 가능해지며 이는 모든 마나를 골고루 올려준다.**
  * 손재주(Sleight of Hand) : 말그대로 무기 바꾸는 속도를 빠르게 해준다.
  * 스펠 숙달(Spell Proficiency) : 스펠을 쓸때 마나 소모량이 줄어든다. 
  * 스탭 숙달(Staff Proficiency) : 스탭무기를 발사할때 마나 소모량이 줄어든다. 
  * 재능있는 연금술사(Talented Alchemist) : 연금마나가 아주 천천히 자동으로 차오른다.
  * 재능있는 주문영창자(Talented Spellcaster) : 스펠마나가 아주 천천히 자동으로 차오른다.
  * 재능있는 지팡이꾼(Talented Staff Wielder) : 스탭마나가 아주 천천히 자동으로 차오른다.
  * 변이(Transmutation) : 갑자기 새로운 연금술무기를 획득한다.
  * 완드 링크(Wand Link) : 완드마나 충전속도를 늘린다.
  * 연금술 친밀(Alchemy Affinity) : 연금무기의 데미지를 늘리는 대신 다른 무기의 데미지를 깎아버린다. 이외에도 ~친밀 계열 무기 강화 퍼크는 같은 효과니 특정 무기에 올인하고자 할때만 찍자. 
  * 연금술 마스터리(Alchemy Mastery) : 연금술 무기의 연사속도를 올린다.
  * 마법 구속(Magic Binding) : 현재 장비중인 무기 하나가 기본완드 슬롯으로 강제 이동되며, **기본완드 마나로 해당 무기를 쓸수 있다.** 그러나 데미지도 줄어든다.
  * 희생(Sacrifice) : 맞을때마다 소정의 마나가 회복된다. 
  * 전문가(Specialist) : 랜덤한 종류의 마나통을 세배로 올리지만, 나머지 두 마나통이 반쪽난다.
  * 스펠 친밀(Spell Affinity) : 스펠무기의 데미지를 늘리는 대신 나머지 무기의 데미지를 깎는다. 
  * 스펠 마스터리(Spell Mastery) : 스펠 연사속도를 늘린다.
  * 스탭 친밀(Staff Affinity) : 스탭무기의 데미지를 늘리는 대신 나머지 무기의 데미지를 깎는다.
  * 스탭 마스터리(Staff mastery) : 스탭 연사속도를 늘린다. 
  * 완드 마스터리(Wand Mastery) : 완드의 연사속도를 늘린다. 
  * 완드 쇼크(Wand Shock) : 완드공격에 일정확률로 스턴속성을 부여한다.

## 3. 속도 관련 Perk ¶

  * 순 겁쟁이(Scaredy-Cat) : 플레이어가 얻어맞을 경우 일시적으로 속도가 증가한다. 
  * 빠름(Haste) : 기본적인 이동속도가 빨라진다. 특히 알고렙으로 플레이할시 레벨업화면에 뜨기가 절실해지는 스킬이다.
  * 전력 질주(Sprinter) : 전력질주 속도가 증가된다. 이속 못지않을 정도로 유용한 이속버프다.

## 4. 반격 관련 Perk ¶

  * 매직 미사일(Magic Missile) : 지식젬을 먹을때마다 일정확률로 적을 얼리는 유도 고리가 사출된다. 
  * 매직 펄스(Magic Pulse) : 마나 크리스탈을 먹을때마다 일정확률로 마법탄이 사출된다.
  * 복수(Revenge) : 맞을때 플레이어의 일정범위로 충격파를 방출한다. 기본적으로 적을 스턴상태에 빠지게 하므로 근접전투하는 적을 상대할때 유용하다.
  * 독 오라(Poison Aura) : 적이 접근하면 독오라를 방출한다.

## 5. 아이템 관련 Perk ¶

  * 지속(Persistance) : 적이 떨구는 모든 **아이템의 놓여져있는 시간이 길어진다.** 한번만 찍어도 꽤 유용하다. 
  * 스캐빈저(Scavanger) : 말 그대로 적의 드랍률이 높아진다. 
  * 염동력(Telekinetic) : 조금 떨어진 아이템을 획득가능하다. 찍을수록 취득가능한 범위가 늘어난다.
  * 치료사(Healer) : 마나젬보다 체력포션의 드랍확률을 높여준다.
  * 과충전(Overload) : 치료사의 반대효과로 체력포션보다 마나젬의 드랍확률을 높여준다.
  * 마나 포커스(Mana Focus) : 마나젬 획득시 획득가능한 마나의 양을 늘려주되, 체력포션으로 얻을수있는 체력의 양을 깎는다.
  * 휘발성(Volatile) : 적의 드랍량이 늘어나나 아이템이 더 빨리 사라진다.`[3]` 아이템 회수를 감당하기 충분한 상황이라면 찍어도 무방하지만...
  * 탐욕(Avarice) : **먼 거리의 아이템을 끌어모을수 있으나 적의 드랍률이 눈에 띄게 줄어든다.**
  * 브루트(Brute) : 적이 다른 아이템을 지식 젬보다 더 많이 떨어트린다.
  * 사기꾼(Trickster) : 상자를 깔때 아이템이 나올 확율이 높아지며 마나, 경험치, 스코어성배의 양도 증가된다.

## 7. 레벨업, 경험치 관련 Perk ¶

  * 학자(Scholar) : 지식젬으로 얻을수 있는 경험치가 증가한다.
  * 책벌레(Bookworm) : 레벨업시 퍼크 선택권을 **3장으로 늘린다**.
  * 박식다학(Erudite) : **스크롤을 읽자마자 갑자기 레벨업한다**. 원래 스크롤의 경험치는 1레벨->2레벨 요구 경험치의 반인데 이것을 찍을경우 현재 레벨 경험치 바의 전체에 해당하는 경험치를 획득 가능하다. 1판에서 찍으면 어느정도 효과를 볼수 있다. <del>하지만 5판까지 스크롤방이 별로 안나오면 [망했어요](%EB%A7%9D%ED%96%88%EC%96%B4%EC%9A%94.md)</del>
  * 민감함(Sensitivity) : 지식젬으로 얻을수 있는 **경험치의 양이 두배가 되나, 그 댓가로 플레이어가 받는 데미지는 2배로 늘어난다**. 컨트롤에 자신이 있거나 방어관련 퍼크가 충분하면 찍어도 무방하지만, 특히 체력이 낮은 캐릭터들로 한다면 글쎄...
  * 사채업자(Loan Shark) : 점수 성배로 **점수가 아니라 경험치를 획득한다**. 점수 따위는 아무래도 좋고 클리어가 목표라면 찍어도 무방하다.
  * 재능있음(Talented) : 갑자기 2단계의 레벨업이 이루어지나, 이후 경험치가 절반으로 깎인다. 막판에 찍자.

## 8. 기타 Perk ¶

  * 광전사(Berserker) : 현재 체력이 25% 미만일때 데미지가 늘어난다.
  * 큰 판돈(High Stakes) : 죽음의 사신과 체력을 두고 내기를 한다. 만약 사신에게 이길경우 체력과 마나가 모두 회복되지만, **질 경우 피통이 3/4가 된다**. 어차피 이겨도 1회성이기 때문에 굳이 찍을 가치는 없다.
  * 흡수(Consumption) : 최대 체력의 20%을 깎는 댓가로 2개의 퍽을 선택할수 있다.
  * 호기심(Curiosity) : 보물방을 인카운터할 확률이 높아진다.
  * 죽음의 약속(Death Pact) : 설명을 보면 치명적인 일격으로부터 당신을 구한다고 되어있다. 한번 체력이 다하면 부활로 추정되는데 이 퍽의 효과를 보신 분은 [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md). 
  * 견딤(Endure) : 맞는 순간 **아주 짧은 시간동안 무적이 된다**. 별로 효과가 없을것 같지만 연속으로 공격을 맞는경우 까이는 HP를 어느정도 줄일수 있는 효과를 기대할수 있다. 
  * 명상 : 사용즉시 체력과 마나의 25%를 회복한다. 별로 찍을 필요는 없다.
  * 운명의 거울 : 맵상의 중요장소를 마킹해준다. 이리저리 레벨업을 열심히 하려면 던전 크롤링은 필수이므로 찍을 필요가 없어진다. 하지만 코부스로 플레이하고 있다면? 최속 진행을 위한 의외의 가치가 있을지도 모른다.
  * 신탁(Oracle) : 석상에게 소원을 빌때, **절대로 징벌을 받지 않는다**. 석상에게 소원을 빌때의 부담감을 덜어주고 있다. 
  * 놀랐지?(Surprise) : 퍽을 랜덤으로 얻을수 있다.
  * 검은 구체(Black Orb) : **HP의 50%를 깎는 댓가로 연사력이 증가한다.**
  * 껍질(Husk) : 지식 젬을 얻을때마다 **아주 잠깐 무적이 된다**. 십자포화를 뚫고 아이템을 회수하고자 할때 매우 유용하다. 견딤 퍽과 조합하면 효과가 더 좋다. 
  * 분노(Fury) : 플레이어가 맞을때까지 2배의 데미지를 입힐수 있다. 
  * 영웅(Hero) : 챔피언급 적이 나올 확률을 증가시키지만, 대신 챔피언을 격퇴시 드랍되는 아이템이 많아진다.
  * 경량(Lightweight) : 만피를 깎는 댓가로 이속을 늘린다. 다만 그 만피를 깎은 퍼센테이지가 처절하므로 생각해보고 찍을것.`[4]`
  * 말살자(Obliterator) : 엠통을 깎는 댓가로 연사속도를 늘려준다.
  * 안전한 길(Safe Path) : 부비트랩 통로를 마주칠 확률을 줄여준다. 다른 직업에 컨트롤이 된다거나 어지간한 부비트랩 패턴을 파악했으면 찍지 않아도 되지만 발이 느린 알고랩으로 플레이한다면 의외로 절실할지도 모른다(...)
  * 싸움꾼(Brawler) : 적의 숫자를 늘리는 대신 드랍률도 늘린다.
  * 어르신(Elder) : **체력과 속도, 데미지를 깎는 대신에 연사속도와 마나재생이 부쩍 늘어난다.**
  * 기술의 제왕(Master of the Arts) : **현재 찍은 모든 퍼크의 레벨을 한단계 올려준다**.
  * 지점(Solstice) : **체력을 2/3 깎는 댓가로 모든 전투의 길이를 20%로 줄인다**. <del>보스전은?</del>
  * 고문(Torture) : 적의 상태이상 지속시간을 늘린다. 
  * 무한한 힘(Unlimited Power) : <del>[팰퍼틴](%ED%8C%B0%ED%8D%BC%ED%8B%B4.md)황제</del> **다음 전투에서 30초간 무적이 된다**. 체력이 낮은 상태에서 보스전에 임할 상황이거나 최종보스를 앞두고 레벨업했을때 이게 걸리면 반드시 찍자.
