## 1. 설명 ¶

장점다운 장점이 척보기에는 없는 캐릭터.

  

기술도 대공기 커맨드로 대공기가 아닌 기술 하나뿐이지만 이 캐릭터의 장점은 일격 필살에 가까운 궁극기에 있다. 두개의 궁극기중 하나는
난무형. 나머지 하나는 서포트 캐릭터와 합체하는 광역 섬멸기(…). 서포트 캐릭터와의 합체는 서포트 캐릭터에 따라 외형과 성능이 전부
다르다.

  

서포트 캐릭터와의 합체는 서포트 캐릭터의 기 게이지도 5까지 전부 찰 필요가 있으며, 이 캐릭터는 그걸 위해서인지는 몰라도 서포트 캐릭터의
기 게이지 차오르는 속도가 전 캐릭터중에서도 가장 빠르며 압도적으로 빠르다. 그걸 이용해서 서포트 캐릭터의 기술을 활용하는게 포인트. 또한
체인콤보와 특수기가 가장 풍부한 캐릭터이니 결과적으로 기본기와 서포트로 싸우게 된다.

  

콤보가 상당히 현란하긴 한데 구석이 아니면 맞추기 힘든 콤보들 뿐이고 그 전에 몰아붙이기 자체가 아주 어렵다. 에코가 있다면 콤보가
폭발하겠지만 아무래도 견제가 좀.. 게다가 체인아츠 중에서 6B 로 끝나는건 쿠루미에게 전혀 이득이 오지 않는 공격이며 그나마 다운시키는
2C로 끝내야 한다. 2C가 슬라이딩이라서 기습적으로 쓰긴 좋지만.. 쿠루미는 콤보 자체 데미지도 별로고 구석까지 몰아 붙일 힘이 없다.
차라리 시에라로 견제성능을 강화하는게 낫다. 기모아서 유니온 아츠.. 구석 에코 콤보는 사실상 불가능하다고 봐야 한다. 여러 모로 부족한
캐릭터로 보일수도 있지만 앉아중의 사기적인 발동속도와(약 발동수준) 내려차기와의 이지선다 및 서포터와의 합체기가 굉장히 강력하기 때문에
다루기에 따라 충분히 강캐급의 성능을 낼 수 있다.

  

구석까지 몰아넣기가 성공했을때 내려차기가 B 이후 캔슬이 가능해서 히트백이 없어진다는 것에 착안하여 무한콤보가 가능하다(..)

  

등장신이 [타오카카](%ED%83%80%EC%98%A4%EC%B9%B4%EC%B9%B4.md) 닮았다.(앞에서 불쑥 튀어나왔다가
상대쪽으로 돌아간다.)

  

그리고 승리포즈중 하나가 [춘리](%EC%B6%98%EB%A6%AC.md)의 승리포즈중 하나를 흉내냈다.(스트리트 파이터2의 춘리.)

  

버전 1.01부터는 서서 B 범위 증가, 기력 게이지 길이 감소, 버그기 삭제, 내려차기 판정 범위가 낮아지고 낙법 가능, 초규멸기난무
연속기 불가 (대신 공격력 상승, 공중 전탄히트 가능), 앉아 C에서 서포트 소환 가능 등 꽤 많이 변경되었다. 1.02부터는 루프 콤보도
안되고 내려차기 공중 추가타가 불가능해져 문제가 많아졌는데... 다음 버전에 뭔가 추가할 예정이라고 한다.

  

참고로 이 게임에서 **최연소 캐릭터다** 나머지 1명은 [하스미에리](%ED%95%98%EC%8A%A4%EB%AF%B8%20%EC%97%90%EB%A6%AC.md). 둘 다 **14살** 발육이

## 4. 기술 ¶

  * 내려차기 (→↓↘+A/B/C)  
약간 준비 자세를 취한 다음에 크게 돌려서 내려찍는다. 커맨드가 대공기지만 웬만해서는 대공기로 사용 불가. 더군다나 이 계열이 잘못하면
삑샬나기 쉬운 커맨드중 하나라 실수하면 통상기만 나간다. B 이후 캔슬 가능이지만 속도가 느려서 콤보로는 불가능. 적중시 상대가 떠오르는데
추가타 가능. 사용중에 서포터 사용 가능.  

  * 앞구르기 (↓↙←+C)  

  * 급강하 (공중에서 ↓↓)  

  * 섬광마술사 (에코 서포터로 ↓↓+B)  
1.04에서 추가된 서포터 전용 기술. 필살기지만 서포터 기술로 취급되어 서포터 게이지를 0.5정도 소비한다. 에코가 쿠루미를 슬쩍
엉덩이치기로 넘어트려서 [OTL](OTL.md)자세로 만든 후 등을 밟고 점프 무릎치기로 가격. 그야말로 [샤이닝위저드](%EC%83%A4%EC%9D%B4%EB%8B%9D%20%EC%9C%84%EC%A0%80%EB%93%9C.md). 쿠루미가
넘어질때도 공격판정이 있으며 만약 쿠루미의 OTL공격(?)에 상대가 히트하게 되면 강제로 앉은 자세가 되고 이후 에코의 샤이닝 위저드에 맞고
중급 카운터(엉덩방아)가 되어서 추가타를 넣을 수 있다.둘이 호흡(?)을 맞춰야 하는지라 에코가 근처에 없으면 발동하지 않는다. 참고로 기술
발동시 에코가 샤이닝 위저드를 날리는 순간 **팬티가 나온다**.  

  * 내동댕이 (줄리엣 서포터로 ↓↓+B)  
1.04에서 추가된 서포터 전용 기술. 필살기지만 서포터 기술로 취급되어 서포터 게이지를 2정도 소비한다. 쿠루미를 내던져서 공격하는 주종이
뒤바뀐 기술. 던지는 동안의 쿠루미에게도 맞는 판정이 있어서 판정이 강한 기술과 붙으면 보통 진다.  

  * 시에라 트래픽 (시에라 서포터로 ↓↓+B)  
1.04에서 추가된 서포터 전용 기술. 필살기지만 서포터 기술로 취급되어 서포터 게이지를 2정도 소비한다. 공중으로 쿠루미를 들어올려서
자유낙하시킨다. 착지하면서 하단 슬라이딩(↓+C)으로 공격한다. 낙하 도중에 기술을 캔슬하고 점프 공격을 쓸 수도 있다. 비행 중엔 무적.  

  * 서포터 가챠 (카나에 서포터로 ↓↙←+B)  
1.04에서 추가된 서포터 전용 기술. 필살기지만 서포터 기술로 취급되어 서포터 게이지를 2정도 소비한다. 데미지가 없는 커맨드 잡기로,
상대를 잡은 후 상대의 서포터를 UFO캐치로 잡아간다. 상대방의 서포터는 강제적으로 서포터 카운터 상태가 된다. 서포터는 일정 시간 후에
돌아온다. 잡혀있는 동안 D버튼을 연타해서 보다 더 빨리 돌아오게 만들 수 있다. 커맨드 잡기지만 풀기가 가능하다. 쿠루미의 뇌쇄적인 허리
흔들기라든가, 서포터 캐릭터가 잡혔을 때만 볼 수 있는 특수 모션과 팬티 덕분에 여러모로 심금을 울리는 기술.

## 5. 리버티 아츠 ¶

  * 초규멸기난무 (↓↙←+BC)  
초근접의 타격기. 첫 타격이 성공하면 난무에 들어간다. 데미지도 별로 안좋고 맞추기도 힘들고 그나마 무적시간이 좀 있어서 리버설 쪽으로 활용
가능. 판정이 좋아서 대공기로도 쓸수있다.  

  * 유니온 아츠 (↓↘→+BC)  
서포트 캐릭터와 **알몸으로** 합체하여(스고이~라고 교성을 낸다(...)) 광역공격을 가한다. 해머를 든 서포트 캐릭터와의 합체는 범위가
짧다. 드럽게 빠른 시전속도에 무적판정도 장난아니고 데미지도 상당한 기술. 물론 서포터의 기 게이지가 5개나 필요하다는 단점이 있지만
진짜진짜 좋은 기술. [서포트 캐릭터별 복장](http://bbs1.ruliweb.daum.net/gaia/do/ruliweb/default
/pc/41/read?itemId=78501&bbsId=G001&articleId=3544166)  
  
일단 거의 즉시발동급이라 상대가 방어를 할수없는 확실한 상황일때쓰면 확실히 먹일수있다. 단점이라면 데미지가 좀 애매해서 일발역전기라 부르기
힘들고 가드시에는 **가드데미지가 전혀 나오질않는다**.

## 6. 콤보 ¶

  * 점프 C -> 리버티 아츠  

  * 내려차기 -> C  

  * 내려차기 -> 에코 ↓D -> 내려차기 -> C  
내려차기 한번 더 넣는 콤. 필드에서 쓸수있는 가장 강한 콤보 일수도..  

  * (상대구석) -> 공중 C -> A -> C -> 에코 ↓D -> 공중 B -> 서서 C  
하루카, 루나 전용 콤보. 키가 커야 다 맞는다. 공중 B를 좀 낮게 맞춰주는게 포인트.

  * (상대 구석) 공중 C -> A -> C -> 에코 ↓D -> 내려차기 -> 약간 앞으로 걸어가서 C -> 에코 ↓D -> 공중 B -> 서서C  
그 외 애들 스턴 콤. 사실 하루카, 루나한테도 맞출수 있지만 에코가 2히트가 되어버려서 타이밍이 조금 어긋난다.  

  * (상대 구석) 공중 C -> A -> C -> 에코 ↓D -> 내려차기 -> 약간 앞으로 걸어가서 C -> 에코 ↓D -> 공중 B -> 서서C -> 에코 ↓D -> 공중 B -> 서서C -> 에코 ↓D -> 공중 B -> 서서 C -> 에코 ↓D  
  
스턴 상태의 적 마무리 콤보. 하루카나 루나는 스턴이 한번 더 걸린다.
