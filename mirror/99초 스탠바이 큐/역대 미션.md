### 1.1. 1회 - 소방서 편 ¶

  * 1단계 : 우유든 주전자들고 뛰어와서 '출동하려면 힘을 내야 해'란 대사를 한 다음에 우유 따르기
  * 2단계 : 우유를 마시고 휘파람불기
  * 3단계 : '출동이야 출동' 대사한 다음에 출동봉 3번 두드리기
  * 4단계 : 출동봉 타고 내려와서 엉덩이로 풍선 터뜨리기
  * 5단계 : 민원실 안내 한숨 세번 쉬고 사투리 대사 (민원실이 어디라예?/예,바로 여기입니다/고맙습니다예~(윙크))
  * 6단계 : 시뮬레이터 화재 진화 후 '불 껐다, 신난다, 야!'라는 대사를 외친 다음에 출동
  * 7단계 : 소방복입고 소방모를 한바퀴 회전한 후 받기
  * 8단계 : 12미터 상공의 모형소화기 받기
  * 9단계 : 소방차로 달려가 경광등 싸이렌 켜기

### 1.2. 2회 - 전통음식 편 ¶

  * 1단계 : 물동이 들고 뛰어와 독에 물 붓기
  * 2단계 : 독 안에 든 쥐 나와 거지 얼굴에 물 뿌리기
  * 3단계 : 후라이팬의 부침개 차례로 뒤집기
  * 4단계 : 부어준 막걸리(실제로는 두유임)를 마시고 파 씹으며 대사(사또 나리 안주 드셨어요/안주에는 역시 우리 농산물이 최고지, 파!)
  * 5단계 : 대사와 함께 물레방앗간에서 미꾸라지 잡아 올리기(어우동, 뭐하냐?/추어탕을 끊이려고요)
  * 6단계 : 대사 연기후 박바가지 머리로 깨기(대사:한푼만 줍쇼/이 미련곰탱아 썩 꺼져/주모,사랑했구만요!)
  * 7단계 : '풍악을 올려라' 대사 후에 노래와 율동하기
  * 8단계 : 일지매 '정의의 화살을 받아라' 대사 후에 화살 던져 풍선 맞추기
  * 9단계 : 전 출연자 단체 줄넘기 4회 후에 '한국의 음식 세계로 뻗어갑니다' 표어 제창

### 1.3. 3회 - 지하철 편 ¶

  * 1단계 : 업혀서 계단 내려오면서 대사하기(여자 출연자 : 야! 재미있다)
  * 2단계 : 계단 내려온 다음 순서대로 하늘땅 별땅 하기
  * 3단계 : 자판기 음료 100원 짜리 4개 뽑아 단숨에 마시기
  * 4단계 : 휴지통에 음료수 빈 컵 던진 후 율동과 함께 '골인이다 골인' 대사하기
  * 5단계 : 지하철 탑승 후 한 바퀴 구른 후 3~5회 총 쏘는 연기하기
  * 6단계 : '지하철에서는 공중예절을 지키세요' 대사 후에 엉덩이 3회 때리기
  * 7단계 : 붉은색 깃발과 녹색깃발 순서대로 한 바퀴 돌리기
  * 8단계 : 목말타고 붉은 깃발 받아 한 바퀴 돌리기
  * 9단계 : 플래카드 펼치고 '지하철은 약속을 지켜드립니다' 표어 전 출연자 제창하기

### 1.4. 6회 - 워터파크 편 ¶

  * 1단계 : 원형 슬라이더 타고 내려와 물에 빠진 여인을 구한다.
  * 2단계 : 번쩍 들고 내려와서 남자가 '인공호흡을 해야지' 대사하면 여자가 '이게 무슨 짓이에요' 대사후 따귀를 때리기
  * 3단계 : '보물을 찾아주세요' '예 효리 공주님' 대사 후 엉덩이치고 남자가 다이빙하며 열쇠잡기
  * 4단계 : 잠수한 다음 보물함에 있는 미소 목걸이 꺼낸 다음 던져서 받기
  * 5단계 : 오리발 신고 출연자 이름 외치며 줄넘기 4회 하기
  * 6단계 : 벤치에 앉아 대사 연기 후 파인애플 조각 던져 받아 먹기
  * 7단계 : 공중에서 던진 튜브 한 번에 받은 다음 '타~잔'하고 외치기
  * 8단계 : 타잔처럼 외치고 줄타기해서 물위에 떠있는 목표물에 착지
  * 9단계 : 다 같이 '당신의 미소가 한국의 미소입니다' 표어 제창하기

### 1.5. 8회 - 레저스포츠 편 ¶

  * 1단계 : 놀이방에서 나와서 도서관으로 뛰어가기
  * 2단계 : (대사:천사님/왜 호동아?/천사,사랑했구만요/사랑은 아무나 하나) 대사 후에 화살로 책 도미노 실시
  * 3단계 : 레인 위의 볼링핀을 야구공으로 던져 맞추기
  * 4단계 : 날깨란 깨서 컵에 넣어 마시고 대사(대사:계란 두개 말고 하나만 주세요/더 주세요/그만 먹어,이 미련 곰탱아!)
  * 5단계 : 한바퀴 구른 후 풍선 받아 등위에서 터뜨리고 '겸손한 브이' 대사
  * 6단계 : 당구 큐 아래로 림보한 후 당구 큐 전달
  * 7단계 : 묘기 당구 성공한 후 율동과 함께 '나이스 큐 짝짝짝'대사하기
  * 8단계 : 볼링공 던져 스트라이크 하기
  * 9단계 : 인간 피라미드 쌓고 다함께 '건강한 국민, 건강한 미소' 표어 제창하기

### 1.6. 9회 - 겨울리조트 편 ¶

  * 1단계 : 썰매타고 내려와 구르며 도착
  * 2단계 : 대사 한 후 눈뭉치로 내려치기(오랫만이군/나 잡아봐라/우리 심심한데 뽀뽀나 할까?)
  * 3단계 : 스노우보드타고 깃발 낚아채기
  * 4단계 : 반바퀴 돌아서 스노우보드 잡기
  * 5단계 : 스키타고 점프후 폴로 풍선 찍어 터뜨리기
  * 6단계 : 골프공을 넣은 다음에 '우리 수산물이 최고드래요'라는 대사하기
  * 7단계 : 눈덩이 던져 바구니로 받기
  * 8단계 : 인디안 소리내고 릴레이로 과자 받아 먹기
  * 9단계 : 전출연자 '한국의 사계절, 무한한 관광자원' 표어 제창

### 1.7. 10회 - 동물원 편 ¶

  * 1단계 : 달려가 출연자에게 목걸이 걸기
  * 2단계 : 마늘,쑥 먹고 '사람이 되려면 마늘과 쑥을 먹어야 돼'라는 대사하기
  * 3단계 : '웅녀 내 입술을 받아줘' '입술을 대' 대사 후에 노로 입술 때리고 보트에서 노 받아 풀장 건너기
  * 4단계 : '돌고래 쇼 스타트' 대사 후에 엉덩이 치면, 돌고래에게 공 던져 다시 받기
  * 5단계 : 공 받아 몸 뒤집은 다음 '지금 뭐가 지나갔나' '글쎄 너무 순식간에 벌어진 일이라'란 대사하기
  * 6단계 : '용왕님을 위해 토끼간이 필요해요'라는 대사 한 다음 두 마리 토끼 잡아 거북이에게 전달하기
  * 7단계 : 호랑이 탈 쓴 사람 번쩍 들어 올린 다음에 '동물의 제왕 호랑이'란 대사 후에 뽀뽀하기
  * 8단계 : 물개가 농구공 골인 시킨후 출연자도 공 던져 골인시킨다.
  * 9단계 : 한 사람이 선창한후 모든 출연자가 '오는 손님 친구처럼, 가는 손님 가족처럼' 표어 제창.

### 1.8. 11회- 해양테마파크 편 ¶

  * 1단계 : 힘차게 달려와 정글의 야수에게 접근
  * 2단계 : 뽀뽀해서 잠자는 야수 깨우고 대사하기(잠자는 나의 왕자님/공주, 참으로 고맙구만요/야 다시 써!)
  * 3단계 : 원형수조를 수족관에 있는 수초에다 정확하게 넣기
  * 4단계 : '수달 장군님, 식사하세요' 대사 후, 미꾸라지를 잡아서 수달에게 주기
  * 5단계 : 2인 1조로 같이 줄넘기 3회 실시
  * 6단계 : 황소 개구리 잡은 후 '개굴 개굴 개구리'란 대사를 율동과 같이 하기
  * 7단계 : '인어왕자, 바다의 보물을 찾아 주세요' 대사후 수중에서 보물상자에서 보물 꺼내 던지기
  * 8단계 : 문어다리에 고리 던져 걸기
  * 9단계 : '우리의 향기는 친절입니다' 표어 전 출연자 제창.

### 1.9. 12회 - 예식장 편 ¶

  * 3단계 : 옛 여인이 뛰쳐나와 '이 결혼식은 무효야, 뭐가 나보다 어디가 더 예뻐?'란 대사하기
  * 4단계 : 방석 위에 올라가 차례대로 방석 빼기
  * 5단계 : '자기, 잘 받아야 돼' '걱정마' 대사와 함께 대추 던져 먹기
  * 6단계 : '식권 한 장 주세요' 대사 전달 후 입으로 나르기
  * 7단계 : 신랑,신부 웨딩케익 촛불 한 번에 불어 끄기
  * 8단계 : '주현, 우리 둘만의 시간이군.' '아잉, 몰라 몰라' 대사 후에 풍선터뜨리기
  * 9단계 : 다함께 '신혼 여행은 한국으로 오세요' 대사 후에 사진 찍기

### 1.10. 13회 - 민속촌 편 ¶

  * 1단계 : 낮잠자던 놀부, 마누라에게로 달려가기
