### 4. 기술명 ¶

한글판, 일어판, 영어판 기술명과 더불어 콘솔판 기술표에 있는 설명문을 정리한 항목입니다. 언리미티드 캐릭터 시 추가되는 기술들은 공식
매체에서 정식 명칭이 소개된 게 아니라면 함부로 추가하지 마세요.

  

★ 드라이브 - 볼틱 배틀러(ボルテックバトラー / Voltec Battler) : 강력한 자력으로 공격합니다. 명중하면 상대는 일정 시간
동안 자력이 붙은 상태가 되어 특정 공격으로 끌어올 수 있습니다.  
★ 오버 드라이브 - 볼틱 필드(ボルテックフィールド / Voltec Field)

  

★ 필살기  

  * 기간틱 테이거 드라이브(ギガンティックテイガードライバー / Gigantic Tager Driver) : 강력한 잡기 공격입니다. 버튼을 누르고 있으면 자력이 붙은 상대를 끌어올 수 있습니다.
  * 아토믹 콜라이더(アトミックコレダー / Atomic Collider) : 공중에서 상대를 잡아 던집니다. 상대가 공중에 있고 자력이 붙은 상태인 경우, 버튼을 누르고 있으면 끌어올 수 있습니다.
  * 슬레지 해머(スレッジハンマー / Sledge Hammer) : 앞으로 돌진하는 공격입니다. 동작 도중에는 일부 장풍 공격을 맞지 않습니다.  
┗ 추가공격 : 상대를 추가로 공격합니다.

  * 스파크 볼트(スパークボルト / Spark Bolt) : 고속으로 직진하는 장풍을 날립니다. 명중하면 상대가 일정 시간 동안 자력이 붙은 상태가 됩니다. 충전 게이지가 MAX일 때 사용할 수 있습니다.
  * 볼틱 차지(ボルテックチャージ / Voltic Charge) : 충전 게이지를 모읍니다. 모으는 동안에는 일부 공격을 받지 않습니다.
  * 가제트 핑거(ガジェットフィンガー / Gadget Finger) : 다운된 상대를 일으켜 세우는 기술입니다.
  * 에어 드라이버(エアードライバー / Air Driver) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)
  * 그랑 퍼니쉬(グレンパニッシュ / Grand Punish) : [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md)  

★ 디스토션 드라이브  

  * 마그네틱 휠(マグナテックホイール / Magna-Tech Wheel) : 고속으로 회전하면서 공격합니다. 공격 도중에는 자력이 붙은 상대를 끌어올 수 있습니다.  
┗ 테라 브레이크(テラブレイク / Terra break) : 상대를 날리는 강력한 추가 공격을 합니다.

  * 제네식 에메랄드 테이거 버스터(ジェネシックエメラルドテイガーバスター / Genesic Emerald Tager Buster) : 아주 강력한 잡기 기술을 사용합니다. 버튼을 누르고 있으면 자력이 붙은 상대를 끌어올 수 있습니다.  

★ 아스트랄 히트  

  * 킹 오브 테이거(キングオブテイガー / King of Tager) : 아주 강력한 잡기 기술을 사용합니다. 버튼을 누르고 있으면 자력이 붙은 상대를 끌어들일 수 있습니다.
